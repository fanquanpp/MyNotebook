## 1. 简介 | Introduction

Lua 是一种轻量级、高效、可嵌入的脚本语言，由 Roberto Ierusalimschy、Waldemar Celes 和 Luiz Henrique de Figueiredo 于 1993 年创建。它设计为可扩展、可嵌入，具有简单而强大的语法，被广泛应用于游戏开发（如 Roblox、World of Warcraft、Angry Birds）、嵌入式系统和 Web 开发等领域。
