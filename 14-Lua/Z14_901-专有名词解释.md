# Lua 专有名词解释

## 1. 基础概念类

### 1.1 Lua

**英文名称**：Lua

**中文译名**：Lua 脚本语言

**出现位置**：C14_101-概述与环境.md 第1章

**定义**：
Lua 是一门简洁、轻量、可扩展的脚本语言，由巴西里约热内卢天主教大学于 1993 年开发，核心设计目标是易于嵌入到宿主程序中。

### 1.2 脚本语言

**英文名称**：Scripting Language

**中文译名**：脚本语言

**出现位置**：C14_101-概述与环境.md 第1章

**定义**：
脚本语言是一种解释型语言，代码不需要编译，在运行时逐行解释执行，通常用于自动化任务和扩展宿主程序功能。

### 1.3 嵌入式语言

**英文名称**：Embedded Language

**中文译名**：嵌入式语言

**出现位置**：C14_101-概述与环境.md 第1章

**定义**：
嵌入式语言是指可以嵌入到其他应用程序中运行的语言，Lua 设计之初就将嵌入性作为核心目标。

### 1.4 ANSI C

**英文名称**：American National Standards Institute C

**中文译名**：ANSI C 标准

**出现位置**：C14_101-概述与环境.md 第1.2章

**定义**：
ANSI C 是 C 语言的标准化版本，Lua 使用 ANSI C 编写，确保了其在各种平台上的可移植性。

### 1.5 动态类型

**英文名称**：Dynamic Typing

**中文译名**：动态类型

**出现位置**：C14_101-概述与环境.md 第1.2章

**定义**：
动态类型是指变量无需声明类型，类型在运行时确定，Lua 是典型的动态类型语言。

## 2. 数据结构类

### 2.1 Table

**英文名称**：Table

**中文译名**：表

**出现位置**：C14_103-数据类型与Table.md 第1章

**定义**：
Table 是 Lua 唯一的数据结构，可以模拟数组、哈希表、对象、模块等，使用关联数组实现。

### 2.2 数组

**英文名称**：Array

**中文译名**：数组

**出现位置**：C14_103-数据类型与Table.md 第2.1章

**定义**：
在 Lua 中，数组使用 Table 实现，下标从 1 开始而非 0，支持任意类型的元素。

### 2.3 哈希表

**英文名称**：Hash Table

**中文译名**：哈希表

**出现位置**：C14_103-数据类型与Table.md 第2.2章

**定义**：
在 Lua 中，哈希表使用 Table 实现，通过键值对存储数据，键可以是任意非 nil 值。

### 2.4 元表

**英文名称**：Metatable

**中文译名**：元表

**出现位置**：G14_201-元表与OOP.md 第1章

**定义**：
元表是 Lua 中的重要机制，允许改变表的行为，通过设置元表可以自定义表的算术运算、比较运算等行为。

### 2.5 元方法

**英文名称**：Metamethod

**中文译名**：元方法

**出现位置**：G14_201-元表与OOP.md 第1章

**定义**：
元方法是元表中定义的特殊方法，如 __add、__index、__newindex 等，用于实现运算符重载和自定义行为。

### 2.6 __index 元方法

**英文名称**：Index Metamethod

**中文译名**：索引元方法

**出现位置**：G14_201-元表与OOP.md 第2.1章

**定义**：
__index 是访问表中不存在键时的默认行为，可用于实现继承等高级特性。

### 2.7 __newindex 元方法

**英文名称**：New Index Metamethod

**中文译名**：新索引元方法

**出现位置**：G14_201-元表与OOP.md 第2.1章

**定义**：
__newindex 是向表中不存在键赋值时的行为，可用于实现只读表等特性。

## 3. 函数与闭包类

### 3.1 函数

**英文名称**：Function

**中文译名**：函数

**出现位置**：C14_104-函数与闭包.md 第1章

**定义**：
函数是 Lua 中的基本代码组织单元，用于封装可重复使用的代码块，支持多返回值。

### 3.2 多返回值

**英文名称**：Multiple Return Values

**中文译名**：多返回值

**出现位置**：C14_104-函数与闭包.md 第1.1章

**定义**：
Lua 函数可以返回多个值，调用者可以选择接收全部或部分返回值。

### 3.3 可变参数

**英文名称**：Variadic Parameters

**中文译名**：可变参数

**出现位置**：C14_104-函数与闭包.md 第1.2章

**定义**：
Lua 函数可以使用 `...` 表示可变参数，函数内部通过 `{...}` 或 select 获取参数。

### 3.4 闭包

**英文名称**：Closure

**中文译名**：闭包

**出现位置**：C14_104-函数与闭包.md 第2章

**定义**：
闭包是引用了外部局部变量的函数，Lua 的闭包机制支持函数式编程范式。

### 3.5 匿名函数

**英文名称**：Anonymous Function

**中文译名**：匿名函数

**出现位置**：C14_104-函数与闭包.md 第3章

**定义**：
匿名函数是没有命名的函数表达式，通常用于回调函数或作为参数传递给高阶函数。

### 3.6 高阶函数

**英文名称**：Higher-order Function

**中文译名**：高阶函数

**出现位置**：C14_104-函数与闭包.md 第4章

**定义**：
高阶函数是指接受函数作为参数或返回函数作为结果的函数，Lua 的 table.sort 就是高阶函数。

### 3.7 尾调用

**英文名称**：Tail Call

**中文译名**：尾调用

**出现位置**：C14_104-函数与闭包.md 第5章

**定义**：
尾调用是指在函数最后一步调用另一个函数，Lua 虚拟机对尾调用进行了优化，不会占用额外的栈空间。

## 4. 协程类

### 4.1 协程

**英文名称**：Coroutine

**中文译名**：协程

**出现位置**：C14_105-协程与异步.md 第1章

**定义**：
协程是 Lua 提供的非抢占式多任务机制，允许函数暂停执行并在后续恢复。

### 4.2 coroutine.create

**英文名称**：Coroutine Create

**中文译名**：创建协程

**出现位置**：C14_105-协程与异步.md 第1.1章

**定义**：
coroutine.create 用于创建协程，接收一个函数作为参数，返回协程对象。

### 4.3 coroutine.resume

**英文名称**：Coroutine Resume

**中文译名**：恢复协程

**出现位置**：C14_105-协程与异步.md 第1.2章

**定义**：
coroutine.resume 用于启动或恢复协程执行，第一次调用启动协程，后续调用恢复执行。

### 4.4 coroutine.yield

**英文名称**：Coroutine Yield

**中文译名**：挂起协程

**出现位置**：C14_105-协程与异步.md 第1.3章

**定义**：
coroutine.yield 用于暂停协程执行，将控制权返回给调用者。

### 4.5 非抢占式多任务

**英文名称**：Non-preemptive Multitasking

**中文译名**：非抢占式多任务

**出现位置**：C14_105-协程与异步.md 第1章

**定义**：
非抢占式多任务是指程序主动放弃执行权而非被系统强制中断，Lua 协程是典型的非抢占式多任务。

### 4.6 异步编程

**英文名称**：Asynchronous Programming

**中文译名**：异步编程

**出现位置**：C14_105-协程与异步.md 第2章

**定义**：
异步编程是一种编程范式，通过协程实现非阻塞执行，提高程序并发效率。

## 5. 面向对象类

### 5.1 面向对象编程

**英文名称**：Object-Oriented Programming

**中文译名**：面向对象编程

**出现位置**：G14_201-元表与OOP.md 第3章

**定义**：
面向对象编程是一种程序设计范式，通过对象封装数据和行为，Lua 通过元表实现 OOP 特性。

### 5.2 对象

**英文名称**：Object

**中文译名**：对象

**出现位置**：G14_201-元表与OOP.md 第3.1章

**定义**：
在 Lua 中，对象使用表实现，表的字段作为对象的属性，表的函数作为对象的方法。

### 5.3 继承

**英文名称**：Inheritance

**中文译名**：继承

**出现位置**：G14_201-元表与OOP.md 第3.2章

**定义**：
继承是面向对象的核心特性，允许子类复用父类的属性和方法，Lua 通过元表和 __index 实现继承。

### 5.4 多态

**英文名称**：Polymorphism

**中文译名**：多态

**出现位置**：G14_201-元表与OOP.md 第3.3章

**定义**：
多态是同一操作作用于不同对象产生不同行为的能力，Lua 通过方法覆写实现多态。

### 5.5 封装

**英文名称**：Encapsulation

**中文译名**：封装

**出现位置**：G14_201-元表与OOP.md 第3章

**定义**：
封装是将数据和操作数据的方法捆绑在一起，Lua 通过模块和表实现封装。

## 6. 模块与包类

### 6.1 模块

**英文名称**：Module

**中文译名**：模块

**出现位置**：G14_202-模块与包.md 第1章

**定义**：
模块是 Lua 中组织代码的方式，通常是一个返回表的函数，用于封装相关的功能和数据。

### 6.2 require

**英文名称**：Require

**中文译名**：require函数

**出现位置**：G14_202-模块与包.md 第1.1章

**定义**：
require 是 Lua 的模块加载函数，用于加载和缓存模块，返回模块提供的功能表。

### 6.3 package

**英文名称**：Package

**中文译名**：包

**出现位置**：G14_202-模块与包.md 第2章

**定义**：
包是 Lua 中管理模块和命名空间的机制，通过 package.path 和 package.cpath 配置模块搜索路径。

### 6.4 package.path

**英文名称**：Package Path

**中文译名**：包路径

**出现位置**：G14_202-模块与包.md 第2.1章

**定义**：
package.path 是 Lua 用于搜索 Lua 模块文件的路径列表。

### 6.5 package.cpath

**英文名称**：Package C Path

**中文译名**：C库路径

**出现位置**：G14_202-模块与包.md 第2.1章

**定义**：
package.cpath 是 Lua 用于搜索 C 编写的共享库的路径列表。

## 7. 内存管理类

### 7.1 垃圾回收

**英文名称**：Garbage Collection

**中文译名**：垃圾回收

**出现位置**：C14_101-概述与环境.md 第1.2章

**定义**：
Lua 自动管理内存，通过垃圾回收机制自动释放不再使用的内存，开发者无需手动释放。

### 7.2 增量式垃圾回收

**英文名称**：Incremental Garbage Collection

**中文译名**：增量式垃圾回收

**出现位置**：C14_101-概述与环境.md 第1.2章

**定义**：
增量式垃圾回收是 Lua 5.0 引入的机制，将垃圾回收分为多个增量步骤执行，减少停顿时间。

### 7.3 弱表

**英文名称**：Weak Table

**中文译名**：弱表

**出现位置**：C14_103-数据类型与Table.md

**定义**：
弱表的引用不会阻止垃圾回收，适用于缓存和备忘录等场景，通过设置元表的 __mode 字段创建。

### 7.4 循环引用

**英文名称**：Circular Reference

**中文译名**：循环引用

**出现位置**：C14_103-数据类型与Table.md

**定义**：
循环引用是指两个或多个对象相互引用形成环，在弱表出现之前会导致内存泄漏。

## 8. 语法特性类

### 8.1 局部变量

**英文名称**：Local Variable

**中文译名**：局部变量

**出现位置**：C14_102-基础语法.md 第2章

**定义**：
局部变量使用 local 关键字声明，作用域限于声明它的代码块内，比全局变量访问更快。

### 8.2 全局变量

**英文名称**：Global Variable

**中文译名**：全局变量

**出现位置**：C14_102-基础语法.md 第2章

**定义**：
在 Lua 中，直接赋值的变量是全局变量，存储在 _G 表中，应尽量避免使用全局变量。

### 8.3 字符串

**英文名称**：String

**中文译名**：字符串

**出现位置**：C14_103-数据类型与Table.md

**定义**：
Lua 字符串是不可变的值，支持多种编码，内部实现为字节数组。

### 8.4 数字

**英文名称**：Number

**中文译名**：数字

**出现位置**：C14_103-数据类型与Table.md

**定义**：
Lua 5.3+ 支持整数和浮点数两种数字类型，使用双精度浮点表示，可通过 jit.floatmode 配置。

### 8.5 布尔值

**英文名称**：Boolean

**中文译名**：布尔值

**出现位置**：C14_103-数据类型与Table.md

**定义**：
Lua 的布尔类型只有 true 和 false 两个值，nil 和 false 被视为假，其他值被视为真。

### 8.6 nil

**英文名称**：Nil

**中文译名**：空值

**出现位置**：C14_103-数据类型与Table.md

**定义**：
nil 是 Lua 中的特殊类型，表示无效值或未初始化的变量，表中删除键的方法是将其赋值为 nil。

### 8.7 迭代器

**英文名称**：Iterator

**中文译名**：迭代器

**出现位置**：C14_103-数据类型与Table.md

**定义**：
迭代器是返回连续元素的函数，Lua 的 for 循环使用迭代器遍历集合，pairs 和 ipairs 是内置迭代器。

### 8.8 ipairs

**英文名称**：Integer Pairs

**中文译名**：数组迭代器

**出现位置**：C14_103-数据类型与Table.md

**定义**：
ipairs 是 Lua 内置的数组迭代器，从 1 开始遍历数组，遇到 nil 停止。

### 8.9 pairs

**英文名称**：Pairs

**中文译名**：表迭代器

**出现位置**：C14_103-数据类型与Table.md

**定义**：
pairs 是 Lua 内置的表迭代器，遍历表中的所有键值对，不保证顺序。
