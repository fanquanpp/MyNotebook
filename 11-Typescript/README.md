## 1. 简介 | Introduction

TypeScript 是由 Microsoft 开发的自由和开源的编程语言，是 JavaScript 的超集。它添加了可选的静态类型和基于类的面向对象编程，能够编译成普通的 JavaScript。TypeScript 使得大型 JavaScript 应用的开发和维护变得更加容易。
