## 1. 简介 | Introduction

Java 是由 Sun Microsystems（现已被 Oracle 收购）于 1995 年推出的高级编程语言。其"一次编写，到处运行"（Write Once, Run Anywhere）的特性使其成为企业级应用开发、Android 应用开发和大数据处理等领域的主流选择。
