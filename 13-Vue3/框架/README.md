# Vue3 框架生态

Vue3 生态系统包含多个官方和社区维护的框架，这些框架可以帮助开发者构建更复杂、更强大的应用。本目录包含 Vue3 生态中最核心的框架：Vue Router 和 Pinia 状态管理。

## 目录结构

| 文件名 | 内容简介 |
| --- | --- |
| 01-Vue Router.md | Vue Router 基础、路由配置、路由守卫、嵌套路由、动态路由、编程式导航、路由元信息 |
| 02-Pinia 状态管理.md | Pinia 基础、Store 定义、状态管理、Actions、Getters、模块化、持久化、插件 |

## 学习建议

1. **先掌握 Vue3 基础**：在学习这些框架之前，建议先熟悉 Vue3 的核心概念和组合式 API
2. **循序渐进**：先学习 Vue Router，再学习 Pinia，因为路由是大多数应用的基础
3. **实践为主**：通过实际项目练习巩固所学知识
4. **查阅官方文档**：遇到问题时参考官方文档
