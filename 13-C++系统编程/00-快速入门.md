# 00-快速入门 (C++)

## 1. 引言
C++ 是一种通用的、强类型的、多范式的编程语言，由 Bjarne Stroustrup 开发。它是 C 语言的扩展，增加了面向对象编程、泛型编程等特性，广泛用于系统开发、游戏引擎、嵌入式系统和高性能计算。

## 2. 环境搭建
### 2.1 编译器安装
- **Windows**: 安装 [Visual Studio](https://visualstudio.microsoft.com/) (推荐) 或 [MinGW-w64](https://www.mingw-w64.org/)。
- **macOS**: `xcode-select --install`。
- **Linux**: `sudo apt install build-essential`。

### 2.2 IDE 配置
- **VS Code**: 安装 C/C++ 扩展包。
- **CLion**: 专业的 C++ IDE。

## 3. Hello World 示例
创建一个名为 `hello.cpp` 的文件：
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, C++!" << std::endl;
    return 0;
}
```
**编译与运行**:
```bash
g++ hello.cpp -o hello
./hello
```

## 4. 核心工作流
1. **编写**: 使用 `.cpp` (源文件) 和 `.h`/`.hpp` (头文件)。
2. **构建**: 使用 `CMake` 或 `Makefile` 管理复杂项目。
3. **调试**: 使用 `GDB` 或 IDE 内置调试器。
