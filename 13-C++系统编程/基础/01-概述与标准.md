## 1. C++ 概述 (Overview)
C++ 是由 **Bjarne Stroustrup** 于 1979 年在贝尔实验室开发的，最初称为 "C with Classes"。它是一门编译型、静态类型、多范式（过程式、面向对象、泛型）的编程语言。

### 1.1 核心优势 (Key Strengths)
- **高性能**: 零开销抽象 (Zero-overhead Abstractions)，直接操作硬件。
- **灵活性**: 支持底层内存管理与高层抽象。
- **可移植性**: 广泛应用于各种系统平台。
- **庞大的生态**: 数十年的库积累，尤其在系统级、游戏、嵌入式领域。

## 2. 现代 C++ 标准 (Standards)
- **C++98/03**: 经典 C++。
- **C++11**: 革命性的里程碑 (auto, nullptr, lambda, 智能指针, 多线程)。
- **C++14/17**: 细节优化与功能增强 (std::optional, std::variant, 文件系统)。
- **C++20**: 又一重大更新 (Concepts, Coroutines, Modules, Ranges)。
- **C++23**: 持续改进。

## 3. 应用领域 (Applications)
- **操作系统**: Windows, macOS, 移动端内核。
- **游戏引擎**: Unreal Engine (虚幻引擎)。
- **金融系统**: 高频交易、量化分析。
- **科学计算**: 物理模拟、航天。
- **嵌入式**: 汽车电子、机器人。

## 4. 学习路线建议 (Learning Path)
1. **基础语法**: 变量、控制流、函数。
2. **内存管理**: 指针、引用、动态内存、RAII。
3. **面向对象**: 封装、继承、多态。
4. **泛型编程**: 模板 (STL)。
5. **现代 C++**: C++11 及更高版本的新特性。
