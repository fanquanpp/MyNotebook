# Markdown 核心知识点

<a id="1"></a>
## 1 . Markdown 概述

Markdown 是一种轻量级标记语言，由 John Gruber 和 Aaron Swartz 于 2004 年创建。它允许人们使用易读易写的纯文本格式编写文档，然后可以转换为 HTML、PDF 等格式。

<a id="1.1"></a>
### 1.1 Markdown 的核心特点
- **简单易用**：语法简洁，易于学习和使用
- **跨平台**：可以在任何文本编辑器中编写
- **可转换**：可以转换为多种格式，如 HTML、PDF 等
- **可读性强**：即使不转换，纯文本也具有良好的可读性
- **广泛支持**：被 GitHub、Stack Overflow 等平台广泛使用

<a id="1.2"></a>
### 1.2 Markdown 的应用场景
- **文档编写**：技术文档、API 文档等
- **博客写作**：许多博客平台支持 Markdown
- **代码注释**：用于代码库的 README 文件
- **版本控制**：与 Git 等版本控制系统配合使用
- **在线协作**：在 GitHub、GitLab 等平台上进行协作

## 2 . 标题语法

<a id="2.1"></a>
### 2.1 基本标题

使用`#`符号创建标题，`#`的数量表示标题级别。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

<a id="2.2"></a>
### 2.2 标题样式

- **Setext 样式**：使用`=`和`-`创建一级和二级标题

```markdown
一级标题
========

二级标题
--------
```

<a id="3"></a>
## 3 . 段落与换行

<a id="3.1"></a>
### 3.1 段落

段落之间使用空行分隔。

```markdown
这是第一个段落。

这是第二个段落。
```

<a id="3.2"></a>
### 3.2 换行

- 在行尾添加两个或更多空格，然后按 Enter 键
- 使用 HTML 的`<br>`标签

```markdown
这是第一行  
这是第二行

这是第一行<br>
这是第二行
```

<a id="4"></a>
## 4 . 基础文本格式

<a id="4.1"></a>
### 4.1 粗体

使用`**`或`__`包裹文本。

```markdown
**粗体文本**
__粗体文本__
```

<a id="4.2"></a>
### 4.2 斜体

使用`*`或`_`包裹文本。

```markdown
*斜体文本*
_斜体文本_
```

<a id="4.3"></a>
### 4.3 粗斜体

使用`***`或`___`包裹文本。

```markdown
***粗斜体文本***
___粗斜体文本___
```

<a id="4.4"></a>
### 4.4 删除线

使用`~~`包裹文本。

```markdown
~~删除线文本~~
```

<a id="4.5"></a>
### 4.5 行内代码

使用`` ` ``包裹代码。

```markdown
`行内代码`
```

<a id="4.6"></a>
### 4.6 代码块

使用三个反引号```包裹代码块，可指定语言。

```markdown
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
```

<a id="5"></a>
## 5 . 列表语法

<a id="5.1"></a>
### 5.1 无序列表

使用`-`、`*`或`+`创建无序列表。

```markdown
- 项目 1
- 项目 2
- 项目 3

* 项目 1
* 项目 2
* 项目 3

+ 项目 1
+ 项目 2
+ 项目 3
```

<a id="5.2"></a>
### 5.2 有序列表

使用数字加`.`创建有序列表。

```markdown
1. 项目 1
2. 项目 2
3. 项目 3
```

<a id="5.3"></a>
### 5.3 嵌套列表

在列表项前添加缩进创建嵌套列表。

```markdown
- 一级列表项
  - 二级列表项
    - 三级列表项

1. 一级列表项
   1. 二级列表项
      1. 三级列表项
```

<a id="6"></a>
## 6 . 链接

<a id="6.1"></a>
### 6.1 行内链接

```markdown
[链接文本](链接地址 "可选标题")

示例：
[Google](https://www.google.com "Google 搜索")
```

<a id="6.2"></a>
### 6.2 参考链接

```markdown
[链接文本][参考标签]

[参考标签]: 链接地址 "可选标题"

示例：
[Google][1]

[1]: https://www.google.com "Google 搜索"
```

<a id="6.3"></a>
### 6.3 自动链接

使用尖括号包围 URL 或邮箱地址。

```markdown
<https://www.google.com>
<example@example.com>
```

<a id="7"></a>
## 7 . 图片

<a id="7.1"></a>
### 7.1 基本图片

```markdown
![图片描述](图片地址 "可选标题")

示例：
![示例图片](https://example.com/image.jpg "示例图片")
```

<a id="7.2"></a>
### 7.2 参考图片

```markdown
![图片描述][参考标签]

[参考标签]: 图片地址 "可选标题"

示例：
![示例图片][1]

[1]: https://example.com/image.jpg "示例图片"
```

## 8 . 引用

<a id="8.1"></a>
### 8.1 基本引用

使用`>`创建引用。

```markdown
> 这是一段引用文本。
```

<a id="8.2"></a>
### 8.2 嵌套引用

使用多个`>`创建嵌套引用。

```markdown
> 一级引用
>> 二级引用
>>> 三级引用
```

<a id="8.3"></a>
### 8.3 引用中包含其他元素

```markdown
> #### 引用中的标题
> - 引用中的列表
> - 引用中的列表项
> 
> 引用中的段落。
```

<a id="9"></a>
## 9 . 表格

<a id="9.1"></a>
### 9.1 基本表格

```markdown
| 表头 1 | 表头 2 | 表头 3 |
| :--- | :--- | :--- |
| 单元格 1 | 单元格 2 | 单元格 3 |
| 单元格 4 | 单元格 5 | 单元格 6 |
```

<a id="9.2"></a>
### 9.2 对齐方式

- `:---`：左对齐
- `:---:`：居中对齐
- `---:`：右对齐

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :--- | ---: |
| 内容 1 | 内容 2 | 内容 3 |
| 内容 4 | 内容 5 | 内容 6 |
```

<a id="10"></a>
## 10 . 分割线

使用三个或更多的`-`、`*`或`_`创建分割线。

```markdown
---

***

___
```

<a id="11"></a>
## 11 . 转义字符

使用`\`转义特殊字符。

```markdown
\# 这不是标题
\* 这不是斜体
\` 这不是行内代码
```

使用`$`包裹数学公式。

```markdown
$E=mc^2$
```

<a id="13.2"></a>

```

## 14 . 代码高亮

在代码块的反引号后指定语言，实现代码高亮。

```markdown
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```python
def hello():
    print("Hello, World!")
```
```

## 15 . 脚注

使用`[^]`创建脚注。

```markdown
这是一段带有脚注的文本[^1]。

[^1]: 这是脚注内容。
```

```

## 19 . 特殊功能

<a id="19.1"></a>
### 19.1 GitHub Flavored Markdown (GFM)

GitHub 支持一些扩展功能：

- **任务列表**：
  ```markdown
  - [x] 已完成任务
  - [ ] 未完成任务
  ```

- **代码块语法高亮**：
  ```markdown
  ```java
  public class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
  ```

- **表格**：
  ```markdown
  | 表头 1 | 表头 2 |
  | :--- | :--- |
  | 单元格 1 | 单元格 2 |
  ```

- **自动链接**：
  ```markdown
  https://www.github.com
  ```

- **表情符号**：
  ```markdown
  :smile:
  :+1:
  ```

<a id="19.2"></a>
### 19.2 其他扩展

- **KaTeX**：用于数学公式渲染
- **Mermaid**：用于绘制图表
- **PlantUML**：用于绘制 UML 图

## 21 . 常见问题与解决方案

<a id="21.1"></a>
### 21.1 换行问题

**问题**：Markdown 中的换行不生效
**解决方案**：在行尾添加两个或更多空格，然后按 Enter 键

<a id="21.2"></a>
### 21.2 表格对齐

**问题**：表格列对齐不正确
**解决方案**：使用`:`来指定对齐方式

<a id="21.3"></a>
### 21.3 代码高亮

**问题**：代码高亮不生效
**解决方案**：在代码块的反引号后指定语言

<a id="21.4"></a>
### 21.4 数学公式

**问题**：数学公式不渲染
**解决方案**：使用支持数学公式的 Markdown 处理器，如 KaTeX

## 22 . 总结

Markdown 是一种简单易用的标记语言，它允许人们使用纯文本格式编写文档，然后可以转换为多种格式。通过掌握 Markdown 的核心知识点，你可以编写更加清晰、结构化的文档。

<a id="22.1"></a>
### 22.1 核心要点

- **标题**：使用`#`创建不同级别的标题
- **文本格式**：粗体、斜体、删除线等
- **列表**：无序列表、有序列表、嵌套列表
- **链接**：行内链接、参考链接、自动链接
- **图片**：插入图片并添加描述
- **引用**：创建引用和嵌套引用
- **表格**：创建表格并设置对齐方式
- **代码**：行内代码和代码块
- **其他**：分割线、转义字符、任务列表等

<a id="22.2"></a>
### 22.2 学习建议

- **实践**：通过实际编写 Markdown 文档来练习
- **参考**：查看优秀的 Markdown 示例
- **工具**：使用支持 Markdown 的编辑器
- **规范**：遵循 Markdown 最佳实践
- **持续学习**：关注 Markdown 的新特性和扩展

通过不断学习和实践，你将能够熟练使用 Markdown，编写更加优秀的文档。
