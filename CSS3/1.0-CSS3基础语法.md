## 1. 概述

CSS（Cascading Style Sheets）是用于描述HTML或XML文档样式的语言，CSS3是CSS的最新版本，带来了许多新特性和改进。本教程将详细介绍CSS3的基础语法和使用方法。

## 2. CSS基本语法

### 2.1 语法结构

CSS规则由选择器和声明块组成：

```css
选择器 {
  属性1: 值1;
  属性2: 值2;
  /* 注释 */
}
```

- **选择器**：指定要应用样式的HTML元素
- **声明块**：包含一个或多个样式声明
- **样式声明**：由属性和值组成，用冒号分隔，以分号结尾

### 2.2 CSS注释

CSS注释用于添加说明，不会影响样式效果：

```css
/* 这是单行注释 */

/*
这是多行注释
可以跨越多行
*/
```

## 3. CSS引入方式

### 3.1 内联样式

直接在HTML元素中使用`style`属性：

```html
<div style="color: red; font-size: 16px;">文本内容</div>
```

**特点**：
- 优先级最高
- 只影响单个元素
- 不利于维护

### 3.2 内部样式

在HTML文档头部使用`style`标签：

```html
<style>
  div {
    color: red;
    font-size: 16px;
  }
</style>
```

**特点**：
- 影响当前HTML文件中的所有匹配元素
- 优先级高于外部样式
- 适用于单个页面的样式

### 3.3 外部样式

使用`link`标签引入外部CSS文件：

```html
<link rel="stylesheet" href="style.css">
```

**特点**：
- 可以被多个页面共享
- 便于维护和管理
- 优先级低于内联和内部样式

### 3.4 @import

在CSS文件中导入其他CSS文件：

```css
@import url("other.css");
```

**特点**：
- 可以在CSS文件中组织样式
- 会增加HTTP请求
- 优先级低于外部样式

## 4. CSS优先级

CSS优先级决定了当多个样式规则应用于同一元素时，哪个规则会被优先使用。

### 4.1 优先级规则

从高到低：
1. **!important声明**：最高优先级
2. **内联样式**：使用style属性的样式
3. **ID选择器**：以#开头的选择器
4. **类选择器、属性选择器、伪类选择器**：以.开头的选择器等
5. **元素选择器、伪元素选择器**：如div、p等
6. **通用选择器**：* 选择器

### 4.2 优先级计算

- **内联样式**：1000
- **ID选择器**：100
- **类选择器、属性选择器、伪类选择器**：10
- **元素选择器、伪元素选择器**：1
- **通用选择器**：0

计算示例：
- `#header .nav a`：100 + 10 + 1 = 111
- `.container div`：10 + 1 = 11
- `div`：1

### 4.3 优先级规则应用

```css
/* 优先级：10 */
.nav {
  color: blue;
}

/* 优先级：100 */
#main-nav {
  color: red; /* 会覆盖上面的样式 */
}

/* 优先级：1000 */
<div id="main-nav" class="nav" style="color: green;">导航</div> /* 会覆盖上面的样式 */

/* 优先级：最高 */
.nav {
  color: purple !important; /* 会覆盖内联样式 */
}
```

## 5. CSS属性和值

### 5.1 属性

CSS属性用于指定要设置的样式特性，如颜色、字体、间距等。

### 5.2 值

CSS值用于指定属性的具体设置，如颜色值、长度值、百分比等。

### 5.3 常见属性值类型

| 类型 | 示例 | 描述 |
|------|------|------|
| 颜色 | `red`, `#ff0000`, `rgb(255,0,0)` | 颜色值 |
| 长度 | `10px`, `1em`, `50%` | 长度值 |
| 数字 | `1`, `0.5`, `2` | 数值 |
| 字符串 | `"Arial"`, `'sans-serif'` | 字符串 |
| 关键字 | `solid`, `block`, `center` | 预定义的关键字 |
| URL | `url("image.jpg")` | 资源路径 |
| 函数 | `rgb(255,0,0)`, `calc(100% - 20px)` | 函数表达式 |

## 6. CSS单位

### 6.1 绝对单位

| 单位 | 描述 |
|------|------|

| `pt` | 点（1pt = 1/72英寸） |
| `pc` | 派卡（1pc = 12pt） |

| `cm` | 厘米 |
| `mm` | 毫米 |

### 6.2 相对单位

| 单位 | 描述 |
|------|------|
| `em` | 相对于父元素的字体大小 |
| `rem` | 相对于根元素的字体大小 |
| `vw` | 视口宽度的1% |
| `vh` | 视口高度的1% |
| `vmin` | 视口宽度和高度中较小值的1% |
| `vmax` | 视口宽度和高度中较大值的1% |

## 8. CSS选择器

### 8.1 基础选择器

| 选择器 | 描述 | 示例 |
|------|------|------|
| 元素选择器 | 选择指定类型的元素 | `div { color: red; }` |
| ID选择器 | 选择指定ID的元素 | `#header { background: blue; }` |
| 类选择器 | 选择指定类的元素 | `.container { width: 100%; }` |
| 通用选择器 | 选择所有元素 | `* { margin: 0; padding: 0; }` |
| 属性选择器 | 选择具有指定属性的元素 | `[type="text"] { border: 1px solid #ccc; }` |

### 8.2 组合选择器

| 选择器 | 描述 | 示例 |
|------|------|------|
| 后代选择器 | 选择元素的后代元素 | `div p { color: red; }` |
| 子选择器 | 选择元素的直接子元素 | `div > p { color: red; }` |
| 相邻兄弟选择器 | 选择元素的下一个兄弟元素 | `div + p { color: red; }` |
| 通用兄弟选择器 | 选择元素的所有兄弟元素 | `div ~ p { color: red; }` |

### 8.3 伪类选择器

| 选择器 | 描述 | 示例 |
|------|------|------|
| `:hover` | 鼠标悬停时 | `a:hover { color: red; }` |
| `:active` | 元素被激活时 | `button:active { background: blue; }` |
| `:focus` | 元素获得焦点时 | `input:focus { border: 2px solid blue; }` |
| `:first-child` | 第一个子元素 | `li:first-child { font-weight: bold; }` |
| `:last-child` | 最后一个子元素 | `li:last-child { color: red; }` |
| `:nth-child(n)` | 第n个子元素 | `li:nth-child(2) { color: blue; }` |
| `:not(selector)` | 排除指定选择器 | `p:not(.special) { color: gray; }` |

### 8.4 伪元素选择器

| 选择器 | 描述 | 示例 |
|------|------|------|
| `::before` | 在元素内容前插入内容 | `p::before { content: "前缀"; }` |
| `::after` | 在元素内容后插入内容 | `p::after { content: "后缀"; }` |
| `::first-letter` | 选择元素的第一个字母 | `p::first-letter { font-size: 2em; }` |
| `::first-line` | 选择元素的第一行 | `p::first-line { font-weight: bold; }` |
| `::selection` | 选择用户选中的内容 | `::selection { background: yellow; }` |

## 9. CSS3新特性

### 9.1 颜色系统增强

- **RGBA**：带透明度的RGB颜色
  ```css
  color: rgba(255, 0, 0, 0.5);
  ```

- **HSL**：色相、饱和度、亮度
  ```css
  color: hsl(0, 100%, 50%);
  ```

- **HSLA**：带透明度的HSL颜色
  ```css
  color: hsla(0, 100%, 50%, 0.5);
  ```

### 9.2 边框圆角

```css
border-radius: 5px;
/* 或 */
border-radius: 5px 10px 15px 20px;
```

### 9.3 阴影效果

```css
/* 文本阴影 */
text-shadow: 1px 1px 2px rgba(0, 0, 0, 0.5);

/* 盒子阴影 */
box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
```

### 9.4 渐变效果

```css
/* 线性渐变 */
background: linear-gradient(to right, red, blue);

/* 径向渐变 */
background: radial-gradient(circle, red, blue);
```

### 9.5 过渡效果

```css
transition: all 0.3s ease;
```

### 9.6 动画效果

```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.element {
  animation: fadeIn 1s ease;
}
```

## 10. 最佳实践

### 10.1 代码组织

- **模块化**：将CSS按功能模块组织
- **命名规范**：使用BEM、SMACSS等命名规范
- **注释**：为复杂样式添加注释
- **缩进**：使用4个空格或1个制表符进行缩进
- **空格**：在选择器和大括号之间、属性名和冒号之间添加空格

### 10.2 性能优化

- **减少HTTP请求**：合并CSS文件
- **压缩CSS**：使用工具压缩CSS文件
- **减少选择器复杂度**：避免使用过于复杂的选择器
- **使用CSS变量**：减少重复代码
- **避免使用@import**：@import会增加HTTP请求
- **使用简写属性**：减少代码量

### 10.3 可维护性

- **使用CSS预处理器**：如Sass、Less、Stylus
- **使用CSS模块化**：如CSS Modules、Styled Components
- **版本控制**：使用Git等版本控制系统管理CSS文件
- **文档化**：为CSS代码添加文档
- **一致性**：保持代码风格一致
