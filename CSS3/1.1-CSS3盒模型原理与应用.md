## 3 . 盒模型概述

盒模型（Box Model）是 CSS 中用于布局的核心概念，它将每个 HTML 元素视为一个矩形盒子，包含内容区域、内边距、边框和外边距。

<a id="4"></a>
## 4 . 盒模型组成

盒模型由以下部分组成：

- **内容区域**：width 和 height 属性定义的区域，用于显示元素的内容
- **内边距**：padding 属性定义，出现在内容与盒子边缘之间，用于控制内容与边框之间的距离
- **边框**：border 属性定义，围绕内边距和内容区域的边界
- **外边距**：margin 属性定义，出现在盒子外面，用于控制元素与其他元素之间的距离

<a id="5"></a>
## 5 . 盒模型类型

CSS 中有两种盒模型类型：

<a id="5.1"></a>
### 5.1 标准盒模型（Content Box）

在标准盒模型中，width 和 height 只包括内容区域的宽度和高度，不包括内边距和边框。

```css
.box {
  box-sizing: content-box; /* 默认值 */
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid #000;
  /* 实际宽度：200px + 10px*2 + 2px*2 = 224px */
  /* 实际高度：100px + 10px*2 + 2px*2 = 124px */
}
```

<a id="5.2"></a>
### 5.2 IE 盒模型（Border Box）

在 IE 盒模型中，width 和 height 包括内容区域、内边距和边框的宽度和高度。

```css
.box {
  box-sizing: border-box;
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 2px solid #000;
  /* 实际宽度：200px（包含 padding 和 border） */
  /* 实际高度：100px（包含 padding 和 border） */
  /* 内容区域宽度：200px - 10px*2 - 2px*2 = 176px */
  /* 内容区域高度：100px - 10px*2 - 2px*2 = 76px */
}
```

<a id="5.3"></a>
### 5.3 盒模型切换

可以通过 `box-sizing` 属性切换盒模型：

```css
/* 标准盒模型 */
.box {
  box-sizing: content-box;
}

/* IE 盒模型 */
.box {
  box-sizing: border-box;
}
```

## 6 . 盒模型属性

<a id="6.1"></a>
### 6.1 宽度和高度

```css
.box {
  width: 200px; /* 宽度 */
  height: 100px; /* 高度 */
  min-width: 100px; /* 最小宽度 */
  max-width: 300px; /* 最大宽度 */
  min-height: 50px; /* 最小高度 */
  max-height: 200px; /* 最大高度 */
}
```

<a id="6.2"></a>
### 6.2 内边距（padding）

```css
.box {
  padding: 10px; /* 四个方向都为 10px */
  padding: 10px 20px; /* 上下 10px，左右 20px */
  padding: 10px 20px 30px; /* 上 10px，左右 20px，下 30px */
  padding: 10px 20px 30px 40px; /* 上、右、下、左 */
  
  /* 单独设置 */
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
}
```

<a id="6.3"></a>
### 6.3 边框（border）

```css
.box {
  border: 1px solid #000; /* 宽度、样式、颜色 */
  
  /* 单独设置 */
  border-width: 1px;
  border-style: solid;
  border-color: #000;
  
  /* 单独设置某一边 */
  border-top: 2px dashed red;
  border-right: 1px solid blue;
  border-bottom: 3px double green;
  border-left: 1px dotted yellow;
  
  /* 边框圆角 */
  border-radius: 5px;
  border-radius: 5px 10px 15px 20px; /* 左上、右上、右下、左下 */
  border-radius: 50%; /* 圆形 */
}
```

<a id="6.4"></a>
### 6.4 外边距（margin）

```css
.box {
  margin: 10px; /* 四个方向都为 10px */
  margin: 10px 20px; /* 上下 10px，左右 20px */
  margin: 10px 20px 30px; /* 上 10px，左右 20px，下 30px */
  margin: 10px 20px 30px 40px; /* 上、右、下、左 */
  
  /* 单独设置 */
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  
  /* 居中对齐 */
  margin: 0 auto; /* 水平居中 */
}
```

<a id="6.5"></a>
### 6.5 溢出处理（overflow）

| 属性值 | 效果 |
| :--- | :--- |
| `visible` | 默认值，内容溢出时显示在盒子外面 |
| `hidden` | 溢出隐藏 |
| `scroll` | 溢出滚动（无论是否溢出，都显示滚动条位置） |
| `auto` | 溢出滚动（溢出才显示滚动条位置） |
| `clip` | 溢出裁剪，不显示滚动条 |

```css
.box {
  overflow: hidden;
  overflow-x: auto; /* 水平方向溢出处理 */
  overflow-y: scroll; /* 垂直方向溢出处理 */
}
```

<a id="6.6"></a>
### 6.6 显示方式（display）

| 属性值 | 效果 |
| :--- | :--- |
| `block` | 块级元素，独占一行 |
| `inline` | 行内元素，不独占一行 |
| `inline-block` | 行内块元素，兼具块级和行内元素特性 |
| `none` | 隐藏元素 |
| `flex` | 弹性布局容器 |
| `grid` | 网格布局容器 |
| `table` | 表格布局 |

```css
.box {
  display: block;
  display: inline;
  display: inline-block;
  display: none;
  display: flex;
  display: grid;
  display: table;
}
```

## 7 . 盒模型应用

<a id="7.1"></a>
### 7.1 居中布局

<a id="7.1.1"></a>
#### 7.1.1 水平居中

```css
/* 块级元素水平居中 */
.center {
  width: 50%;
  margin: 0 auto;
}

/* 行内元素水平居中 */
.parent {
  text-align: center;
}

.child {
  display: inline-block;
}
```

<a id="7.1.2"></a>
#### 7.1.2 垂直居中

```css
/* 使用 Flexbox 垂直居中 */
.parent {
  display: flex;
  align-items: center;
  justify-content: center;
  height: 300px;
}

/* 使用 Grid 垂直居中 */
.parent {
  display: grid;
  place-items: center;
  height: 300px;
}

/* 使用绝对定位垂直居中 */
.parent {
  position: relative;
  height: 300px;
}

.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

<a id="7.2"></a>
### 7.2 多列布局

```css
/* 两列布局 */
.container {
  overflow: hidden;
}

.left {
  float: left;
  width: 30%;
  box-sizing: border-box;
  padding: 20px;
}

.right {
  margin-left: 30%;
  box-sizing: border-box;
  padding: 20px;
}

/* 三列布局 */
.container {
  overflow: hidden;
}

.left {
  float: left;
  width: 20%;
  box-sizing: border-box;
  padding: 20px;
}

.center {
  margin: 0 20%;
  box-sizing: border-box;
  padding: 20px;
}

.right {
  float: right;
  width: 20%;
  box-sizing: border-box;
  padding: 20px;
}
```

<a id="7.3"></a>
### 7.3 响应式盒模型

```css
/* 响应式盒模型 */
.box {
  width: 100%;
  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
  box-sizing: border-box;
}

/* 媒体查询 */
@media (max-width: 768px) {
  .box {
    padding: 10px;
  }
}
```

## 8 . 常见问题与解决方案

<a id="8.1"></a>
### 8.1 盒模型问题

**问题**：元素实际宽度超出预期

**原因**：标准盒模型中，width 和 height 只包括内容区域，不包括 padding 和 border

**解决方案**：使用 box-sizing: border-box

```css
* {
  box-sizing: border-box;
}
```

<a id="8.2"></a>
### 8.2 外边距折叠问题

**问题**：相邻元素的外边距会合并成一个外边距

**原因**：CSS 的外边距折叠规则

**解决方案**：
- 使用 padding 代替 margin
- 使用 border 或 padding 创建分隔
- 使用 overflow: hidden 创建新的块级格式化上下文

```css
/* 使用 overflow: hidden */
.parent {
  overflow: hidden;
}

.child {
  margin: 20px;
}
```

<a id="8.3"></a>
### 8.3 垂直居中问题

**问题**：元素无法垂直居中

**解决方案**：
- 使用 Flexbox
- 使用 Grid
- 使用绝对定位和 transform
- 使用表格布局

```css
/* 使用 Flexbox */
.container {
  display: flex;
  align-items: center;
  justify-content: center;
  height: 300px;
}
```

<a id="8.4"></a>
### 8.4 响应式图片问题

**问题**：图片在小屏幕上溢出容器

**解决方案**：

```css
img {
  max-width: 100%;
  height: auto;
}
```
