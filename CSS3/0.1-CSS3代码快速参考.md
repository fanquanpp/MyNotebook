## 3 . 选择器

<a id="3.1"></a>
### 3.1 基础选择器

```css
/* 元素选择器 */
div { color: red; }

/* ID 选择器 */
#header { background: blue; }

/* 类选择器 */
.container { width: 100%; }

/* 通用选择器 */
* { margin: 0; padding: 0; }

/* 属性选择器 */
[type="text"] { border: 1px solid #ccc; }
```

<a id="3.2"></a>
### 3.2 组合选择器

```css
/* 后代选择器 */
div p { color: red; }

/* 子选择器 */
div > p { color: red; }

/* 相邻兄弟选择器 */
div + p { color: red; }

/* 通用兄弟选择器 */
div ~ p { color: red; }
```

<a id="3.3"></a>
### 3.3 伪类选择器

```css
/* 鼠标悬停 */
a:hover { color: red; }

/* 元素被激活 */
button:active { background: blue; }

/* 元素获得焦点 */
input:focus { border: 2px solid blue; }

/* 第一个子元素 */
li:first-child { font-weight: bold; }

/* 第 n 个子元素 */
li:nth-child(2) { color: blue; }
```

## 5 . 布局技术

<a id="5.1"></a>
### 5.1 Flexbox 布局

```css
/* Flex 容器 */
.flex-container {
  display: flex;
  flex-direction: row; /* row, column, row-reverse, column-reverse */
  justify-content: space-between; /* flex-start, flex-end, center, space-around, space-between */
  align-items: center; /* flex-start, flex-end, center, baseline, stretch */
  flex-wrap: wrap; /* nowrap, wrap, wrap-reverse */
}

/* Flex 项目 */
.flex-item {
  flex: 1 1 200px; /* flex-grow, flex-shrink, flex-basis */
  margin: 10px;
}
```

<a id="5.2"></a>
### 5.2 Grid 布局

```css
/* Grid 容器 */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 列，每列 1 份 */
  grid-template-rows: 100px 200px; /* 2 行，高度分别为 100px 和 200px */
  gap: 10px; /* 行列间距 */
}

/* Grid 项目 */
.grid-item {
  grid-column: 1 / 3; /* 从第 1 列到第 3 列 */
  grid-row: 1 / 2; /* 从第 1 行到第 2 行 */
}
```

## 9 . 动画与过渡

<a id="9.1"></a>
### 9.1 过渡效果

```css
.transition {
  transition: all 0.3s ease;
  /* 单独设置 */
  transition-property: color, background-color;
  transition-duration: 0.3s;
  transition-timing-function: ease;
  transition-delay: 0.1s;
}

.transition:hover {
  color: red;
  background-color: yellow;
}
```

<a id="9.2"></a>
### 9.2 关键帧动画

```css
/* 定义动画 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 使用动画 */
.animation {
  animation: fadeIn 1s ease-in-out;
  /* 单独设置 */
  animation-name: fadeIn;
  animation-duration: 1s;
  animation-timing-function: ease-in-out;
  animation-delay: 0s;
  animation-iteration-count: 1;
  animation-direction: normal;
  animation-fill-mode: forwards;
}
```

## 10 . 响应式设计

<a id="10.1"></a>
### 10.1 媒体查询

```css
/* 基本媒体查询 */
@media (max-width: 768px) {
  .container {
    width: 100%;
  }
}

/* 多个断点 */
@media (min-width: 768px) and (max-width: 992px) {
  .container {
    width: 750px;
  }
}

/* 设备方向 */
@media (orientation: landscape) {
  .container {
    flex-direction: row;
  }
}
```

<a id="10.2"></a>
### 10.2 响应式单位

```css
.responsive {
  width: 100%;
  max-width: 1200px;
  font-size: 16px;
  /* 视口单位 */
  width: 50vw; /* 视口宽度的 50% */
  height: 50vh; /* 视口高度的 50% */
  font-size: 5vmin; /* 视口最小边的 5% */
}
```

## 11 . 常见问题与解决方案

### 11.1 清除浮动

```css
/* 方法 1：使用 clearfix */
.clearfix::after {
  content: '';
  display: table;
  clear: both;
}

/* 方法 2：使用 overflow */
.parent {
  overflow: hidden;
}

/* 方法 3：使用 display: flow-root */
.parent {
  display: flow-root;
}
```

<a id="11.2"></a>
### 11.2 垂直居中

```css
/* 方法 1：使用 Flexbox */
.flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}

/* 方法 2：使用 Grid */
.grid-center {
  display: grid;
  place-items: center;
  height: 100vh;
}

/* 方法 3：使用绝对定位 */
.absolute-center {
  position: relative;
  height: 100vh;
}

.absolute-center > div {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

## 12 . 最佳实践

<a id="12.1"></a>
### 12.1 代码组织

```css
/* 按功能组织 CSS */
/* 1. 重置样式 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* 2. 基础样式 */
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  line-height: 1.5;
  color: #333;
}

/* 3. 布局样式 */
.container {
  width: 100%;
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 20px;
}

/* 4. 组件样式 */
.btn {
  display: inline-block;
  padding: 10px 20px;
  background-color: #007bff;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

/* 5. 响应式样式 */
@media (max-width: 768px) {
  .container {
    padding: 0 10px;
  }
}
```

<a id="12.2"></a>
### 12.2 性能优化

```css
/* 1. 使用简写属性 */
/* 不好的做法 */
.box {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
}

/* 好的做法 */
.box {
  margin: 10px 20px;
}

/* 2. 避免使用过于复杂的选择器 */
/* 不好的做法 */
body > div > header > nav > ul > li > a {
  color: red;
}

/* 好的做法 */
.nav-link {
  color: red;
}

/* 3. 使用 CSS 变量 */
:root {
  --primary-color: #007bff;
  --secondary-color: #6c757d;
  --border-radius: 4px;
}

.btn {
  background-color: var(--primary-color);
  border-radius: var(--border-radius);
}
```

<a id="12.3"></a>
### 12.3 浏览器兼容性

```css
/* 使用浏览器前缀 */
.box {
  /* Chrome, Safari, Opera */
  -webkit-transform: rotate(45deg);
  /* Firefox */
  -moz-transform: rotate(45deg);
  /* IE */
  -ms-transform: rotate(45deg);
  /* 标准语法 */
  transform: rotate(45deg);
}

/* 渐变兼容性 */
.gradient {
  /* Chrome, Safari, Opera */
  background: -webkit-linear-gradient(red, blue);
  /* Firefox */
  background: -moz-linear-gradient(red, blue);
  /* IE */
  background: -ms-linear-gradient(red, blue);
  /* 标准语法 */
  background: linear-gradient(red, blue);
}
```
