## 1. 运算符分类 (Operator Categories)

### 1.1 算术运算符 (Arithmetic)
| 运算符 | 描述 | 示例 (a=10, b=3) |
|---|---|---|
| `+` | 加法 | `a + b = 13` |
| `-` | 减法 | `a - b = 7` |
| `*` | 乘法 | `a * b = 30` |
| `/` | 除法 | `a / b = 3` (整数除法舍去小数) |
| `%` | 取模 | `a % b = 1` |
| `++` | 自增 | `a++` 结果 10, a 变 11; `++a` 结果 11 |
| `--` | 自减 | `b--` 结果 3, b 变 2 |

### 1.2 关系运算符 (Relational)
返回 0 (False) 或 1 (True)。
- `==`, `!=`, `>`, `<`, `>=`, `<=`

### 1.3 逻辑运算符 (Logical)
- `&&`: 逻辑与 (And) - 具有短路特性。
- `||`: 逻辑或 (Or) - 具有短路特性。
- `!`: 逻辑非 (Not)

### 1.4 位运算符 (Bitwise)
直接对二进制位进行操作。
- `&`: 按位与 (AND)
- `|`: 按位或 (OR)
- `^`: 按位异或 (XOR)
- `~`: 按位取反 (NOT)
- `<<`: 左移 (乘 2 的幂)
- `>>`: 右移 (除 2 的幂)

### 1.5 赋值运算符 (Assignment)
- `=`, `+=`, `-=`, `*=`, `/=`, `%=`, `<<=`, `>>=`, `&=`, `^=`, `|=`

### 1.6 其他运算符
- `sizeof()`: 返回大小。
- `&`: 取地址运算符。
- `*`: 指针变量（解引用）。
- `? :`: 条件表达式（三目运算符）。
  ```c
  int max = (a > b) ? a : b;
  ```
- `,`: 逗号运算符（从左向右执行，返回最后一个表达式的值）。

## 2. 运算符优先级 (Precedence)
从高到低排列：
1. **最高**: `()`, `[]`, `->`, `.`
2. **单目**: `!`, `~`, `++`, `--`, `*`, `&`, `(type)`, `sizeof`
3. **算术**: `*`, `/`, `%` 然后 `+`, `-`
4. **位移**: `<<`, `>>`
5. **关系**: `>`, `>=`, `<`, `<=` 然后 `==`, `!=`
6. **位运算**: `&`, `^`, `|`
7. **逻辑**: `&&`, `||`
8. **三目**: `? :`
9. **赋值**: `=`, `+=` 等
10. **逗号**: `,`

## 3. 表达式 (Expressions)
表达式是由运算符和操作数组成的序列，计算后产生一个值。
- **注意**: 避免在复杂的表达式中多次修改同一个变量（未定义行为）。
