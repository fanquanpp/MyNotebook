## 3 . C 语言概述

C 语言是由 Dennis Ritchie 于 1972 年在贝尔实验室开发的，它具有以下特点：

- 简洁、高效
- 接近硬件，可直接操作内存
- 可移植性好
- 丰富的运算符和数据类型
- 强大的控制流语句
- 支持结构化编程

C 语言的应用领域包括：
- 操作系统开发（如 UNIX、Linux）
- 嵌入式系统
- 游戏开发
- 编译器和解释器
- 数据库系统
- 网络协议实现

## 4 . C 语言基本语法

<a id="4.1"></a>
### 4.1 程序结构

一个基本的 C 程序结构如下：

```c
#include <stdio.h>

// 主函数
int main() {
    // 代码语句
    printf("Hello, World!\n");
    return 0;
}
```

<a id="4.2"></a>
### 4.2 注释

```c
// 单行注释

/*
多行注释
可以跨越多行
*/
```

<a id="4.3"></a>
### 4.3 标识符

标识符是用来标识变量、函数、数组等的名称，必须遵循以下规则：
- 只能由字母、数字和下划线组成
- 不能以数字开头
- 区分大小写
- 不能使用 C 语言的关键字

<a id="4.4"></a>
### 4.4 关键字

C 语言的关键字包括：
- 数据类型：int, char, float, double, void
- 控制流：if, else, switch, case, default, for, while, do, break, continue, return
- 存储类：auto, register, static, extern, const, volatile
- 其他：struct, union, enum, typedef, sizeof

## 8 . 控制流

<a id="8.1"></a>
### 8.1 条件语句

```c
// if 语句
if (condition) {
    // 代码块
}

// if-else 语句
if (condition) {
    // 代码块 1
} else {
    // 代码块 2
}

// if-else if-else 语句
if (condition1) {
    // 代码块 1
} else if (condition2) {
    // 代码块 2
} else {
    // 代码块 3
}

// switch 语句
switch (expression) {
    case value1:
        // 代码块 1
        break;
    case value2:
        // 代码块 2
        break;
    default:
        // 代码块 3
        break;
}
```

<a id="8.2"></a>
### 8.2 循环语句

```c
// for 循环
for (initialization; condition; update) {
    // 代码块
}

// while 循环
while (condition) {
    // 代码块
}

// do-while 循环
do {
    // 代码块
} while (condition);

// 循环控制语句
break;    // 跳出循环
continue; // 跳过当前循环迭代
```

## 9 . 函数

<a id="9.1"></a>
### 9.1 函数定义

```c
// 函数声明
return_type function_name(parameter_list);

// 函数定义
return_type function_name(parameter_list) {
    // 函数体
    return return_value;
}
```

<a id="9.2"></a>
### 9.2 函数示例

```c
// 函数声明
int add(int a, int b);

// 函数定义
int add(int a, int b) {
    return a + b;
}

// 函数调用
int result = add(5, 3);
printf("Result: %d\n", result);
```

<a id="9.3"></a>
### 9.3 递归函数

```c
// 阶乘函数
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}
```

## 10 . 数组

<a id="10.1"></a>
### 10.1 一维数组

```c
// 数组定义
int arr[5]; // 未初始化
int arr[5] = {1, 2, 3, 4, 5}; // 初始化
int arr[] = {1, 2, 3, 4, 5}; // 自动计算大小

// 数组访问
arr[0] = 10; // 赋值
printf("%d\n", arr[0]); // 访问

// 数组遍历
for (int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}
printf("\n");
```

<a id="10.2"></a>
### 10.2 二维数组

```c
// 二维数组定义
int arr[2][3]; // 未初始化
int arr[2][3] = {{1, 2, 3}, {4, 5, 6}}; // 初始化

// 二维数组访问
arr[0][0] = 10; // 赋值
printf("%d\n", arr[0][0]); // 访问

// 二维数组遍历
for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 3; j++) {
        printf("%d ", arr[i][j]);
    }
    printf("\n");
}
```

## 11 . 指针

<a id="11.1"></a>
### 11.1 指针定义

```c
// 指针定义
int *ptr; // 整型指针
char *cptr; // 字符指针
float *fptr; // 浮点型指针

// 指针初始化
int a = 10;
int *ptr = &a; // 指向变量 a 的地址

// 指针解引用
printf("Value of a: %d\n", *ptr); // 输出 10

// 指针运算
ptr++; // 指针向后移动一个整型大小
```

<a id="11.2"></a>
### 11.2 指针与数组

```c
int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr; // 数组名是数组首元素的地址

// 通过指针访问数组元素
for (int i = 0; i < 5; i++) {
    printf("%d ", *(ptr + i));
}
printf("\n");
```

<a id="11.3"></a>
### 11.3 指针与函数

```c
// 传递指针给函数
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

// 函数返回指针
int *get_max(int *arr, int size) {
    int *max = arr;
    for (int i = 1; i < size; i++) {
        if (*(arr + i) > *max) {
            max = arr + i;
        }
    }
    return max;
}
```

## 12 . 结构体与联合体

<a id="12.1"></a>
### 12.1 结构体

```c
// 结构体定义
typedef struct {
    char name[50];
    int age;
    float salary;
} Employee;

// 结构体变量
Employee emp1;

// 结构体初始化
Employee emp2 = {"John", 30, 5000.50};

// 访问结构体成员
strcpy(emp1.name, "Alice");
emp1.age = 25;
emp1.salary = 4000.00;

printf("Name: %s, Age: %d, Salary: %.2f\n", emp1.name, emp1.age, emp1.salary);
```

<a id="12.2"></a>
### 12.2 联合体

```c
// 联合体定义
typedef union {
    int i;
    float f;
    char c;
} Data;

// 联合体变量
Data data;

// 联合体使用
data.i = 10;
printf("Integer value: %d\n", data.i);

data.f = 3.14;
printf("Float value: %.2f\n", data.f);

data.c = 'A';
printf("Character value: %c\n", data.c);
```

## 13 . 文件操作

<a id="13.1"></a>
### 13.1 文件打开与关闭

```c
// 文件指针
FILE *fp;

// 打开文件
fp = fopen("file.txt", "w"); // 写入模式
fp = fopen("file.txt", "r"); // 读取模式
fp = fopen("file.txt", "a"); // 追加模式
fp = fopen("file.txt", "r+"); // 读写模式

// 关闭文件
fclose(fp);
```

<a id="13.2"></a>
### 13.2 文件读写

```c
// 写入文件
fprintf(fp, "Hello, World!\n");
fputc('A', fp);
fputs("Hello", fp);

// 读取文件
char buffer[100];
fscanf(fp, "%s", buffer);
char c = fgetc(fp);
fgets(buffer, 100, fp);
```
