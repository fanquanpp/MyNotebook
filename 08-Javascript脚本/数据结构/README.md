## 1. 项目简介 | Introduction

本目录收录了使用 JavaScript 实现的核心数据结构，包括链表、栈、队列、树等常见数据结构的实现。

This directory contains JavaScript implementations of core data structures, including linked lists, stacks, queues, trees, and other common data structures.

| 单链表 (Singly Linked List) | [linked_list_js.js](./linked_list_js.js) | 基础 | 链表 | 支持插入、删除、遍历操作 |

## 3. 运行指南 | How to Run

```bash
# 运行数据结构示例
node linked_list_js.js
node stack_js.js
node queue_js.js
```

## 4. 核心特色 | Key Features

- **完整实现**：每个数据结构都有完整的实现和测试用例
- **详细注释**：代码中包含详细的中英文注释，便于理解
- **性能分析**：提供时间复杂度和空间复杂度分析
- **实用方法**：实现了常用的操作方法，满足实际应用需求

## 5. 学习建议 | Learning Guide

1. 从基础数据结构开始学习，如链表、栈和队列
2. 理解每种数据结构的基本原理和适用场景
3. 尝试自己实现这些数据结构，加深对它们的理解
4. 在实际项目中应用这些数据结构，提高代码效率
