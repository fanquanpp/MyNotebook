## 1. AnimationPlayer 节点 | AnimationPlayer Node

### 1.1 基本用法 | Basic Usage

1. 添加 `AnimationPlayer` 节点
2. 在编辑器中创建/编辑动画
3. 在代码中播放：

```gdscript
extends Node2D

@onready var anim: AnimationPlayer = $AnimationPlayer

func _ready():
    anim.play("Idle")

func run():
    anim.play("Run")
```

### 1.2 常用方法 | Common Methods

| 方法 | 说明 |
| :--- | :--- |
| `play(name)` | 播放指定动画 |
| `stop()` | 停止动画 |
| `is_playing()` | 是否在播放 |
| `current_animation` | 当前播放的动画名 |
| `advance(time)` | 推进动画时间 |

### 1.3 动画信号 | Animation Signals

| 信号 | 说明 |
| :--- | :--- |
| `animation_finished` | 动画播放完触发 |
| `animation_started` | 动画开始触发 |
| `animation_changed` | 动画改变触发 |

## 2. Tween (程序化动画) | Tween (Programmatic Animation)

在 Godot 4 中，Tween 更易用：

```gdscript
extends Node2D

func do_a_fancy_animation():
    var tween: Tween = create_tween()
    
    # 链式调用
    tween.tween_property(self, "position", Vector2(200, 200), 1.0)
    tween.tween_property(self, "rotation", PI, 1.0)
    tween.tween_callback(_on_tween_complete)

func _on_tween_complete():
    print("Tween finished")
```

### 2.1 设置过渡类型 | Setting Transitions

```gdscript
tween.set_ease(Tween.EASE_IN_OUT)
tween.set_trans(Tween.TRANS_SINE)
```

### 2.2 常见 Tween 方法 | Common Tween Methods

| 方法 | 说明 |
| :--- | :--- |
| `tween_property()` | 补间属性 |
| `tween_callback()` | 回调方法 |
| `tween_interval()` | 等待一段时间 |

## 3. 2D 精灵动画 (SpriteFrames + AnimatedSprite2D) | 2D Sprite Animation (SpriteFrames + AnimatedSprite2D)

### 3.1 SpriteFrames 资源 | SpriteFrames Resource

存放多个动画的资源。

### 3.2 AnimatedSprite2D 使用 | Using AnimatedSprite2D

```gdscript
extends AnimatedSprite2D

func _ready():
    play("Idle")

func _process(delta: float):
    if Input.get_axis("left", "right") != 0:
        if not is_playing() or animation != "Walk":
            play("Walk")
    else:
        if animation != "Idle":
            play("Idle")
```

## 4. 小结 | Summary

1. 使用 `AnimationPlayer` 播放编辑器里做的动画
2. 使用 `Tween` (`create_tween()`) 做代码动画
3. 使用 `AnimatedSprite2D` 做帧动画
4. 动画信号非常有用
