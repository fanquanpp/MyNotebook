## 1. Godot 4.x 物理节点 | Godot 4.x Physics Nodes

Godot 4.x 对物理系统做了重大更新，主要节点如下：

### 1.1 主要物理节点类型

| 节点类型 | 说明 | 用途 |
| :--- | :--- | :--- |
| `CharacterBody2D` | 角色物理体 | 玩家、敌人等可控制角色 |
| `RigidBody2D` | 刚体 | 受物理控制的物体（箱子、球） |
| `StaticBody2D` | 静态体 | 不会移动的物体（墙壁、地面） |
| `Area2D` | 区域 | 检测重叠/进入/离开 |

### 1.2 碰撞形状 | Collision Shapes

- `CollisionShape2D`：配合 Shape2D 资源使用
- `CollisionPolygon2D`：自由绘制多边形
- `CollisionPolygon2D` + `CollisionShape2D`：组合使用

## 2. CharacterBody2D (推荐用于玩家) | CharacterBody2D (Recommended for Players)

### 2.1 简单移动 | Simple Movement

```gdscript
extends CharacterBody2D

@export var speed: float = 300.0

func _physics_process(delta: float):
    var direction: Vector2 = Vector2.ZERO
    
    # 输入
    if Input.is_action_pressed("ui_left"):
        direction.x -= 1
    if Input.is_action_pressed("ui_right"):
        direction.x += 1
    if Input.is_action_pressed("ui_up"):
        direction.y -= 1
    if Input.is_action_pressed("ui_down"):
        direction.y += 1
    
    # 标准化以防对角线移动过快
    if direction.length() > 0:
        direction = direction.normalized()
    
    # 设置速度
    velocity = direction * speed
    
    # 移动
    move_and_slide()
```

### 2.2 平台游戏移动 | Platformer Movement

```gdscript
extends CharacterBody2D

const SPEED: float = 300.0
const JUMP_VELOCITY: float = -400.0

@onready var sprite: Sprite2D = $Sprite2D

func _physics_process(delta: float):
    # 重力
    if not is_on_floor():
        velocity.y += 1200.0 * delta
    
    # 跳跃
    if Input.is_action_just_pressed("jump") and is_on_floor():
        velocity.y = JUMP_VELOCITY
    
    # 左右移动
    var direction: float = Input.get_axis("ui_left", "ui_right")
    velocity.x = direction * SPEED
    
    if direction != 0:
        sprite.flip_h = direction < 0
    
    move_and_slide()
```

## 4. Area2D (用于检测) | Area2D (For Detection)

### 4.1 常用信号 | Common Signals

| 信号 | 说明 |
| :--- | :--- |
| `body_entered` | 另一个物理体进入 |
| `body_exited` | 另一个物理体离开 |
| `area_entered` | 另一个 Area2D 进入 |
| `area_exited` | 另一个 Area2D 离开 |

### 4.2 示例：金币拾取 | Example: Coin Pickup

```gdscript
extends Area2D

func _ready():
    body_entered.connect(_on_body_entered)

func _on_body_entered(body: Node):
    if body.is_in_group("player"):
        collect()

func collect():
    queue_free()
```

## 5. 碰撞检测 | Collision Detection

### 5.1 碰撞层/蒙版 | Collision Layers & Masks

在物理节点检查器中设置：
- **Collsion Layer**：这个物体在第几层
- **Collision Mask**：这个物体检测哪几层

### 5.2 常见配置 | Common Setups

| 物体类型 | Layer | Mask |
| :--- | :--- | :--- |
| 玩家 | 1 | 2, 3 |
| 墙壁 | 2 | 1 |
| 敌人 | 3 | 1 |

## 6. 射线与形状查询 | Ray & Shape Queries

### 6.1 射线检测 | Raycasting (RayCast2D)

```gdscript
extends Node2D

@onready var ray: RayCast2D = $RayCast2D

func _process(_delta: float):
    if ray.is_colliding():
        var collider: Node = ray.get_collider()
        print("Hit: ", collider.name)
```

### 6.2 形状检测 (ShapeQuery) | Shape Queries (PhysicsDirectSpaceState2D)

```gdscript
func _physics_process(delta: float):
    var space: PhysicsDirectSpaceState2D = get_world_2d().direct_space_state
    
    # 线段查询
    var query: PhysicsRayQueryParameters2D = PhysicsRayQueryParameters2D.create(
        global_position, global_position + Vector2.RIGHT * 100
    )
    var result: Dictionary = space.intersect_ray(query)
    
    if result:
        print("Hit at: ", result.position)
```

## 7. 小结 | Summary

1. 用 `CharacterBody2D` 做玩家和角色
2. 用 `RigidBody2D` 做物理对象
3. 用 `StaticBody2D` 做墙壁和地面
4. 用 `Area2D` 做碰撞检测（触发区域）
5. 使用碰撞层/蒙版优化性能
6. 用 `RayCast2D` 或 `direct_space_state` 做射线检测
