## 1. 音频节点 | Audio Nodes

### 1.1 主要音频节点 | Main Audio Nodes

| 节点 | 说明 |
| :--- | :--- |
| `AudioStreamPlayer2D` | 2D 位置音效 |
| `AudioStreamPlayer3D` | 3D 位置音效 |
| `AudioStreamPlayer` | 通用/背景音乐 |

---

## 2. 基本用法 | Basic Usage

```gdscript
extends Node

@onready var bgm: AudioStreamPlayer = $BGM
@onready var sfx_jump: AudioStreamPlayer2D = $SFX_Jump

func _ready():
    bgm.play()

func jump():
    sfx_jump.play()
```

### 2.1 常用属性 | Common Properties

| 属性 | 说明 |
| :--- | :--- |
| `stream` | 音频资源 |
| `volume_db` | 音量（分贝） |
| `pitch_scale` | 音调 |
| `playing` | 是否在播放 |

---

## 3. 音频总线 | Audio Buses

在底部面板的"音频"标签中可以配置总线：
- `Master`：主总线
- `Music`：音乐
- `SFX`：音效
- `Voice`：语音

在代码中设置音量：

```gdscript
AudioServer.set_bus_volume_db(AudioServer.get_bus_index("Master"), -10.0)
```

---

## 4. 2D/3D 位置音效 | 2D/3D Positional Audio

`AudioStreamPlayer2D` 和 `AudioStreamPlayer3D` 会根据与收听者的距离自动调整音量和立体声位置。

---

## 5. 小结 | Summary

1. 用 `AudioStreamPlayer` 播放背景音乐
2. 用 `AudioStreamPlayer2D`/`3D` 做位置音效
3. 使用音频总线控制不同类型的音量
