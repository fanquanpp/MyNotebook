## 2. Size Flags 与 Anchor | Size Flags & Anchor

### 2.1 Size Flags

控制控件如何分配空间：
- **Fill**：填满可用空间
- **Expand**：尝试获取尽可能多的空间
- **Shrink Center** / **Shrink End**：在分配的空间内居中或靠右

### 2.2 Anchor (锚点) | Anchor

决定控件相对于父级如何定位：
- `anchor_left` / `anchor_top` / `anchor_right` / `anchor_bottom`：
- 0.0 = 左边/上边，0.5 = 中间，1.0 = 右边/下边

## 3. 常见 UI 信号 | Common UI Signals

### 3.1 Button 信号 | Button Signals

```gdscript
extends Button

func _ready():
    pressed.connect(_on_pressed)
    button_up.connect(_on_button_up)

func _on_pressed():
    print("Button pressed")
```

### 3.2 LineEdit 信号 | LineEdit Signals

```gdscript
extends LineEdit

func _ready():
    text_changed.connect(_on_text_changed)
    text_submitted.connect(_on_text_submitted)

func _on_text_submitted(new_text: String):
    print("Entered: ", new_text)
```

## 4. 主题 (Theme) | Theme

### 4.1 使用自定义主题 | Using Custom Themes

1. 创建主题资源：`新建资源` → `Theme`
2. 设置各种控件的样式
3. 应用到根 UI 容器或整个项目

### 4.2 代码中设置主题 | Setting Theme in Code

```gdscript
extends Control

var custom_theme: Theme = preload("res://CustomTheme.tres")

func _ready():
    theme = custom_theme
```

## 6. UI 动画 | UI Animation

可以用 `Tween` 做简单动画：

```gdscript
extends Button

func _pressed():
    var tween: Tween = create_tween()
    tween.tween_property(self, "scale", Vector2(1.2, 1.2), 0.2)
    tween.tween_property(self, "scale", Vector2(1.0, 1.0), 0.2)
```

## 7. 小示例：简单的开始菜单 | Mini Example: Simple Start Menu

场景树：
```
- Control (CenterContainer)
   - VBoxContainer
      - Label ("My Game")
      - Button ("Start")
      - Button ("Options")
      - Button ("Exit")
```

```gdscript
extends CenterContainer

@onready var start_btn: Button = $VBoxContainer/Start
@onready var options_btn: Button = $VBoxContainer/Options
@onready var exit_btn: Button = $VBoxContainer/Exit

func _ready():
    start_btn.pressed.connect(_on_start_pressed)
    options_btn.pressed.connect(_on_options_pressed)
    exit_btn.pressed.connect(_on_exit_pressed)

func _on_start_pressed():
    get_tree().change_scene_to_file("res://Game.tscn")

func _on_options_pressed():
    print("Open options")

func _on_exit_pressed():
    get_tree().quit()
```

## 8. 小结 | Summary

1. 所有 UI 节点继承自 `Control`
2. 使用容器 (`VBoxContainer`, `HBoxContainer` 等) 自动排列
3. 使用 `Anchor` 和 `Size Flags` 控制大小和位置
4. 使用信号响应用户操作
5. 使用 `Theme` 统一风格
