## 1. 简介 | Introduction

Godot 是由 Juan Linietsky 和 Ariel Manzur 创建的免费开源游戏引擎。它使用 GDScript（类似 Python 的语言）、C# 或 C++ 进行开发，支持 2D 和 3D 游戏开发，拥有完整的编辑器和工具链。
