## 1. 变量和常量 | Variables & Constants

### 1.1 变量声明 | Variable Declaration

在 GDScript 中，使用 `var` 关键字声明变量：

```gdscript
var a = 10
var b = "Hello"
var c = 3.14
var d = true
```

GDScript 是动态类型语言，自动推断变量类型。

### 1.2 显式类型声明 | Explicit Type Declaration

你也可以显式指定类型，这有助于编译器进行类型检查和优化：

```gdscript
var health: int = 100
var name: String = "Player"
var speed: float = 5.5
var is_alive: bool = true
```

### 1.3 常量 | Constants

使用 `const` 关键字声明常量（在初始化后不能改变）：

```gdscript
const MAX_HEALTH: int = 100
const GAME_TITLE: String = "My Awesome Game"
```

常量名通常使用全大写加下划线（蛇形命名法）。

## 2. 基本数据类型 | Basic Data Types

### 2.1 整数 | Integer (int)

```gdscript
var level: int = 10
var score: int = 0
var enemies_killed: int = 5
```

### 2.2 浮点数 | Float (float)

```gdscript
var pi: float = 3.14159
```


| 类型 | 说明 |
| :--- | :--- |
| `void` | 表示无返回值 |

## 3. 类型转换 | Type Casting & Conversion

### 3.1 显式转换 | Explicit Conversion

```gdscript
var s: String = "123"
var i: int = int(s)  # 字符串转整数
var f: float = float(s)  # 字符串转浮点数

var num: int = 456
var str: String = str(num)  # 整数转字符串
```

### 3.2 检查类型 | Type Checking

```gdscript
var value: Variant = something

if value is int:
    print("It's an integer!")

if value is String:
    print("It's a string!")
```

## 5. 控制流 | Control Flow

### 5.1 if-elif-else

```gdscript
var health: int = 50

if health > 80:
    print("Perfect health!")
elif health > 50:
    print("Okay")
elif health > 20:
    print("Low health")
else:
    print("Critical!")
```

### 5.2 for 循环

```gdscript
# 范围循环
for i in range(0, 10):
    print(i)

# 遍历数组
var inventory: Array = ["Sword", "Shield", "Potion"]
for item in inventory:
    print("You have a ", item)

# 遍历字典
var stats: Dictionary = {"HP": 100, "MP": 50}
for key in stats:
    print(key, ": ", stats[key])
```

### 5.3 while 循环

```gdscript
var count: int = 0
while count < 5:
    print(count)
    count += 1
```

### 5.4 match 语句（类似 switch）

```gdscript
var day: int = 2

match day:
    1:
        print("Monday")
    2:
        print("Tuesday")
    3:
        print("Wednesday")
    _:
        print("Other day")
```

`_` 是默认分支，类似 `default`。

## 6. 函数基础 | Functions Basics

### 6.1 定义函数 | Defining a Function

使用 `func` 关键字：

```gdscript
func greet(name: String):
    print("Hello, ", name, "!")
```

### 6.2 带返回值的函数 | Return Values

```gdscript
func add(a: int, b: int) -> int:
    return a + b
```

返回类型在 `->` 后面指定。

### 6.3 调用函数 | Calling a Function

```gdscript
greet("Alice")
var sum: int = add(3, 5)
```

## 7. 注释 | Comments

### 7.1 单行注释 | Single-line Comments

```gdscript
# 这是一个注释
var health: int = 100  # 内联注释
```

### 7.2 多行注释 | Multi-line Comments

使用 `#` 开头，也可以用三个双引号（虽然 GDScript 没有官方的多行注释，但可以用字符串代替）：

```gdscript
# 这是第一行
# 这是第二行

# 或者
"""
这是一个字符串
可以作为多行注释
"""
```

## 8. 枚举 | Enums (Enumerations)

枚举用于定义一组命名常量：

```gdscript
enum GameState {
    MENU,
    PLAYING,
    PAUSED,
    GAME_OVER
}

var state: int = GameState.PLAYING

if state == GameState.PAUSED:
    print("Game is paused")
```

## 9. 常用内置函数 | Built-in Functions

### 9.1 输出到控制台 | Print

```gdscript
print("Hello!")
printt("Tab-separated", "values")
prints("Space-separated", "values")
```

### 9.2 随机数 | Random Numbers

```gdscript
rand_range(1, 10)        # 1 到 10 之间的随机浮点数
randi_range(1, 10)       # 1 到 10 之间的随机整数
```

### 9.3 数学函数 | Math Functions

```gdscript
sin(angle)
cos(angle)
tan(angle)
floor(2.5)  # 2
ceil(2.5)   # 3
round(2.5)  # 3
```

### 9.4 字符串函数 | String Functions

```gdscript
var s: String = "Hello"
s.length()        # 5
s.to_upper()      # "HELLO"
s.to_lower()      # "hello"
s.contains("ell") # true
```
