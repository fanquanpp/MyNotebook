## 1. 基础概念类

| 中文 | 英文 | 说明 |
| :--- | :--- | :--- |
| 节点 | Node | Godot 中最基本的构建块，所有东西都是节点 |
| 场景 | Scene | 节点的树形层级结构，可以保存为 .tscn/.scn 文件 |
| 场景树 | Scene Tree | 所有当前加载的节点组成的树，根是 Viewport/SceneTree |
| 实例化 | Instantiate | 根据场景资源创建出一个新的节点对象的过程 |
| 子节点 | Child Node | 在场景树中位于另一节点下方的节点 |
| 父节点 | Parent Node | 在场景树中包含其他节点的节点 |
| 信号 | Signal | Godot 的观察者模式实现，用于对象之间的通信 |
| 组 | Group | 节点的分组，用于批量操作或识别 |
| 资源 | Resource | 数据容器，比如 Texture、Audio、PackedScene 等 |

## 2. 脚本类

| 中文 | 英文 | 说明 |
| :--- | :--- | :--- |
| GDScript | GDScript | Godot 内置的类 Python 脚本语言 |
| C# | C# | Godot 支持的另一种主要脚本语言 |
| 扩展 | extends | 在脚本开头声明继承自哪个类，如 `extends Node2D` |
| 类名 | class_name | 在脚本开头给脚本起个类型名，方便其他脚本引用 |
| 注解 | annotation | 以 `@` 开头的关键字，如 `@onready`, `@export` |
| 就绪 | _ready | 节点生命周期回调，当节点进入场景树且子节点都就绪时调用 |
| 过程 | _process | 每帧调用的回调， delta 为距上一帧的时间 |
| 物理过程 | _physics_process | 每个物理帧调用的回调，用于物理相关逻辑 |

## 3. 节点类型类

| 中文 | 英文 | 说明 |
| :--- | :--- | :--- |
| 节点2D | Node2D | 2D 空间中的基类节点，有 position、rotation、scale |
| 角色身体2D | CharacterBody2D | 用于玩家或敌人的物理节点，推荐用于游戏角色 |
| 刚体2D | RigidBody2D | 受到物理引擎控制的2D物体 |
| 静态身体2D | StaticBody2D | 不会移动的物理体，比如墙壁、地面 |
| 区域2D | Area2D | 用于检测重叠、进入、离开的区域 |
| 精灵2D | Sprite2D | 显示 2D 纹理的节点 |
| 颜色矩形 | ColorRect | 显示纯色矩形的 UI/2D 节点 |
| 控制 | Control | UI 系统的基类 |
| 按钮 | Button | UI 按钮 |
| 标签 | Label | UI 文本标签 |

## 4. 编辑器类

| 中文 | 英文 | 说明 |
| :--- | :--- | :--- |
| 检查器 | Inspector | 右侧面板，显示选中节点/资源的属性 |
| 文件系统 | FileSystem | 左下角面板，显示项目文件 |
| 场景 | Scene | 左上角面板，显示当前场景的节点树 |
| 动画 | Animation | 底部面板，用于编辑动画 |
| 导入 | Import | 调整资源导入设置 |

## 5. 其他常见术语

| 中文 | 英文 | 说明 |
| :--- | :--- | :--- |
| 补间 | Tween | 一种用于平滑属性过渡的工具/类 |
| 锚点 | Anchor | UI 节点的定位基准点 |
| 容器 | Container | UI 中用于自动布局子节点的节点 |
| 主题 | Theme | UI 样式资源 |
| 射线检测 | Raycast | 发射一条线段来检测碰撞 |
| 碰撞层/蒙版 | Collision Layer/Mask | 物理系统中用于筛选检测对象的位掩码 |
| 总线程 | Bus | 音频系统中的混音通道 |
| 自动加载 | Autoload | 在项目设置中配置的单例，全局可用 |

## 6. 路径前缀

| 前缀 | 说明 | 示例 |
| :--- | :--- | :--- |
| res:// | 项目资源路径 | res://Player.tscn |
| user:// | 用户数据路径 | user://save.sav |
| /root/ | 场景树根 | /root/Main |

## 7. 文件扩展名

| 扩展名 | 说明 |
| :--- | :--- |
| .tscn | 文本格式的场景文件（推荐） |
| .scn | 二进制格式的场景文件 |
| .tres | 文本格式的资源文件（推荐） |
| .res | 二进制格式的资源文件 |
| .gd | GDScript 脚本文件 |
| .gdshader | Godot 着色器文件 |
