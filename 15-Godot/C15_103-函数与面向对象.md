## 1. 函数高级用法 | Advanced Functions

### 1.1 默认参数 | Default Arguments

```gdscript
func heal(amount: int = 10):
    health += amount
```

调用时可以不传该参数：
```gdscript
heal()       # 使用默认值 10
heal(50)     # 使用 50
```

### 1.2 可变参数 | Variable Arguments

使用 `...` 接受任意数量的参数：

```gdscript
func sum(...numbers):
    var total: int = 0
    for num in numbers:
        total += num
    return total

sum(1, 2, 3, 4, 5)  # 15
```

### 1.3 函数引用 | Function References

可以将函数赋值给变量，或者作为参数传递：

```gdscript
func add(a: int, b: int) -> int:
    return a + b

var func_ref: Callable = add
var result: int = func_ref.call(3, 5)  # 8
```

## 2. 类与对象 | Classes & Objects

### 2.1 定义类 | Defining a Class

在 GDScript 中，脚本本身就是一个类。通常继承自某个 Godot 类：

```gdscript
extends Node
class_name Player

var health: int = 100
var speed: float = 5.0

func take_damage(amount: int):
    health -= amount
    if health <= 0:
        die()

func die():
    print("Player died")
```

- `extends Node` 表示继承自 Node 类
- `class_name Player` 给这个类起个名字（可选，但推荐）

### 2.2 实例化对象 | Instantiating Objects

```gdscript
# 从场景实例化
var player = preload("res://Player.tscn").instantiate()
add_child(player)

# 或者从脚本实例化（如果脚本不依赖场景）
var enemy = Enemy.new()
enemy.name = "Goblin"
add_child(enemy)
```

### 2.3 构造函数与析构函数 | Constructors & Destructors

```gdscript
func _init():
    # 初始化逻辑，在实例创建时调用
    print("Object created")

func _ready():
    # 当节点进入场景树时调用（最常用）
    print("Node ready")

func _exit_tree():
    # 当节点离开场景树时调用
    print("Node leaving tree")

func _notification(what):
    # 处理各种通知
    if what == NOTIFICATION_PREDELETE:
        print("Object will be deleted")
```

## 3. 继承 | Inheritance

### 3.1 继承基础 | Inheritance Basics

一个类可以继承自另一个类，并获得其所有属性和方法：

```gdscript
class_name Character
extends Node

var health: int = 100
var damage: int = 10

func take_damage(amount: int):
    health -= amount
```

```gdscript
class_name Player
extends Character

var level: int = 1

func level_up():
    level += 1
    max_health += 10
    damage += 2
```

### 3.2 重写方法 | Overriding Methods

子类可以重写（Override）父类的方法：

```gdscript
class_name Enemy
extends Character

# 重写 take_damage
func take_damage(amount: int):
    # 调用父类方法（可选）
    super.take_damage(amount)
    
    # 额外的行为
    print("Enemy took damage!")
    if health > 0:
        become_angry()
```

## 4. 信号 | Signals

### 4.1 信号简介 | Signal Overview

信号是 Godot 中的观察者模式实现，允许一个对象通知其他对象发生了某件事。

### 4.2 定义信号 | Defining a Signal

```gdscript
extends Node
class_name HealthComponent

signal health_changed(new_health: int)
signal died()

var health: int = 100

func take_damage(amount: int):
    health -= amount
    health_changed.emit(health)  # 发射信号
    if health <= 0:
        died.emit()
```

### 4.3 连接信号 | Connecting Signals

在编辑器中连接（推荐）：
1. 选择发送信号的节点
2. 转到 "信号" 选项卡
3. 双击要连接的信号
4. 选择接收函数的节点和方法

在代码中连接：

```gdscript
extends Node2D

@onready var health_component = $HealthComponent

func _ready():
    # 方式一：通过 connect 连接
    health_component.health_changed.connect(_on_health_changed)
    
    # 方式二：通过 connect 带参数
    health_component.died.connect(_on_died)

func _on_health_changed(new_health: int):
    print("Health is now: ", new_health)

func _on_died():
    print("Game over")
```

使用 `@onready` 注解：
```gdscript
extends Node2D

@onready var health_component = $HealthComponent
```

## 5. 访问控制 | Access Control

### 5.1 私有成员 | Private Members

在 GDScript 4 中，以下划线 `_` 开头的成员表示私有（私有只是约定，并非强制）：

```gdscript
class_name Player

var health: int = 100        # 公开
var _internal_score: int = 0 # 私有（约定）

func take_damage(amount: int):
    health -= amount
    _update_score()

func _update_score():
    _internal_score += 10
```

### 5.2 公开成员 | Public Members

没有下划线前缀的是公开成员，可以在外部访问：

```gdscript
var player = Player.new()
player.health = 100
```

## 6. 多态 | Polymorphism

多态是面向对象编程的三大特性之一，在 Godot 中非常常见：

```gdscript
func _ready():
    var entities: Array = [
        Player.new(),
        Enemy.new(),
        NPC.new()
    ]
    
    for entity in entities:
        # 虽然是不同类型，但都有 update() 方法
        entity.update()
```

## 7. 静态成员与方法 | Static Members & Methods

使用 `static` 关键字声明：

```gdscript
class_name MathUtils

static var default_max: int = 100

static func clamp(value: int, min_val: int, max_val: int) -> int:
    if value < min_val:
        return min_val
    if value > max_val:
        return max_val
    return value

# 使用
var result: int = MathUtils.clamp(150, 0, 100)
```

## 10. 小结 | Summary

1. 每个 GDScript 脚本都是一个类
2. 使用 `extends` 继承另一个类
3. 使用 `class_name` 给类起个名字，方便其他脚本引用

6. 使用 `static` 声明静态成员和方法
