# Glossary | 技术名词术语表

按字母序统一全仓库技术名词译法。任何文档出现下述词汇必须引用本条目。

| 英 -> 中 | 中 -> 英 | 定义 / 备注 |
| :--- | :--- | :--- |
| API | 应用程序接口 | Application Programming Interface |
| AST | 抽象语法树 | Abstract Syntax Tree |
| ATL | 动画与变换语言 | Animation and Transformation Language |
| Boilerplate | 样板代码 | 需重复编写的标准化代码 |
| CLI | 命令行界面 | Command Line Interface |
| Decorator | 装饰器 | TypeScript/Python 等语言的元编程特性 |
| DSL | 领域特定语言 | Domain-Specific Language |
| Generic | 泛型 | 强类型语言中的参数化类型 |
| Middleware | 中间件 | 处理请求/响应的中间层逻辑 |
| Refactoring | 重构 | 在不改变外部行为的前提下优化内部结构 |
| Runtime | 运行时 | 程序执行的环境 |
| Serialization | 序列化 | 对象转为可存储/传输格式的过程 |
| Singleton | 单例模式 | 确保一个类只有一个实例的设计模式 |
| Snake Case | 蛇形命名法 | `snake_case` |
| Pascal Case | 大驼峰命名法 | `PascalCase` |
| Camel Case | 小驼峰命名法 | `camelCase` |
| Kebab Case | 短横线命名法 | `kebab-case` |
| Venv | 虚拟环境 | Python Virtual Environment |
| Webhook | 钩子 | 自动触发的 HTTP 回调 |
