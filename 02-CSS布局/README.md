## 1. 项目简介 | Introduction

本目录涵盖现代 Web 开发中的核心布局技术，重点介绍 Flexbox、Grid 及传统布局方案。

This module covers core layout techniques in modern web development, focusing on Flexbox, Grid, and traditional layout solutions.

## 3. 环境要求 | Environment Requirements

- **浏览器**：Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **开发工具**：VS Code, Sublime Text, 或任何文本编辑器
- **在线工具**：[Codepen](https://codepen.io/), [JSFiddle](https://jsfiddle.net/)

## 4. 快速开始 | Quick Start

1. 学习 `04-盒模型原理.md` 理解空间占用。
2. 配合 `06-Flexbox可视化指南.md` 在浏览器中实时测试样式。

## 5. 学习路线 | Learning Path

`盒模型` -> `选择器` -> `Flexbox` -> `Grid` -> `综合实战`

## 6. 核心特色 | Key Features

- **可视化指南**：提供 Flexbox 和 Grid 的可视化学习资源。
- **实战示例**：包含响应式布局、居中技巧等实用示例。
- **浏览器兼容性**：提供各布局技术的浏览器支持情况。
- **性能优化**：包含布局性能优化的最佳实践。
