## 3 . CSS 基础概念

CSS（Cascading Style Sheets）是一种用于描述 HTML 或 XML 文档样式的语言，它允许开发者控制网页的布局、颜色、字体、间距等视觉效果。

<a id="3.1"></a>
### 3.1 CSS3 核心特性
- 选择器增强（伪类、伪元素）
- 盒模型优化
- 弹性布局（Flexbox）
- 网格布局（Grid）
- 动画与过渡效果
- 颜色系统增强
- 字体与文本样式改进
- 响应式设计支持

## 4 . CSS 基本语法

<a id="4.1"></a>
### 4.1 语法结构

CSS 规则由选择器和声明块组成：

```css
选择器 {
  属性 1: 值 1;
  属性 2: 值 2;
  /* 注释 */
}
```

- **选择器**：指定要应用样式的 HTML 元素
- **声明块**：包含一个或多个样式声明
- **样式声明**：由属性和值组成，用冒号分隔，以分号结尾

<a id="4.2"></a>
### 4.2 CSS 注释

CSS 注释用于添加说明，不会影响样式效果：

```css
/* 这是单行注释 */

/*
这是多行注释
可以跨越多行
*/
```

## 5 . CSS 引入方式

<a id="5.1"></a>
### 5.1 内联样式

直接在 HTML 元素中使用`style`属性：

```html
<div style="color: red; font-size: 16px;">文本内容</div>
```

**特点**：
- 优先级最高
- 只影响单个元素
- 不利于维护

<a id="5.2"></a>
### 5.2 内部样式

在 HTML 文档头部使用`style`标签：

```html
<style>
  div {
    color: red;
    font-size: 16px;
  }
</style>
```

**特点**：
- 影响当前 HTML 文件中的所有匹配元素
- 优先级高于外部样式
- 适用于单个页面的样式

<a id="5.3"></a>
### 5.3 外部样式

使用`link`标签引入外部 CSS 文件：

```html
<link rel="stylesheet" href="style.css">
```

**特点**：
- 可以被多个页面共享
- 便于维护和管理
- 优先级低于内联和内部样式

<a id="5.4"></a>
### 5.4 @import

在 CSS 文件中导入其他 CSS 文件：

```css
@import url("other.css");
```

**特点**：
- 可以在 CSS 文件中组织样式
- 会增加 HTTP 请求
- 优先级低于外部样式

## 6 . CSS 优先级

CSS 优先级决定了当多个样式规则应用于同一元素时，哪个规则会被优先使用。

<a id="6.1"></a>
### 6.1 优先级规则

从高到低：
1. **!important 声明**：最高优先级
2. **内联样式**：使用 style 属性的样式
3. **ID 选择器**：以#开头的选择器
4. **类选择器、属性选择器、伪类选择器**：以.开头的选择器等
5. **元素选择器、伪元素选择器**：如 div、p 等
6. **通用选择器**：* 选择器

<a id="6.2"></a>
### 6.2 优先级计算

- **内联样式**：1000
- **ID 选择器**：100
- **类选择器、属性选择器、伪类选择器**：10
- **元素选择器、伪元素选择器**：1
- **通用选择器**：0

计算示例：
- `#header .nav a`：100 + 10 + 1 = 111
- `.container div`：10 + 1 = 11
- `div`：1

<a id="6.3"></a>
### 6.3 优先级规则应用

```css
/* 优先级：10 */
.nav {
  color: blue;
}

/* 优先级：100 */
#main-nav {
  color: red; /* 会覆盖上面的样式 */
}

/* 优先级：1000 */
<div id="main-nav" class="nav" style="color: green;">导航</div> /* 会覆盖上面的样式 */

/* 优先级：最高 */
.nav {
  color: purple !important; /* 会覆盖内联样式 */
}
```

## 7 . CSS 属性和值

<a id="7.1"></a>
### 7.1 属性

CSS 属性用于指定要设置的样式特性，如颜色、字体、间距等。

<a id="7.2"></a>
### 7.2 值

CSS 值用于指定属性的具体设置，如颜色值、长度值、百分比等。

<a id="7.3"></a>
### 7.3 常见属性值类型

| 类型 | 示例 | 描述 |
|------|------|------|
| 颜色 | `red`, `#ff0000`, `rgb(255,0,0)` | 颜色值 |
| 长度 | `10px`, `1em`, `50%` | 长度值 |
| 数字 | `1`, `0.5`, `2` | 数值 |
| 字符串 | `"Arial"`, `'sans-serif'` | 字符串 |
| 关键字 | `solid`, `block`, `center` | 预定义的关键字 |
| URL | `url("image.jpg")` | 资源路径 |
| 函数 | `rgb(255,0,0)`, `calc(100% - 20px)` | 函数表达式 |

<a id="8"></a>
## 8 . CSS 单位

<a id="8.1"></a>
### 8.1 绝对单位

| 单位 | 描述 |
|------|------|
| `px` | 像素 |
| `pt` | 点（1pt = 1/72 英寸） |
| `pc` | 派卡（1pc = 12pt） |
| `in` | 英寸 |
| `cm` | 厘米 |
| `mm` | 毫米 |

<a id="8.2"></a>
### 8.2 相对单位

| 单位 | 描述 |
|------|------|
| `em` | 相对于父元素的字体大小 |
| `rem` | 相对于根元素的字体大小 |
| `vw` | 视口宽度的 1% |
| `vh` | 视口高度的 1% |
| `vmin` | 视口宽度和高度中较小值的 1% |
| `vmax` | 视口宽度和高度中较大值的 1% |
| `%` | 相对于父元素的百分比 |
