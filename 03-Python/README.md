## 1. 简介 | Introduction

Python 由 Guido van Rossum 于 1991 年首次发布，是一种解释型、面向对象、动态类型的高级编程语言。其"优雅"、"明确"、"简单"的设计哲学使其成为最受欢迎的编程语言之一，在数据科学、机器学习、Web开发、自动化等领域有着广泛应用。
