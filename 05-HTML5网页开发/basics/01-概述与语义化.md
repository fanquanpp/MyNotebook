## 1. HTML5 概述 (Overview)
HTML5 是超文本标记语言 (HyperText Markup Language) 的第五次重大修改。它不仅是标记语言，更是一个完整的 Web 应用平台。

### 1.1 核心特性 (Key Features)
- **语义化标签**: 增强代码可读性与 SEO。
- **多媒体支持**: 原生 `<video>` 和 `<audio>`。
- **图形绘制**: `<canvas>` 和 SVG。
- **离线与存储**: LocalStorage, SessionStorage, IndexedDB。
- **设备访问**: 地理定位 (Geolocation)、摄像头、传感器。
- **新表单控件**: `date`, `color`, `range` 等。

## 2. 文档结构 (Document Structure)
```html
<!DOCTYPE html> <!-- 简洁的声明 -->
<html lang="zh-CN">
<head>
    <meta charset="UTF-8"> <!-- 默认编码 -->
    <title>HTML5 页面</title>
</head>
<body>
    <!-- 内容区域 -->
</body>
</html>
```

## 3. 语义化标签 (Semantic Tags)
**核心思想**: 用正确的标签做正确的事。
- `<header>`: 头部。
- `<nav>`: 导航。
- `<main>`: 主要内容。
- `<article>`: 独立文章。
- `<section>`: 区块。
- `<aside>`: 侧边栏。
- `<footer>`: 底部。

## 4. 优势
- **SEO 友好**: 搜索引擎能更好地理解页面结构。
- **无障碍性 (Accessibility)**: 屏幕阅读器更容易解析。
- **开发效率**: 减少对 `div` 的滥用。
