# HTML5 基础标签与全局属性 (Basic Tags & Global Attributes)

> @Author: Anonymous
> @Category: HTML5 Basics
> @Description: 常用 HTML 标签、超链接、图像及全局属性详解。 | Common tags, links, images, and global attributes.

## 1. 基础文本标签 (Text Tags)
- `<h1>` - `<h6>`: 标题。
- `<p>`: 段落。
- `<span>`: 行内文本容器。
- `<strong>` / `<em>`: 加粗/倾斜 (带语义)。
- `<br>`: 换行。
- `<hr>`: 分割线。

## 2. 列表标签 (Lists)
- **无序列表**: `<ul>` + `<li>`。
- **有序列表**: `<ol>` + `<li>`。
- **定义列表**: `<dl>` + `<dt>` (术语) + `<dd>` (描述)。

## 3. 超链接与多媒体 (Media)
- **超链接**: `<a href="url" target="_blank">链接文字</a>`。
- **图像**: `<img src="url" alt="描述文字" width="300">`。
  - **注意**: `alt` 属性对 SEO 和无障碍至关重要。

## 4. 全局属性 (Global Attributes)
几乎所有 HTML 元素都支持的属性：
- **`id`**: 唯一标识符。
- **`class`**: 样式类名。
- **`style`**: 行内样式。
- **`title`**: 悬停提示文字。
- **`data-*`**: 自定义数据属性 (JS 访问方便)。
- **`hidden`**: 隐藏元素。
- **`contenteditable`**: 使元素内容可编辑。
