# MyNotebook

## 项目简介

MyNotebook 是一个学习笔记仓库，包含 Git 版本控制和 Markdown 语法的学习资料。本仓库旨在为学习者提供简洁明了的技术文档，帮助快速掌握相关技能。

## 目录结构

```
MyNotebook/
├── README.md            # 项目说明文档
├── md-git基础.md        # Git 基础操作指南
├── md-Git命令手册.md    # Git 命令参考手册
├── md-Markdown基础.md   # Markdown 语法指南
├── hm-git基础.html      # Git 基础操作指南（HTML 版本）
├── hm-Git命令手册.html  # Git 命令参考手册（HTML 版本）
└── hm-Markdown基础.html # Markdown 语法指南（HTML 版本）
```

## 文件说明

| 文件名 | 内容说明 |
| :--- | :--- |
| `md-git基础.md` | Git 基础操作指南，包括环境配置、仓库初始化、代码提交等基础流程 |
| `md-Git命令手册.md` | Git 命令参考手册，详细列举常用 Git 命令及其用法 |
| `md-Markdown基础.md` | Markdown 语法指南，介绍 Markdown 的基本语法和使用技巧 |
| `README.md` | 项目说明文档，提供仓库概览和文件索引 |

## 学习路径

1. **Markdown 基础**：先学习 Markdown 语法，掌握文档编写技能
2. **Git 基础**：了解 Git 的基本概念和操作流程
3. **Git 命令手册**：参考常用 Git 命令，解决实际操作问题

## 如何使用

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/fanquanpp/MyNotebook.git
   ```

2. 打开对应的 Markdown 文件进行学习
3. 可以使用 Markdown 编辑器（如 VSCode、Typora 等）查看和编辑文档
