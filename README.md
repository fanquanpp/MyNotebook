## 项目简介

MyNotebook 是 fanquanpp 的个人学习笔记仓库。本仓库旨在为学习者提供简洁明了、结构清晰的技术文档，帮助快速掌握相关技能，建立良好的版本控制和文档编写习惯。

### 项目目标

- 提供系统化的 Git 学习资料，从基础操作到高级命令
- 提供全面的 Markdown 语法指南，帮助编写规范的技术文档
- 提供详细的 Java 学习资料，从基础语法到高级特性
- 提供全面的 HTML5 学习资料，从基础标签到布局技巧
- 建立清晰的学习路径，便于不同水平的学习者参考
- 保持文档的简洁性和实用性，注重实际应用场景

### 项目特色

- **结构清晰**：文档按照逻辑顺序组织，便于查找和学习
- **内容全面**：涵盖 Git、Markdown、Java 和 HTML5 的核心知识点和常用技巧
- **示例丰富**：提供大量实用的代码示例和使用场景
- **持续更新**：根据学习进度和实际需求不断完善内容

## 学习路径

1. **Markdown 基础**：先学习 Markdown 语法，掌握文档编写技能
2. **Git 基础**：了解 Git 的基本概念和操作流程
3. **Git 命令手册**：参考常用 Git 命令，解决实际操作问题
4. **HTML5 基础**：学习 HTML5 的基础标签和结构
5. **Java 基础入门**：了解 Java 的基本概念和程序结构
6. **Java 基础语法**：掌握 Java 的数据类型和基本语法
7. **Java 运算符与表达式**：学习 Java 的各种运算符
8. **Java 流程控制**：掌握 Java 的流程控制语句
9. **Java 数组**：学习 Java 数组的使用方法

## 适用人群

本仓库适合以下人群使用：

- **Git 初学者**：希望系统学习 Git 版本控制的基本概念和操作流程
- **Markdown 学习者**：想要掌握 Markdown 语法，提高文档编写效率
- **Java 初学者**：希望系统学习 Java 编程语言的基础知识
- **HTML5 初学者**：想要学习 HTML5 的基础标签和结构
- **开发人员**：需要一个技术参考手册，快速查询常用命令和语法
- **学生**：学习版本控制、文档编写、Java 和 HTML5 的基础知识
- **教师**：寻找 Git、Markdown、Java 和 HTML5 的教学资料

## 如何使用

### 方法一：直接查看

1. 在 GitHub 仓库页面直接查看 Markdown 文件
2. 点击文件名即可在线预览内容

### 方法二：克隆到本地

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/fanquanpp/MyNotebook.git
   ```
2. 打开对应的 Markdown 文件进行学习
3. 使用 Markdown 编辑器查看和编辑文档：
   - **VSCode**：安装 Markdown 扩展后使用
   - **Typora**：所见即所得的 Markdown 编辑器
   - **Sublime Text**：安装 MarkdownPreview 插件
   - **Atom**：内置 Markdown 支持

## 操作规范

### 提交(commit)标注格式

为了保持仓库提交历史的清晰和一致性，我们规定以下提交标注格式：

- **text：文件结构优化相关** - 用于文件结构调整类提交
  - 示例：`text：调整文档目录结构`
  - 适用场景：添加、删除、重命名文件或目录，调整文件组织方式等
- **note：笔记内容优化相关** - 用于笔记内容更新类提交
  - 示例：`note：更新 Git 命令手册内容`
  - 适用场景：修改笔记内容，添加新知识点，优化文档结构等

### 版本上传时间规范

- 每次修改文件后，请更新文件顶部的"最新版本上传时间"字段
- 格式统一为：`最新版本上传时间：YYYY-MM-DD HH:MM:SS`
- 时间应使用修改完成的实际时间
