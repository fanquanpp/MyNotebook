## 项目简介

个人学习笔记仓库，用于存储和管理技术学习资料。

### 仓库内容

- **Git**：版本控制工具的使用指南和命令手册
- **Markdown**：标记语言的语法指南和最佳实践
- **Java**：编程语言的基础语法和核心概念
- **HTML5**：网页标记语言的基础标签和结构
- **CSS3**：网页样式语言的选择器、布局和动画效果
- **MySQL**：关系型数据库的基本操作和SQL语法

### 组织特点

- **结构清晰**：文档按技术领域分类，便于查找和管理
- **内容实用**：注重实际应用场景和常见问题解决
- **持续更新**：根据学习进度和实际需求不断完善
- **个人化**：记录个人学习过程和实践经验

## 使用方法

### 本地查看

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/fanquanpp/MyNotebook.git
   ```
2. 使用 Markdown 编辑器打开文件：
   - **VSCode**：安装 Markdown 扩展
   - **Typora**：所见即所得编辑器
   - **Sublime Text**：安装 MarkdownPreview 插件

### 在线查看

直接在 GitHub 仓库页面查看 Markdown 文件内容
