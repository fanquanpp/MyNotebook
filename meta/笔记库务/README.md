## 1. 项目简介 | Introduction
本目录存放仓库级元数据、维护规范、术语对照及写作模板。它是保证笔记库长期“可检索、可维护”的核心基石。
This module contains warehouse-level metadata, maintenance standards, terminology, and writing templates for long-term maintainability.

## 3. 环境要求 | Environment Requirements
- **文本编辑器**：推荐 VS Code + Markdown All in One 插件。
- **Lint 校验**：markdownlint v0.32+。
- **拼写检查**：Code Spell Checker (EN)。

## 4. 快速开始 | Quick Start
```bash
# 1. 阅读规范
# 访问: [笔记目录规范](./笔记目录规范.md)

# 2. 复制模板创建新笔记
cp 笔记模板.md ../XX-分册/XX-新笔记.md
```

## 5. 贡献指南 | Contribution Guide
- **风格一致性**：中文全角标点，英文半角标点。
- **链接引用**：外部链接必须带 `<!-- nofollow -->` 注释。
- **更新记录**：每篇笔记末尾必须包含 Changelog。
