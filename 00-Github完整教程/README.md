## 背景与定位

**GitHub** 是全球使用最广泛的代码协作平台之一，在 **Git（分布式版本控制系统）** 之上提供 Web 界面、权限管理、代码审查与自动化工作流。本教程与仓库内 [`04-Git版本控制`](../04-Git版本控制/README.md) 互补：后者讲解 Git 命令与核心概念，本系列专注于 **GitHub 网页端操作与平台特有能力**。
