## 背景与定位

**GitHub** 是全球使用最广泛的代码协作平台之一，在 **Git（分布式版本控制系统）** 之上提供 Web 界面、权限、审查与自动化。本教程与仓库内 [`04-Git版本控制`](../04-Git版本控制/README.md) 互补：后者讲命令与概念，本系列讲 **GitHub 网页端与平台能力**。
