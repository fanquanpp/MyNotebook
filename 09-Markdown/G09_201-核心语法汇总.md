
# Markdown 核心知识点


## 1 . Markdown 概述

Markdown 是一种轻量级标记语言，由 John Gruber 和 Aaron Swartz 于 2004 年创建。它允许人们使用易读易写的纯文本格式编写文档，然后可以转换为 HTML、PDF 等格式。


### 1.1 Markdown 的核心特点
- **简单易用**：语法简洁，易于学习和使用
- **跨平台**：可以在任何文本编辑器中编写
- **可转换**：可以转换为多种格式，如 HTML、PDF 等
- **可读性强**：即使不转换，纯文本也具有良好的可读性
- **广泛支持**：被 GitHub、Stack Overflow 等平台广泛使用


### 1.2 Markdown 的应用场景
- **文档编写**：技术文档、API 文档等
- **博客写作**：许多博客平台支持 Markdown
- **代码注释**：用于代码库的 README 文件
- **版本控制**：与 Git 等版本控制系统配合使用
- **在线协作**：在 GitHub、GitLab 等平台上进行协作


## 2 . 标题语法


### 2.1 基本标题

使用`#`符号创建标题，`#`的数量表示标题级别。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```


### 2.2 标题样式

- **Setext 样式**：使用`=`和`-`创建一级和二级标题

```markdown
一级标题
========

二级标题
--------
```


## 3 . 段落与换行


### 3.1 段落

段落之间使用空行分隔。

```markdown
这是第一个段落。

这是第二个段落。
```


### 3.2 换行

- 在行尾添加两个或更多空格，然后按 Enter 键
- 使用 HTML 的`<br>`标签

```markdown
这是第一行  
这是第二行

这是第一行<br>
这是第二行
```


## 4 . 基础文本格式


### 4.1 粗体

使用`**`或`__`包裹文本。

```markdown
**粗体文本**
__粗体文本__
```


### 4.2 斜体

使用`*`或`_`包裹文本。

```markdown
*斜体文本*
_斜体文本_
```


### 4.3 粗斜体

使用`***`或`___`包裹文本。

```markdown
***粗斜体文本***
___粗斜体文本___
```


### 4.4 删除线

使用`~~`包裹文本。

```markdown
~~删除线文本~~
```


### 4.5 行内代码

使用`` ` ``包裹代码。

```markdown
`行内代码`
```


### 4.6 代码块

使用三个反引号```包裹代码块，可指定语言。

```markdown
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
```


## 5 . 列表语法


### 5.1 无序列表

使用`-`、`*`或`+`创建无序列表。

```markdown
- 项目 1
- 项目 2
- 项目 3

* 项目 1
* 项目 2
* 项目 3

+ 项目 1
+ 项目 2
+ 项目 3
```


### 5.2 有序列表

使用数字加`.`创建有序列表。

```markdown
1. 项目 1
2. 项目 2
3. 项目 3
```


### 5.3 嵌套列表

在列表项前添加缩进创建嵌套列表。

```markdown
- 一级列表项
  - 二级列表项
    - 三级列表项

1. 一级列表项
   1. 二级列表项
      1. 三级列表项
```


## 6 . 链接


### 6.1 行内链接

```markdown
[链接文本](链接地址 "可选标题")

示例：
[Google](https://www.google.com "Google 搜索")
```


### 6.2 参考链接

```markdown
[链接文本][参考标签]

[参考标签]: 链接地址 "可选标题"

示例：
[Google][1]

[1]: https://www.google.com "Google 搜索"
```


### 6.3 自动链接

使用尖括号包围 URL 或邮箱地址。

```markdown
<https://www.google.com>
<example@example.com>
```

## 什么是异步编程？

异步编程是一种编程范式，允许程序在等待某个操作完成时继续执行其他任务。

## 异步编程的实现方式

### 1. 回调函数

```javascript
function fetchData(callback) {
  setTimeout(() => {
    callback('数据');
  }, 1000);
}

fetchData(data => {
  console.log(data);
});
```

### 2. Promise

```javascript
function fetchData() {
  return new Promise(resolve => {
    setTimeout(() => {
      resolve('数据');
    }, 1000);
  });
}

fetchData().then(data => {
  console.log(data);
});
```

### 3. async/await

```javascript
async function fetchData() {
  return new Promise(resolve => {
    setTimeout(() => {
      resolve('数据');
    }, 1000);
  });
}

async function main() {
  const data = await fetchData();
  console.log(data);
}

main();
```
```

**解决方案**：

- 使用一致的换行和缩进
- 避免不必要的空行
- 使用自动格式化工具

### 29.4 性能问题

**问题**：大型 Markdown 文件渲染缓慢
**解决方案**：
- 拆分文件
- 优化图片
- 使用预渲染
- 减少嵌套
