## 1. 简介 | Introduction

Markdown 由 John Gruber 于 2004 年创建，是一种轻量级标记语言。它允许人们使用易读易写的纯文本格式编写文档，然后转换为有效的 HTML。Markdown 广泛应用于 GitHub README、技术文档、博客文章等场景。
