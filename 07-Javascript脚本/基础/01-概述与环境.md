## 1. JavaScript 概述 (Overview)
JavaScript 是一门具有函数优先特性的轻量级、解释型或即时编译型 (JIT) 的编程语言。最初由 **Brendan Eich** 在 1995 年为 Netscape 浏览器设计。

### 1.1 核心特点 (Key Features)
- **单线程**: 配合事件循环 (Event Loop) 处理并发。
- **动态类型**: 变量无需显式声明类型。
- **函数式编程**: 函数是一等公民 (First-class Citizens)。
- **原型继承**: 基于原型的对象模型。
- **跨平台**: 浏览器、服务器 (Node.js)、移动端 (React Native)、桌面端 (Electron)。

## 2. 运行环境 (Runtime)
1. **浏览器 (Browser)**: 最早的运行环境，拥有 DOM 和 BOM API。
2. **Node.js**: 服务端运行环境，拥有文件系统、网络等底层 API。

## 3. 环境配置 (Environment Setup)
- **浏览器**: 无需配置，所有现代浏览器均内置引擎 (如 Chrome 的 V8)。
- **Node.js**: 
  - 下载: [nodejs.org](https://nodejs.org/)。
  - 验证: `node -v`, `npm -v`。
- **编辑器**: 推荐 **VS Code** 配合相关插件。

## 4. 发展标准 (ECMAScript)
- **ES5 (2009)**: 基础标准。
- **ES6 / ES2015**: 重大变革，引入了 `let/const`, `Promise`, `Class`, 箭头函数。
- **ES2016+**: 每年一个小版本更新。
