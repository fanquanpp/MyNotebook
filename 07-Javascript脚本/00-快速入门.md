# 00-快速入门 (JavaScript)

## 1. 引言
JavaScript (JS) 是一种具有函数优先特性的轻量级、解释型或即时编译型编程语言。它是 Web 的核心技术之一，广泛用于网页交互、服务器端开发 (Node.js) 和移动应用。

## 2. 环境搭建
### 2.1 浏览器
- 现代浏览器 (Chrome, Firefox, Safari) 均内置 JS 引擎。
- 按 `F12` 打开开发者工具，在 `Console` 面板中直接运行 JS。

### 2.2 Node.js
- 下载并安装 [Node.js](https://nodejs.org/)。
- 验证安装: `node -v`, `npm -v`。

## 3. Hello World 示例
创建一个名为 `hello.js` 的文件：
```javascript
console.log("Hello, JavaScript!");
```
**运行**:
```bash
node hello.js
```

## 4. 核心工作流
1. **编写**: 使用 `.js` 后缀。
2. **调试**: 使用浏览器开发者工具或 VS Code 调试功能。
3. **部署**: 前端通过 HTML 引入，后端通过 Node.js 运行。
