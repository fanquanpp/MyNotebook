## 1. 简介 | Introduction

MySQL 是由瑞典 MySQL AB 公司开发的开源关系型数据库管理系统，现在属于 Oracle 公司。它是最流行的数据库之一，被广泛应用于 Web 应用开发（如 WordPress、Facebook 早期版本）、企业级应用等领域。
