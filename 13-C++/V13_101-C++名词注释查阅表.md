# C++-专有名词注释查阅表 | C++

## 1. 基础概念类

### 1.1 C++

**名称**：C++ 编程语言（C++）

**首次出现位置**：C13_101-概述与标准.md 第1章

**定义**：
C++ 是由 Bjarne Stroustrup 于 1979 年在贝尔实验室开发的编译型、静态类型、多范式编程语言，设计目标是在提供与 C 语言兼容的同时支持面向对象编程和泛型编程。

### 1.2 编译型语言

**名称**：编译型语言（Compiled Language）

**首次出现位置**：C13_101-概述与标准.md 第1章

**定义**：
编译型语言是指代码在运行前需要通过编译器将源代码转换为机器码的语言，C++ 是典型的编译型语言。

### 1.3 静态类型

**名称**：静态类型（Static Typing）

**首次出现位置**：C13_101-概述与标准.md 第1章

**定义**：
静态类型是指变量的类型在编译时确定，类型检查在编译阶段完成，可以提前发现类型错误。

### 1.4 多范式编程

**名称**：多范式编程（Multi-paradigm Programming）

**首次出现位置**：C13_101-概述与标准.md 第1章

**定义**：
多范式编程是指一种语言支持多种编程范式，C++ 支持过程式、面向对象和泛型编程。

### 1.5 零开销抽象

**名称**：零开销抽象（Zero-overhead Abstractions）

**首次出现位置**：C13_101-概述与标准.md 第1.1章

**定义**：
零开销抽象是 C++ 的核心原则之一，指抽象不应该带来运行时开销，只为使用的功能付出代价。

### 1.6 RAII

**名称**：资源获取即初始化（Resource Acquisition Is Initialization）

**首次出现位置**：C13_103-指针引用与内存管理.md

**定义**：
RAII 是 C++ 的核心技术，通过构造函数获取资源、析构函数释放资源来管理资源，确保资源正确释放。

## 3. 类型系统类

### 3.1 基本数据类型

**名称**：基本数据类型（Fundamental Data Types）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
C++ 的基本数据类型包括整型（int、short、long）、浮点型（float、double）、字符型（char）、布尔型（bool）等。

### 3.2 指针

**名称**：指针（Pointer）

**首次出现位置**：C13_103-指针引用与内存管理.md 第1章

**定义**：
指针是存储另一个变量内存地址的变量，通过指针可以直接访问和操作内存。

### 3.3 引用

**名称**：引用（Reference）

**首次出现位置**：C13_103-指针引用与内存管理.md 第2章

**定义**：
引用是变量的别名，通过引用可以间接访问变量，但不能为空，且不能重新绑定。

### 3.4 const 修饰符

**名称**：常量修饰符（Const Modifier）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
const 修饰符用于定义常量，表示变量的值在定义后不能被修改。

### 3.5 constexpr

**名称**：常量表达式（Constant Expression）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
constexpr 用于声明编译时可计算的常量或函数，在编译时进行求值。

### 3.6 auto

**名称**：自动类型推导（Auto）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
auto 是 C++11 引入的关键字，用于自动推导变量的类型，简化类型声明。

### 3.7 decltype

**名称**：类型推导（Decltype）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
decltype 是 C++11 引入的关键字，用于推导表达式的类型并获取其类型信息。

### 3.8 结构体

**名称**：结构体（Struct）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
结构体是 C++ 中用于封装不同类型数据的自定义类型，与类的区别是默认成员访问权限不同。

### 3.9 共用体

**名称**：共用体（Union）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
共用体是一种特殊的数据类型，所有成员共享同一段内存空间，大小由最大成员决定。

### 3.10 枚举

**名称**：枚举（Enum）

**首次出现位置**：C13_102-基础语法与类型系统.md

**定义**：
枚举是一组命名的整型常量，用于定义一组相关的常量值。

## 4. 面向对象类

### 4.1 类

**名称**：类（Class）

**首次出现位置**：C13_104-面向对象.md 第1章

**定义**：
类是 C++ 面向对象编程的核心，用于封装数据和操作数据的函数，提供对象的蓝图。

### 4.2 对象

**名称**：对象（Object）

**首次出现位置**：C13_104-面向对象.md 第1章

**定义**：
对象是类的实例，拥有状态（属性）和行为（方法）。

### 4.3 封装

**名称**：封装（Encapsulation）

**首次出现位置**：C13_104-面向对象.md 第2章

**定义**：
封装是将数据和操作数据的方法捆绑在一起，并对外部隐藏实现细节的机制。

### 4.4 继承

**名称**：继承（Inheritance）

**首次出现位置**：C13_104-面向对象.md 第3章

**定义**：
继承是子类复用父类数据和行为的机制，子类可以新增或重写父类成员。

### 4.5 多态

**名称**：多态（Polymorphism）

**首次出现位置**：C13_104-面向对象.md 第4章

**定义**：
多态是指同一接口的不同实现方式，C++ 通过虚函数和重写实现运行时多态。

### 4.6 构造函数

**名称**：构造函数（Constructor）

**首次出现位置**：C13_104-面向对象.md 第5章

**定义**：
构造函数是类的一种特殊成员函数，在创建对象时自动调用，用于初始化对象。

### 4.7 析构函数

**名称**：析构函数（Destructor）

**首次出现位置**：C13_104-面向对象.md 第5章

**定义**：
析构函数是类的一种特殊成员函数，在对象销毁时自动调用，用于释放资源。

### 4.8 虚函数

**名称**：虚函数（Virtual Function）

**首次出现位置**：C13_104-面向对象.md 第4章

**定义**：
虚函数是在基类中声明为 virtual 的函数，可以在派生类中重写，实现运行时多态。

### 4.9 纯虚函数

**名称**：纯虚函数（Pure Virtual Function）

**首次出现位置**：C13_104-面向对象.md 第4章

**定义**：
纯虚函数是在基类中声明为 virtual 并初始化为 0 的函数，含有纯虚函数的类为抽象类。

### 4.10 抽象类

**名称**：抽象类（Abstract Class）

**首次出现位置**：C13_104-面向对象.md 第4章

**定义**：
抽象类是含有至少一个纯虚函数的类，不能直接实例化，只能作为基类使用。

### 4.11 访问修饰符

**名称**：访问修饰符（Access Specifiers）

**首次出现位置**：C13_104-面向对象.md 第1章

**定义**：
C++ 提供三种访问修饰符：public（公共）、protected（受保护）、private（私有），控制成员的访问权限。

### 4.12 友元

**名称**：友元（Friend）

**首次出现位置**：C13_104-面向对象.md

**定义**：
友元机制允许特定函数或类访问另一个类的私有成员，破坏了封装性但提供了灵活性。

### 4.13 静态成员

**名称**：静态成员（Static Members）

**首次出现位置**：C13_104-面向对象.md

**定义**：
静态成员属于类本身而不是类的实例，可以通过类名直接访问，包括静态数据成员和静态成员函数。

### 4.14 this 指针

**名称**：this指针（This Pointer）

**首次出现位置**：C13_104-面向对象.md

**定义**：
this 指针是类成员函数中指向当前对象实例的隐式指针，用于访问成员数据和调用其他成员函数。

## 5. 模板与 STL 类

### 5.1 模板

**名称**：模板（Template）

**首次出现位置**：C13_105-模板与STL.md 第1章

**定义**：
模板是 C++ 的泛型编程机制，允许编写与类型无关的代码，实例化时根据参数类型生成具体代码。

### 5.2 函数模板

**名称**：函数模板（Function Template）

**首次出现位置**：C13_105-模板与STL.md 第1.1章

**定义**：
函数模板是定义泛型函数的模板，实例化时根据参数类型生成具体的函数实现。

### 5.3 类模板

**名称**：类模板（Class Template）

**首次出现位置**：C13_105-模板与STL.md 第1.2章

**定义**：
类模板是定义泛型类的模板，实例化时根据模板参数生成具体的类。

### 5.4 STL

**名称**：标准模板库（Standard Template Library）

**首次出现位置**：C13_105-模板与STL.md 第2章

**定义**：
STL 是 C++ 的标准库的一部分，提供了容器、迭代器、算法和函数对象等组件。

### 5.5 容器

**名称**：容器（Container）

**首次出现位置**：C13_105-模板与STL.md 第2.1章

**定义**：
容器是 STL 提供的数据结构，如 vector、list、map、set 等，用于存储和管理元素集合。

### 5.6 迭代器

**名称**：迭代器（Iterator）

**首次出现位置**：C13_105-模板与STL.md 第2.2章

**定义**：
迭代器是 STL 中用于遍历容器元素的对象，提供了类似指针的统一访问接口。

### 5.7 算法

**名称**：算法（Algorithm）

**首次出现位置**：C13_105-模板与STL.md 第2.3章

**定义**：
算法是 STL 提供的常用算法实现，如 sort、find、transform、accumulate 等。

### 5.8 vector

**名称**：向量容器（Vector）

**首次出现位置**：C13_105-模板与STL.md 第2.1章

**定义**：
vector 是动态数组容器，支持随机访问，在末尾插入删除效率高。

### 5.9 map

**名称**：映射容器（Map）

**首次出现位置**：C13_105-模板与STL.md 第2.1章

**定义**：
map 是关联容器，存储键值对，按键自动排序，键唯一。

### 5.10 set

**名称**：集合容器（Set）

**首次出现位置**：C13_105-模板与STL.md 第2.1章

**定义**：
set 是关联容器，存储唯一元素的集合，按键自动排序。

### 5.11 Lambda 表达式

**名称**：Lambda表达式（Lambda Expression）

**首次出现位置**：C13_105-模板与STL.md 第2.3章

**定义**：
Lambda 表达式是 C++11 引入的匿名函数对象，可以定义即时的行为或函数对象。

### 5.12 模板特化

**名称**：模板特化（Template Specialization）

**首次出现位置**：C13_105-模板与STL.md

**定义**：
模板特化是为特定类型提供模板的特殊实现，覆盖通用模板行为。

### 5.13 Concepts

**名称**：概念（Concepts）

**首次出现位置**：C13_105-模板与STL.md

**定义**：
Concepts 是 C++20 引入的模板约束机制，用于指定模板参数必须满足的要求。

## 6. 内存管理类

### 6.1 堆内存

**名称**：堆内存（Heap Memory）

**首次出现位置**：C13_103-指针引用与内存管理.md 第3章

**定义**：
堆内存是程序动态分配的内存区域，通过 new 分配，通过 delete 释放。

### 6.2 栈内存

**名称**：栈内存（Stack Memory）

**首次出现位置**：C13_103-指针引用与内存管理.md 第3章

**定义**：
栈内存是程序自动管理的内存区域，用于存储局部变量和函数调用信息。

### 6.3 new 和 delete

**名称**：new和delete运算符（New and Delete）

**首次出现位置**：C13_103-指针引用与内存管理.md 第3章

**定义**：
new 用于在堆上分配内存并调用构造函数，delete 用于释放内存并调用析构函数。

### 6.4 智能指针

**名称**：智能指针（Smart Pointers）

**首次出现位置**：C13_105-模板与STL.md

**定义**：
智能指针是 C++11 引入的 RAII 封装指针，可以自动管理内存，包括 unique_ptr、shared_ptr、weak_ptr。

### 6.5 unique_ptr

**名称**：独占智能指针（Unique Pointer）

**首次出现位置**：C13_105-模板与STL.md

**定义**：
unique_ptr 是独占所有权的智能指针，同一时刻只能有一个指针拥有资源。

### 6.6 shared_ptr

**名称**：共享智能指针（Shared Pointer）

**首次出现位置**：C13_105-模板与STL.md

**定义**：
shared_ptr 是共享所有权的智能指针，使用引用计数，多个指针可以共享同一资源。

### 6.7 weak_ptr

**名称**：弱智能指针（Weak Pointer）

**首次出现位置**：C13_105-模板与STL.md

**定义**：
weak_ptr 是不参与引用计数的智能指针，用于解决 shared_ptr 的循环引用问题。

### 6.8 内存泄漏

**名称**：内存泄漏（Memory Leak）

**首次出现位置**：C13_103-指针引用与内存管理.md

**定义**：
内存泄漏是指程序分配的内存未能正确释放，导致可用内存逐渐减少的现象。

### 6.9 悬垂指针

**名称**：悬垂指针（Dangling Pointer）

**首次出现位置**：C13_103-指针引用与内存管理.md

**定义**：
悬垂指针是指向已释放内存的指针，访问悬垂指针会导致未定义行为。

## 7. 现代 C++ 特性类

### 7.1 nullptr

**名称**：空指针（Nullptr）

**首次出现位置**：C13_101-概述与标准.md 第2.2章

**定义**：
nullptr 是 C++11 引入的空指针常量，比 NULL 更类型安全。

### 7.2 右值引用

**名称**：右值引用（Rvalue Reference）

**首次出现位置**：C13_101-概述与标准.md 第2.2章

**定义**：
右值引用是 C++11 引入的引用类型，用于绑定到临时对象，支持移动语义。

### 7.3 移动语义

**名称**：移动语义（Move Semantics）

**首次出现位置**：C13_101-概述与标准.md 第2.2章

**定义**：
移动语义是 C++11 引入的机制，通过转移资源所有权而非复制来提高性能。

### 7.4 移动构造函数

**名称**：移动构造函数（Move Constructor）

**首次出现位置**：C13_101-概述与标准.md

**定义**：
移动构造函数是接受右值引用的构造函数，通过窃取资源而非复制来构造对象。

### 7.5 范围 for 循环

**名称**：范围for循环（Range-based For Loop）

**首次出现位置**：C13_101-概述与标准.md 第2.2章

**定义**：
范围 for 循环是 C++11 引入的简化遍历语法，语法为 for (auto& x : container)。

### 7.6 override

**名称**：覆盖说明符（Override）

**首次出现位置**：C13_101-概述与标准.md 第2.2章

**定义**：
override 是 C++11 引入的关键字，用于显式标记派生类函数覆盖基类虚函数。

### 7.7 final

**名称**：最终说明符（Final）

**首次出现位置**：C13_101-概述与标准.md 第2.2章

**定义**：
final 是 C++11 引入的关键字，用于禁止派生类继续覆盖虚函数或禁止类被继承。

### 7.8 default 和 delete

**名称**：默认和删除说明符（Default and Delete）

**首次出现位置**：C13_101-概述与标准.md

**定义**：
default 用于显式要求编译器生成默认实现，delete 用于禁止某些函数被调用。

### 7.9 结构化绑定

**名称**：结构化绑定（Structured Bindings）

**首次出现位置**：C13_101-概述与标准.md 第2.4章

**定义**：
结构化绑定是 C++17 引入的语法，允许一次性声明并绑定多个变量，如 auto [x, y] = pair。

### 7.10 if constexpr

**名称**：编译时条件分支（If Constexpr）

**首次出现位置**：C13_101-概述与标准.md 第2.4章

**定义**：
if constexpr 是 C++17 引入的编译时条件语句，在编译时求值并选择代码分支。

### 7.11 std::optional

**名称**：可选值（Std Optional）

**首次出现位置**：C13_101-概述与标准.md 第2.4章

**定义**：
std::optional 是 C++17 引入的模板类，表示一个可能存在也可能不存在的值。

### 7.12 std::variant

**名称**：变体类型（Std Variant）

**首次出现位置**：C13_101-概述与标准.md 第2.4章

**定义**：
std::variant 是 C++17 引入的模板类，表示一个可以存储多种类型中任意一种值的类型。
