## 1. 屏幕系统 (Screen System)
### 1.1 基础屏幕定义
```renpy
screen main_menu:
    vbox:
        xalign 0.5
        yalign 0.5
        spacing 20
        
        text "游戏标题" size 48 bold True
        
        imagebutton:
            idle "button_idle.png"
            hover "button_hover.png"
            action Start()
            
        imagebutton:
            idle "button_idle.png"
            hover "button_hover.png"
            action ShowMenu("load")
            
        imagebutton:
            idle "button_idle.png"
            hover "button_hover.png"
            action Quit()
```

### 1.2 屏幕布局系统
- **`vbox`**: 垂直布局
- **`hbox`**: 水平布局
- **`grid`**: 网格布局
- **`fixed`**: 固定位置布局
- **`frame`**: 带边框的容器

### 1.3 常用控件
- **`text`**: 文本显示
- **`image`**: 图像显示
- **`imagebutton`**: 图像按钮
- **`button`**: 文本按钮
- **`bar`**: 进度条
- **`input`**: 输入框
- **`timer`**: 定时器

### 1.4 屏幕动作 (Actions)
- **`Start()`**: 开始新游戏
- **`Load()`**: 加载存档
- **`Save()`**: 保存游戏
- **`Quit()`**: 退出游戏
- **`Jump(label)`**: 跳转到指定标签
- **`Return()`**: 返回上一层
- **`SetVariable(var, value)`**: 设置变量值
- **`ToggleVariable(var)`**: 切换变量值
- **`ShowScreen(screen)`**: 显示屏幕
- **`HideScreen(screen)`**: 隐藏屏幕

## 2. Python 集成 (Python Integration)
### 2.1 基础用法
```renpy

python:
    def calculate_damage(attacker, defender):
        base_damage = attacker.strength * 2
        defense = defender.defense * 0.5
        return max(1, base_damage - defense)

# 初始化阶段运行的代码
init python:
    import random
    import math
    
    class Inventory:
        def __init__(self):
            self.items = []
        
        def add_item(self, item):
            self.items.append(item)
        
        def remove_item(self, item):

## 4. 构建与发布 (Build & Distribution)
### 4.1 构建流程
1. **准备资源**: 确保所有资源文件都已就绪
2. **设置项目信息**: 在 `options.rpy` 中配置游戏信息
3. **构建发布版本**: 使用 Ren'Py Launcher 的 "Build Distributions"
4. **测试发布版本**: 在目标平台上测试游戏

### 4.2 多平台构建

init python:
    import renpy.display
    
    def create_particles():
        particles = []
        for i in range(50):
            particle = {
                "x": 400,
                "y": 300,
                "vx": random.uniform(-2, 2),
                "vy": random.uniform(-2, 0),
                "alpha": 1.0,
                "size": random.uniform(1, 3)
            }
            particles.append(particle)
        return particles
```
