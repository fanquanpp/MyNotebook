# GitHub 一站式中文教程 | GitHub Platform Tutorial

> **路径**：`01-Github完整教程/` | **GitHub（Git 托管平台）**：网页端协作与 **CI/CD（持续集成/持续交付）**。

## 1. 项目简介 | Introduction

本教程是 fanquanpp 个人综合学习笔记库中的 GitHub 平台教程部分，专注于 GitHub 网页端操作与平台特有能力，包括账户安全、远程协议、仓库生命周期、分支策略、PR/Issue、Actions、Pages、协作规范与排错等内容。本教程与仓库内的 [`07-Git版本控制`](../07-Git版本控制/README.md) 形成互补关系，后者主要讲解 Git 命令与核心概念，而本教程则聚焦于 GitHub 平台的实战应用。

### 教程定位

- **GitHub 平台指南**：专注于 GitHub 网页端操作和平台特有功能
- **实战导向**：提供详细的操作步骤和最佳实践，帮助读者快速上手
- **进阶提升**：从零基础到高级 CI/CD 配置，满足不同层次的学习需求
- **问题解决**：包含常见问题与排查指南，帮助读者解决实际使用中遇到的问题

**使用说明：**

- 本教程已开放为公共资源，允许他人访问和克隆
