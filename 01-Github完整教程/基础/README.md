# GitHub 基础体系 (GitHub Basics)

> 此目录面向 GitHub 平台的「从零到可协作」入门学习：账号安全 → 连接方式 → 仓库创建与维护。每篇独立成篇，按顺序学习可形成完整工作流。

## 1. 学习路径 (Learning Path)

`01 账号与安全` → `02 SSH 与 HTTPS` → `03 仓库创建与管理`

## 2. 文件清单 (Files)

| 序号 | 主题 (Topic) | 文件 (File) | 你将掌握 (Outcome) |
| :--- | :----------- | :----------- | :----------------- |
| 01 | 账号与安全 | [01-账户与安全.md](./01-账户与安全.md) | 账号体系、2FA、安全最佳实践、密钥/令牌边界 |
| 02 | 连接方式 | [02-SSH与HTTPS.md](./02-SSH与HTTPS.md) | SSH/HTTPS 差异、密钥配置、常见报错排查 |
| 03 | 仓库管理 | [03-仓库创建与管理.md](./03-仓库创建与管理.md) | 新建/克隆/权限、README/LICENSE、分支与标签 |

## 3. 先修要求 (Prerequisites)

- 已安装 Git（建议 Git 2.40+）
- 可访问 GitHub（网络与账号注册完成）
