## 3 . 条件语句

<a id="3.1"></a>
### 3.1 if 语句

<a id="3.1.1"></a>
#### 3.1.1 if 的单分支结构

**格式：**

```java
if(条件表达式){
    // 条件成立时执行的代码内容
}
```

**运行逻辑：**
当条件成立时（true），执行{}里面的代码，不成立则直接跳过不执行。

**例子：**

```java
int i = 10086;   // 定义变量 i
if(i > 114514){     // 条件为 i 大于 114514(条件不成立)
     System.out.println("宝宝巴士教我上厕所秘诀");  // 此句不会输出
}
```

<a id="3.1.2"></a>
#### 3.1.2 if 的双分支结构

**格式：**

```java
if(条件表达式){
    // 条件成立时执行的代码
} else {
    // 条件不成立时执行的代码
}
```

**例子：**

```java
int i = 10; // 定义一个变量
if(i >= 10){    // 判断条件
     System.out.println("天上浮云似白衣，斯须改变成苍狗。");    // 成立则执行此命令
} else {
     System.out.println("山重水尽疑无路，柳暗花明又一村。");    // 否则（条件不成立，则执行另外一条命令）
}
```

<a id="3.1.3"></a>
#### 3.1.3 if 的多分支结构

**结构：**

```java
if(条件一){
    // 条件一成立时执行的代码
} else if(条件二){
    // 条件二成立时执行的代码
} else if(条件三){
    // 条件三成立时执行的代码
} else {
    // 三个条件都不成立时执行的代码
}
```

**运行逻辑：**
当条件一成立时，执行{}里面的代码，否则进入下一条判断条件，当条件二成立时，执行{}里面的代码，否则进入下一条判断条件，当条件三成立时，执行{}里面的代码，否则执行 else{}里面的代码。

<a id="3.1.4"></a>
#### 3.1.4 if 的嵌套结构

**嵌套结构：**

```java
if(条件一){ // 当条件一不成立，{}内的代码不会执行，直接跳过。
    if(条件二){
        // 条件一和条件二都成立时执行的代码
    } else {
        // 条件一成立而条件二不成立时执行的代码
    }
} else {
    // 条件一不成立时执行的代码
}
```

<a id="3.2"></a>
### 3.2 switch 语句

<a id="3.2.1"></a>
#### 3.2.1 switch 语句结构(基础)

```java
switch(表达式){
    case 1: 
        语句 1;
        break;
    case 2: 
        语句 2;
        break;
    case 3: 
        语句 3;
        break;
    default: 
        语句 4;
}
```

**例子：**

```java
int i = 3;
switch(i){
    case 1: 
        System.out.println("星期一");   // i=1,输出星期一
        break;
    case 2: 
        System.out.println("星期二");   // i=2,输出星期二
        break;
    case 3: 
        System.out.println("星期三");   // i=3,输出星期三
        break;  // 这个 break 是必须的
    default: 
        System.out.println("输入错误");
}
```

**break 的作用：**
用于跳出循环结构（如 for、while 和 do-while 循环）或 switch 语句。当遇到 break 语句时，程序将立即跳出当前循环或 switch 语句，继续执行紧跟在循环或 switch 语句后面的代码。

<a id="3.2.2"></a>
#### 3.2.2 switch 语句(case 穿透法)

**case 穿透:** 当不同 case 的结果一样时，可以把 break 省略,只留下最后一个 break.

**例子：**

```java
int i = 2;
switch(i){ 
    case 1: 
    case 2:
    case 3:
    case 4:
    case 5:
        System.out.println("工作日");   // i=1,i=2,i=3,i=4,i=5,输出工作日
        break;      
    case 6: 
    case 7: 
        System.out.println("休息日");   // i=6,i=7,输出休息日
        break;
    default: 
        System.out.println("输入错误");
        break;
}
```

<a id="3.2.3"></a>
#### 3.2.3 switch 语句(新特性:箭头语法)

**箭头语法:** switch 语句的 case 穿透方法，也可以使用箭头语法。

**例子(无穿透)：**

```java
int i = 3;
switch(i){ 
    case 1 -> System.out.println("工作日");   // i=1,输出工作日
    case 2 -> System.out.println("工作日");   // i=2,输出工作日
    case 3 -> System.out.println("工作日");   // i=3,输出工作日
    case 4 -> System.out.println("工作日");   // i=4,输出工作日
    case 5 -> System.out.println("工作日");   // i=5,输出工作日
    case 6 -> System.out.println("休息日");   // i=6,输出休息日
    case 7 -> System.out.println("休息日");   // i=7,输出休息日
    default -> System.out.println("输入错误"); 
}
```

**例子(有穿透)：**

```java
int i = 5;
switch(i){ 
    case 1,2,3,4,5 -> System.out.println("工作日");   // i=1,i=2,i=3,i=4,i=5,输出工作日
    case 6,7 -> System.out.println("休息日");   // i=6,i=7,输出休息日
    default -> System.out.println("输入错误");  
}
```

**箭头语法的语法格式为：**

```java
case 值 1, 值 2, 值 3, ... -> 语句;
```

**注意:** 箭头语法时,不需要 break。

**箭头语法的优点:**
1. 代码简洁
2. 可读性高

**箭头语法的特点:**
1. 只能用于 switch 语句
2. 自带阻断效果,不需要 break

## 4 . 循环语句

<a id="4.1"></a>
### 4.1 for 循环

<a id="4.1.1"></a>
#### 4.1.1 格式：

```java
for(初始化语句;条件判断语句;条件控制语句){
    循环体;
}
```

<a id="4.1.2"></a>
#### 4.1.2 解析：

```java
for（定义一个初始值变量;判断结束的条件;变量变化的逻辑）{
    循环体（这里的内容会重复执行，直到满足条件才会结束循环）;
}
```

<a id="4.1.3"></a>
#### 4.1.3 举例：

```java
public class ForDemo1 {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            System.out.println("HelloWorld!!!");    // 输出 10 次
        }
    }
}
```

<a id="4.1.4"></a>
#### 4.1.4 总结：

for 循环，循环次数是固定的，循环次数由 for 循环的初始化语句、条件判断语句、条件控制语句决定。

<a id="4.2"></a>
### 4.2 while 循环

<a id="4.2.1"></a>
#### 4.2.1 while 循环结构：

```java
while(条件表达式){
    循环体
}
```

<a id="4.2.2"></a>
#### 4.2.2 例子：

```java
public class Demo01While {
    public static void main(String[] args) {
        int a = 1;  // 初始化条件
        while(a <= 5){  // 条件表达式
            System.out.println("hello world");  // 循环体
            a++;    // 更新条件
        }
    }
}
// 输出结果：hello world 五次
```

<a id="4.2.3"></a>
#### 4.2.3 循环的结束条件：
1. 条件表达式为 false 时，循环结束
2. 条件表达式为 true 时，循环继续

<a id="4.2.4"></a>
#### 4.2.4 注意：
1. while 循环可能一次都不执行循环体
2. while 循环的结束条件一定要是可判断的，否则会死循环

<a id="4.3"></a>
### 4.3 do-while 循环

<a id="4.3.1"></a>
#### 4.3.1 do-while 循环结构：

```java
do{
    循环体
}while(条件表达式);
```

<a id="4.3.2"></a>
#### 4.3.2 例子：

```java
public class Demo03DoWhile {
    public static void main(String[] args) {
        int a = 1;  // 初始化条件
        do{
            System.out.println("hello world");  // 循环体
            a++;    // 更新条件
        }while(a <= 5);  // 条件表达式
    }
}
// 输出结果：hello world 五次
```

<a id="4.3.3"></a>
#### 4.3.3 注意：
- do-while 循环至少会执行一次循环体
- do-while 循环的结束条件一定要是可判断的，否则会死循环

## 5 . 跳转语句

跳转语句用于改变程序的执行流程，包括 break、continue 和 return。

<a id="5.1"></a>
### 5.1 break 语句

**作用：** 跳出当前循环或 switch 语句。

**示例：**

```java
// 跳出循环
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // 当 i 等于 5 时跳出循环
    }
    System.out.println(i);
}
// 输出：1 2 3 4

// 跳出 switch 语句
int day = 3;
switch (day) {
    case 1:
        System.out.println("星期一");
        break;
    case 2:
        System.out.println("星期二");
        break;
    case 3:
        System.out.println("星期三");
        break; // 跳出 switch 语句
    default:
        System.out.println("输入错误");
}
```

<a id="5.2"></a>
### 5.2 continue 语句

**作用：** 跳过当前循环的剩余部分，直接进入下一次循环。

**示例：**

```java
// 跳过奇数
for (int i = 1; i <= 10; i++) {
    if (i % 2 != 0) {
        continue; // 跳过奇数
    }
    System.out.println(i);
}
// 输出：2 4 6 8 10
```

<a id="5.3"></a>
### 5.3 return 语句

**作用：** 结束当前方法的执行，返回方法的返回值（如果有）。

**示例：**

```java
public int max(int a, int b) {
    if (a > b) {
        return a; // 返回 a
    } else {
        return b; // 返回 b
    }
}
```

## 6 . 循环嵌套

循环嵌套是指在一个循环内部再定义一个或多个循环，用于处理二维或多维数据结构。

<a id="6.1"></a>
### 6.1 for 循环嵌套

**示例：** 打印九九乘法表

```java
public class MultiplicationTable {
    public static void main(String[] args) {
        // 外层循环控制行数
        for (int i = 1; i <= 9; i++) {
            // 内层循环控制每行的列数
            for (int j = 1; j <= i; j++) {
                // 打印乘法表达式
                System.out.print(j + "×" + i + "=" + (i * j) + "\t");
            }
            // 每行结束后换行
            System.out.println();
        }
    }
}
```

**运行结果：**
```
1×1=1    
1×2=2    2×2=4    
1×3=3    2×3=6    3×3=9    
1×4=4    2×4=8    3×4=12    4×4=16    
1×5=5    2×5=10    3×5=15    4×5=20    5×5=25    
1×6=6    2×6=12    3×6=18    4×6=24    5×6=30    6×6=36    
1×7=7    2×7=14    3×7=21    4×7=28    5×7=35    6×7=42    7×7=49    
1×8=8    2×8=16    3×8=24    4×8=32    5×8=40    6×8=48    7×8=56    8×8=64    
1×9=9    2×9=18    3×9=27    4×9=36    5×9=45    6×9=54    7×9=63    8×9=72    9×9=81    
```

<a id="6.2"></a>
### 6.2 while 循环嵌套

**示例：** 打印金字塔

```java
public class Pyramid {
    public static void main(String[] args) {
        int rows = 5;
        int i = 1;
        
        // 外层循环控制行数
        while (i <= rows) {
            int j = 1;
            
            // 打印空格
            while (j <= rows - i) {
                System.out.print(" ");
                j++;
            }
            
            // 打印星号
            j = 1;
            while (j <= 2 * i - 1) {
                System.out.print("*");
                j++;
            }
            
            // 换行
            System.out.println();
            i++;
        }
    }
}
```

**运行结果：**
```
    *
   ***
  *****
 *******
*********
```

<a id="6.3"></a>
### 6.3 循环嵌套的应用场景

1. **二维数组遍历**：使用嵌套循环遍历二维数组的所有元素
2. **矩阵运算**：矩阵的加法、乘法等操作
3. **图形打印**：打印各种图形，如菱形、空心菱形等
4. **组合问题**：生成所有可能的组合
5. **搜索算法**：如深度优先搜索、广度优先搜索等

<a id="6.4"></a>
### 6.4 循环嵌套的性能注意事项

- **时间复杂度**：嵌套循环的时间复杂度通常为 O(n²) 或更高
- **避免过深嵌套**：尽量减少循环嵌套的层数，一般不超过 3 层
- **优化循环条件**：避免在循环条件中进行复杂计算
- **考虑替代方案**：对于大型数据集，考虑使用更高效的算法

## 7 . 常见问题与解决方案

<a id="7.1"></a>
### 7.1 死循环问题

**问题描述：** 循环无法正常结束，导致程序无限执行。

**原因分析：** 循环条件永远为 true，或循环控制变量没有正确更新。

**解决方案：**
- 确保循环条件能够在某个时刻变为 false
- 确保循环控制变量在循环体内正确更新
- 使用 break 语句在适当的时候跳出循环

<a id="7.2"></a>
### 7.2 循环次数错误

**问题描述：** 循环执行的次数与预期不符。

**原因分析：** 循环条件或循环控制变量的更新逻辑错误。

**解决方案：**
- 仔细检查循环条件
- 确认循环控制变量的初始值、更新方式和结束条件
- 使用调试工具或打印语句验证循环执行过程

<a id="7.3"></a>
### 7.3 switch 语句的 case 穿透问题

**问题描述：** switch 语句中的 case 没有 break，导致执行多个 case。

**原因分析：** 忘记在 case 分支末尾添加 break 语句。

**解决方案：**
- 在每个 case 分支末尾添加 break 语句
- 利用 case 穿透特性实现特定功能时，确保逻辑正确
- 使用箭头语法避免 case 穿透问题

<a id="7.4"></a>
### 7.4 循环嵌套的性能问题

**问题描述：** 循环嵌套导致程序执行效率低下。

**原因分析：** 循环嵌套的时间复杂度为 O(n²) 或更高。

**解决方案：**
- 尽量减少循环嵌套的层数
- 优化循环条件和循环体
- 考虑使用更高效的算法

## 8 . 总结与最佳实践

<a id="8.1"></a>
### 8.1 核心概念

- **条件语句**：根据条件执行不同的代码块
- **循环语句**：重复执行代码块
- **跳转语句**：改变程序的执行流程
- **循环嵌套**：在循环内部再定义循环

<a id="8.2"></a>
### 8.2 最佳实践

1. **代码可读性**
   - 使用缩进和空格提高代码可读性
   - 为复杂的条件表达式添加括号
   - 为循环和条件语句添加注释

2. **选择合适的流程控制语句**
   - 对于固定次数的循环，使用 for 循环
   - 对于条件循环，使用 while 循环
   - 对于至少执行一次的循环，使用 do-while 循环
   - 对于多条件选择，使用 switch 语句

3. **避免常见错误**
   - 避免死循环
   - 确保循环控制变量正确更新
   - 在 switch 语句中正确使用 break
   - 合理使用跳转语句，避免过度使用

4. **性能优化**
   - 减少循环嵌套的层数
   - 避免在循环中进行昂贵的操作
   - 合理使用 break 和 continue 语句
   - 考虑使用增强 for 循环（for-each）遍历数组和集合

5. **代码风格**
   - 大括号使用一致的风格（推荐使用大括号始终换行）
   - 条件表达式和循环控制语句保持简洁
   - 一行只包含一个语句

<a id="8.3"></a>
### 8.3 个人实践总结
- 根据具体场景选择合适的流程控制语句
- 保持代码可读性，使用适当的缩进和注释
- 避免死循环和无限循环
- 合理使用 break 和 continue 语句控制循环流程
- 减少循环嵌套的层数，优化性能
- 正确使用 switch 语句，避免 case 穿透问题
