# 06-Java 后端开发 | 企业级应用基础

## 1. 项目概述
本目录收录了 Java 语言核心知识点、面向对象编程思想、JVM 基础及后端开发必备规范。旨在为构建稳定、可扩展的企业级后端服务奠定基础。

## 4. 快速开始
1. 安装 JDK 17+ 并配置环境变量。
2. 编写 `Hello.java` 并编译：`javac Hello.java`。
3. 运行：`java Hello`。

## 5. 学习路线
`0.0-基础入门` -> `0.1-基础语法` -> `0.3-流程控制` -> `0.5-方法` -> `2.0-核心知识点`

- 代码示例需符合 Google Java Style Guide 规范。
