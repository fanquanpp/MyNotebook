# 06-Java 后端开发 | 企业级应用基础

## 1. 项目概述
本目录收录了 Java 语言核心知识点、面向对象编程思想、JVM 基础及后端开发必备规范。旨在为构建稳定、可扩展的企业级后端服务奠定基础。

## 2. 技术栈说明
- **语言版本**: OpenJDK 17 / 21
- **核心框架**: Java SE (Standard Edition)
- **构建工具**: Maven 3.8+ / Gradle 7+
- **单元测试**: JUnit 5 / AssertJ

## 3. 目录结构
- `0.0-Java基础入门.md`: 环境搭建、第一个 Hello World
- `0.1-Java基础语法.md`: 变量、类型转换、常量
- `0.2-Java运算符与表达式.md`: 算术、逻辑、三元运算符
- `0.3-Java流程控制.md`: `if-else`, `switch`, `for/while` 循环
- `0.4-Java数组.md`: 一维/多维数组、内存布局
- `0.5-Java方法.md`: 定义、调用、重载、参数传递
- `07-计算机存储规则.md`: 二进制、十六进制、内存寻址
- `1.1-Java基础骨架.md`: 类结构、包管理、模块化
- `1.2-Java数据类型.md`: 基本类型与引用类型详解
- `1.3-Java标识符.md`: 命名规范、关键字
- `2.0-Java核心知识点.md`: 面向对象(封装/继承/多态)、异常处理、集合框架

## 4. 快速开始
1. 安装 JDK 17+ 并配置环境变量。
2. 编写 `Hello.java` 并编译：`javac Hello.java`。
3. 运行：`java Hello`。

## 5. 学习路线
`0.0-基础入门` -> `0.1-基础语法` -> `0.3-流程控制` -> `0.5-方法` -> `2.0-核心知识点`

## 6. 贡献指南
- 代码示例需符合 Google Java Style Guide 规范。
- 逻辑解析需配以内存模型图（如栈、堆、方法区）。
