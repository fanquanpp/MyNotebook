## 1. 项目简介 | Introduction

本目录收录了 Java 语言核心知识点、面向对象编程思想、JVM 基础及后端开发必备规范。旨在为构建稳定、可扩展的企业级后端服务奠定基础。

This module covers Java core concepts, object-oriented programming, JVM fundamentals, and backend development best practices for building stable and scalable enterprise services.

## 3. 环境要求 | Environment Requirements

- **操作系统**：Windows 10+, Ubuntu 22.04+, macOS 14+
- **运行时**：JDK 17+ (LTS)
- **开发工具**：IntelliJ IDEA, Eclipse, VS Code
- **构建工具**：Maven 3.8+, Gradle 7.0+

## 4. 快速开始 | Quick Start

1. 安装 JDK 17+ 并配置环境变量
2. 编写 `Hello.java` 并编译：`javac Hello.java`
3. 运行：`java Hello`
