## 1. 运算符分类 (Operator Categories)

### 1.1 算术运算符 (Arithmetic)
| 运算符 | 描述 | 示例 (a=10, b=3) |
|---|---|---|
| `+` | 加法 / 字符串拼接 | `a + b = 13` |
| `-` | 减法 | `a - b = 7` |
| `*` | 乘法 | `a * b = 30` |
| `/` | 除法 | `a / b = 3` (整数除法舍去小数) |
| `%` | 取模 (取余) | `a % b = 1` |
| `++` | 自增 | `a++` (先用后加), `++a` (先加后用) |
| `--` | 自减 | `b--`, `--b` |

### 1.2 关系运算符 (Relational)
结果为 `boolean` 类型。
- `==`, `!=`, `>`, `<`, `>=`, `<=`

### 1.3 逻辑运算符 (Logical)
- `&&`: 短路与 (AND) - 第一个为假，则不计算第二个。
- `||`: 短路或 (OR) - 第一个为真，则不计算第二个。
- `!`: 逻辑非 (NOT)
- `&`, `|`, `^`: 逻辑与、或、异或 (无短路特性)。

### 1.4 位运算符 (Bitwise)
对二进制位进行操作。
- `&`: 按位与 (AND)
- `|`: 按位或 (OR)
- `^`: 按位异或 (XOR)
- `~`: 按位取反 (NOT)
- `<<`: 左移 (乘 2 的幂)
- `>>`: 右移 (带符号，除 2 的幂)
- `>>>`: 无符号右移 (高位补 0)

### 1.5 赋值运算符 (Assignment)
- `=`, `+=`, `-=`, `*=`, `/=`, `%=`, `<<=`, `>>=`, `&=`, `^=`, `|=`

### 1.6 三元运算符 (Ternary)
语法: `条件表达式 ? 表达式1 : 表达式2`
```java
int max = (a > b) ? a : b;
```

## 2. 运算符优先级 (Precedence)
1. **最高**: `()`, `[]`, `.`
2. **单目**: `!`, `~`, `++`, `--`
3. **算术**: `*`, `/`, `%` 然后 `+`, `-`
4. **位移**: `<<`, `>>`, `>>>`
5. **关系**: `>`, `>=`, `<`, `<=` 然后 `==`, `!=`
6. **位运算**: `&`, `^`, `|`
7. **逻辑**: `&&`, `||`
8. **三元**: `? :`
9. **赋值**: `=`, `+=` 等

## 3. 常见陷阱
- **浮点精度**: `0.1 + 0.2` 在 Java 中不等于 `0.3` (由于 IEEE 754 精度问题)，应使用 `BigDecimal`。
- **整数溢出**: `int` 最大值加 1 会变成最小值。
