# CSS 布局进阶

本目录包含 CSS 布局的高级技术和实践指南，旨在帮助开发者掌握现代前端布局技术。

## 目录内容

- **01-响应式设计.md**：响应式设计的核心概念、媒体查询、弹性布局技术和最佳实践

## 学习建议

1. 先掌握基础布局技术（盒模型、Flexbox、Grid）
2. 学习响应式设计原则和媒体查询
3. 实践响应式布局案例，如响应式导航栏、卡片布局等
4. 结合实际项目需求，应用响应式设计技术
