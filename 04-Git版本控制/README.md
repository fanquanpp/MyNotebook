## 1. 项目简介 | Introduction

本目录收录了 Git 核心概念与日常协作规范，旨在帮助开发者高效管理代码库，建立清晰的版本历史与协作流程。

This module covers Git core concepts and daily collaboration norms, aiming to help developers efficiently manage code repositories and establish clear version history and collaboration processes.

## 3. 环境要求 | Environment Requirements

- **工具**: Git 2.40+
- **平台**: GitHub / GitLab / Gitee
- **协议**: SSH / HTTPS
- **操作系统**: Windows 10+, Ubuntu 22.04+, macOS 14+

## 4. 快速开始 | Quick Start

1. 配置个人信息：`git config --global user.name "Your Name"`
2. 克隆仓库：`git clone <repo-url>`
3. 查看状态：`git status`
