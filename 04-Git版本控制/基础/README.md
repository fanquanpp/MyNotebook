# Git 基础体系 (Git Basics)

> 此目录包含 Git 的基础概念与日常高频命令，采用「一知识点一文件」的结构，适合从零快速上手并形成可复用的协作习惯。

## 1. 学习路径 (Learning Path)

`01 概述与心智模型` → `02 环境与初始化` → `03 基本操作` → `04 分支管理` → `05 远程协作`

- 安装 Git 2.40+（Windows / macOS / Linux 均可）
- 具备基础命令行操作能力（cd、ls/dir、文本编辑器）
