## 1. 简介 | Introduction

Ren'Py 是由 Tom "PyTom" Rothamel 创建的免费开源视觉小说引擎。它使用基于 Python 的脚本语言，专门用于创建交互式的视觉小说游戏，支持多平台发布（Windows、Mac、Linux、Android、iOS）。
