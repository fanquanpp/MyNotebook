## 1. 角色定义 | Characters

角色是视觉小说中的重要元素，用于展示对话和互动。Ren'Py 提供了灵活的角色定义方式。

### 1.1 基本角色定义 | Basic Character Definition

```renpy
# 基本角色定义
define e = Character("艾琳", color="#c8ffc8")

# 更详细的角色定义
define m = Character(
    "小明",
    color="#c8c8ff",
    what_color="#ffffff",
    what_outlines=[(2, "#000000", 0, 0)]
)
```

### 1.2 角色属性 | Character Attributes

| 属性 | 描述 | 默认值 |
| :--- | :--- | :--- |
| **name** | 角色名称 | 必填 |
| **color** | 角色名称颜色 | #ffffff |
| **what_color** | 对话文本颜色 | #ffffff |
| **who_color** | 角色名称颜色（优先级高于 color） | None |
| **what_outlines** | 对话文本轮廓 | [] |
| **who_outlines** | 角色名称轮廓 | [] |
| **what_size** | 对话文本大小 | None |
| **who_size** | 角色名称大小 | None |
| **font** | 字体文件 | None |
| **what_font** | 对话文本字体 | None |
| **who_font** | 角色名称字体 | None |
| **voice** | 角色语音文件夹 | None |
| **image** | 角色默认图像 | None |

### 1.3 角色定义示例 | Character Definition Examples

```renpy
# 带语音的角色
define l = Character(
    "丽萨",
    color="#ffc8c8",
    voice="voice/lisa/"
)

# 带默认图像的角色
define k = Character(
    "凯文",
    color="#c8ffff",
    image="kevin"
)

# 匿名角色（只有对话，没有角色名称）
define narrator = Character(None)
```

### 1.4 角色使用 | Using Characters

```renpy
# 使用角色进行对话
e "你好，我是艾琳！"
m "你好，艾琳！我是小明。"
l "大家好，我是丽萨！"
narrator "这是一段旁白，没有角色名称。"

# 带表情的角色对话
show eileen happy
e "我很高兴见到你！"

show eileen sad
e "但是我现在有点难过。"
```

## 2. 对话与标签 | Dialogue & Labels

### 2.1 标签 | Labels

标签是 Ren'Py 脚本中的标记位置，用于组织脚本结构和实现跳转。

```renpy
# 主标签
label start:
    "游戏开始"
    jump chapter1

# 章节标签
label chapter1:
    "第一章：相遇"
    return

# 场景标签
label park:
    "公园场景"
    return

# 结束标签
label end:
    "游戏结束"
    return
```

### 2.2 对话 | Dialogue

对话是视觉小说的核心元素，用于展示角色的交流。

#### 2.2.1 基本对话 | Basic Dialogue

```renpy
# 基本对话
e "你好，欢迎来到 Ren'Py 的世界！"

# 多行对话
e "这是第一行对话。\n这是第二行对话。"

# 带变量的对话
$ name = "玩家"
e "你好，[name]！很高兴认识你。"
```

#### 2.2.2 旁白 | Narration

```renpy
# 使用旁白
"这是一段旁白，没有角色名称。"

# 使用匿名角色作为旁白
define narrator = Character(None)
narrator "这是一段旁白。"
```

#### 2.2.3 对话选项 | Menu Choices

```renpy
# 带选择的对话
menu:
    "你好":
        e "你好！"
    "再见":
        e "再见！"
```

### 2.3 标签使用最佳实践 | Best Practices for Labels

- **命名规范**：使用清晰、描述性的标签名称
- **层级结构**：组织标签为逻辑章节和场景
- **避免循环**：确保标签跳转不会导致无限循环
- **使用 return**：在适当的地方使用 return 结束标签执行

## 3. 图像操作 | Images

### 3.1 图像定义 | Image Definition

在 Ren'Py 中，图像可以通过多种方式定义：

```renpy
# 直接定义图像
image bg room = "images/backgrounds/room.png"
image eileen happy = "images/characters/eileen_happy.png"

# 使用 ATL 定义动画图像
image eileen wave:
    "images/characters/eileen_walk1.png"
    pause 0.1
    "images/characters/eileen_walk2.png"
    pause 0.1
    repeat

# 使用变换定义图像
image bg room night:
    "images/backgrounds/room.png"
    im.MatrixColor([
        0.5, 0.5, 0.5, 0, 0,
        0.5, 0.5, 0.5, 0, 0,
        0.5, 0.5, 0.5, 0, 0,
        0, 0, 0, 1, 0,
    ])
```

### 3.2 图像显示 | Image Display

#### 3.2.1 场景 (Scene)

`scene` 语句用于清除当前层并显示背景图像：

```renpy
# 显示背景场景
scene bg room

# 带过渡效果的场景切换
scene bg park with dissolve

# 带位置的场景（较少使用）
scene bg room at center
```

#### 3.2.2 显示 (Show)

`show` 语句用于显示精灵或其他图像：

```renpy
# 显示角色精灵
show eileen happy

# 带位置的显示
show eileen happy at left

# 带过渡效果的显示
show eileen happy with fade

# 显示多个角色
show eileen happy at left
show michael sad at right
```

#### 3.2.3 隐藏 (Hide)

`hide` 语句用于隐藏精灵或其他图像：

```renpy
# 隐藏角色
hide eileen

# 带过渡效果的隐藏
hide eileen with dissolve

# 隐藏所有图像
scene
```

### 3.3 图像位置 | Image Positions

Ren'Py 提供了多种预设位置：

| 位置 | 描述 |
| :--- | :--- |
| **left** | 屏幕左侧 |
| **right** | 屏幕右侧 |
| **center** | 屏幕中心 |
| **truecenter** | 屏幕真正中心 |
| **top** | 屏幕顶部 |
| **bottom** | 屏幕底部 |
| **topleft** | 屏幕左上角 |
| **topright** | 屏幕右上角 |
| **bottomleft** | 屏幕左下角 |
| **bottomright** | 屏幕右下角 |

### 3.4 自定义位置 | Custom Positions

```renpy
# 定义自定义位置
define left = Position(xalign=0.1, yalign=0.9)
define right = Position(xalign=0.9, yalign=0.9)
define center = Position(xalign=0.5, yalign=0.9)

# 使用自定义位置
show eileen happy at left
show michael sad at right
```

## 5. 简单控制流 | Simple Control Flow

### 5.1 跳转 | Jump

`jump` 语句用于跳转到指定标签：

```renpy
# 基本跳转
jump start

# 条件跳转
if score > 50:
    jump good_ending
else:
    jump bad_ending
```

### 5.2 调用 | Call

`call` 语句用于调用指定标签并在结束后返回：

```renpy
# 调用标签
call shop

# 带返回值的调用
$ item = call shop

# 定义可调用标签
label shop:
    menu:
        "购买药水":
            return "potion"
        "购买武器":
            return "sword"
```

### 5.3 菜单 | Menu

`menu` 语句用于显示选择菜单：

#### 5.3.1 基本菜单 | Basic Menu

```renpy
menu:
    "去公园":
        jump park
    "留在家里":
        jump home
```

#### 5.3.2 带条件的菜单 | Conditional Menu

```renpy
menu:
    "去公园" if has_ticket:
        jump park
    "留在家里":
        jump home
```

#### 5.3.3 带变量的菜单 | Menu with Variables

```renpy
menu:
    "选择 [option1]":
        $ choice = "option1"
    "选择 [option2]":
        $ choice = "option2"
```

#### 5.3.4 带描述的菜单 | Menu with Descriptions

```renpy
menu:
    "去公园" if has_ticket:
        "公园里有美丽的风景。"
        jump park
    "留在家里":
        "家里很舒适。"
        jump home
```

### 5.4 条件语句 | Conditional Statements

#### 5.4.1 基本条件语句 | Basic Conditional

```renpy
if score > 50:
    "你赢了！"
elif score > 30:
    "你做得不错！"
else:
    "再试一次！"
```

#### 5.4.2 复杂条件语句 | Complex Conditional

```renpy
if (score > 50) and (items > 3):
    "你是真正的赢家！"
elif (score > 30) or (items > 1):
    "你做得很好！"
else:
    "继续努力！"
```

### 5.5 循环 | Loops

Ren'Py 支持有限的循环功能，主要通过 Python 代码实现：

```renpy
# 使用 Python 循环
python:
    for i in range(5):
        renpy.say(narrator, "这是第 {} 次循环".format(i+1))

# 使用 while 循环
python:
    count = 0
    while count < 3:
        renpy.say(narrator, "计数：{}".format(count))
        count += 1
```

## 7. Python 集成 | Python Integration

### 7.1 内联 Python | Inline Python

使用 `$` 前缀执行单行 Python 代码：

```renpy
# 定义变量
$ name = "玩家"
$ score = 0

# 执行简单操作
$ score += 10
$ name = name.upper()

# 调用函数
$ import random
$ random_number = random.randint(1, 10)
```

### 7.2 Python 块 | Python Blocks

使用 `python:` 块执行多行 Python 代码：

```renpy
python:
    # 定义函数
    def calculate_score(choices):
        total = 0
        for choice in choices:
            if choice == "correct":
                total += 10
            elif choice == "wrong":
                total -= 5
        return total
    
    # 执行操作
    player_choices = ["correct", "correct", "wrong"]
    final_score = calculate_score(player_choices)

# 使用 Python 变量
e "你的最终分数是 [final_score]。"
```

## 9. 实际应用示例 | Practical Examples

### 9.1 简单的视觉小说示例 | Simple Visual Novel Example

```renpy
# script.rpy

# 定义角色
define e = Character("艾琳", color="#c8ffc8")
define m = Character("小明", color="#c8c8ff")
define narrator = Character(None)

# 定义图像
image bg room = "images/backgrounds/room.png"
image bg park = "images/backgrounds/park.png"
image eileen happy = "images/characters/eileen_happy.png"
image eileen sad = "images/characters/eileen_sad.png"
image michael happy = "images/characters/michael_happy.png"

# 主标签
label start:
    # 开始场景
    scene bg room with fade
    narrator "这是一个关于艾琳和小明的故事。"
    
    # 显示角色
    show eileen happy at left with dissolve
    e "你好，我是艾琳！"
    
    show michael happy at right with dissolve
    m "你好，艾琳！我是小明。"
    
    # 对话
    e "今天天气真好，我们去公园吧！"
    
    # 菜单选择
    menu:
        "好啊，一起去公园！":
            jump go_to_park
        "不了，我想留在家里。":
            jump stay_home

# 去公园场景
label go_to_park:
    scene bg park with dissolve
    e "公园的风景真美！"
    m "是啊，我们来玩捉迷藏吧！"
    e "好啊，我来躲，你来找！"
    narrator "他们在公园里度过了愉快的一天。"
    jump end

# 留在家里场景
label stay_home:
    scene bg room with dissolve
    e "那我们在家看书吧。"
    m "好啊，我最近看了一本很有趣的书。"
    narrator "他们在家度过了安静的一天。"
    jump end

# 结束场景
label end:
    narrator "故事结束。"
    return
```

### 9.2 带变量和条件的示例 | Example with Variables and Conditions

```renpy
# script.rpy

# 定义角色
define e = Character("艾琳", color="#c8ffc8")
define narrator = Character(None)

# 定义图像
image bg room = "images/backgrounds/room.png"
image bg park = "images/backgrounds/park.png"
image eileen happy = "images/characters/eileen_happy.png"
image eileen sad = "images/characters/eileen_sad.png"

# 主标签
label start:
    # 初始化变量
    $ score = 0
    $ has_ticket = False
    
    # 开始场景
    scene bg room with fade
    narrator "艾琳的一天开始了。"
    
    show eileen happy at center
    e "今天我要去公园玩！"
    
    # 检查是否有票
    if has_ticket:
        e "我有票，可以直接去公园。"
        jump go_to_park
    else:
        e "我没有票，需要先去买票。"
        jump buy_ticket

# 买票场景
label buy_ticket:
    e "售票处到了。"
    
    menu:
        "购买票 ($10)":
            $ score -= 10
            $ has_ticket = True
            e "我买了票。"
            jump go_to_park
        "不购买":
            e "我决定不买票，回家吧。"
            jump go_home

# 去公园场景
label go_to_park:
    scene bg park with dissolve
    e "公园真漂亮！"
    
    # 随机事件
    $ import random
    $ event = random.randint(1, 3)
    
    if event == 1:
        e "我遇到了一个朋友！"
        $ score += 5
    elif event == 2:
        e "我找到了一个钱包！"
        $ score += 10
    else:
        e "我在公园里散步。"
    
    e "今天玩得真开心！"
    jump end

# 回家场景
label go_home:
    scene bg room with dissolve
    e "回家了。"
    e "今天虽然没去成公园，但在家休息也不错。"
    jump end

# 结束场景
label end:
    e "今天的得分是 [score] 分。"
    
    if score > 0:
        e "今天真是美好的一天！"
    else:
        e "今天过得一般般。"
    
    narrator "故事结束。"
    return
```

## 10. 最佳实践 | Best Practices

### 10.1 脚本组织 | Script Organization

- **模块化**：将脚本分解为多个文件，按章节或功能组织
- **命名规范**：使用清晰、一致的命名约定
- **注释**：为复杂代码添加注释
- **缩进**：使用一致的缩进（推荐 4 个空格）

### 10.2 性能优化 | Performance Optimization

- **图像优化**：使用适当大小和格式的图像
- **音频优化**：使用压缩音频格式
- **代码优化**：避免不必要的计算和重复代码
- **内存管理**：及时隐藏不需要的图像

### 10.3 代码风格 | Code Style

- **变量命名**：使用 snake_case 命名变量
- **标签命名**：使用小写字母和下划线
- **常量命名**：使用 UPPER_SNAKE_CASE 命名常量
- **行长度**：保持代码行长度合理，避免过长

### 10.4 调试技巧 | Debugging Tips

- **使用 `renpy.trace()`**：跟踪变量值
- **使用 `print()`**：在 Python 代码中打印调试信息
