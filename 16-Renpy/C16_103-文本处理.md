## 1. 文本基础 (Text Basics)

### 1.1 基本文本显示

在 Ren'Py 中，文本是游戏与玩家交流的主要方式。以下是基本文本显示的示例：

```renpy
# ==========================================
# 示例 1-1: 旁白文本
# ==========================================
# 这是最简单的文本形式，称为"旁白"
# 旁白用于描述场景、环境或叙述故事
# 注意：旁白文本不需要引号前面加角色名
"这是一段旁白文本。"

# 执行流程:
# 1. Ren'Py 解析到 "..." 语法
# 2. 使用默认的 narrator 角色显示文本
# 3. 文本以打字机效果逐字显示
# 4. 玩家点击后继续下一行

# ==========================================
# 示例 1-2: 角色对话
# ==========================================
# 定义角色后，可以使用角色名直接说话
# 语法: 角色名 "对话内容"
eileen "你好，欢迎来到 Ren'Py 世界！"

# 等价于:
narrator "Eileen 说: 你好，欢迎来到 Ren'Py 世界！"

# 角色对话的执行流程:
# 1. 解析角色名 "eileen"
# 2. 查找之前定义的 Character 对象
# 3. 显示角色名称（带颜色和样式）
# 4. 显示对话内容（带打字机效果）
# 5. 如果设置了 voice，播放语音文件

# ==========================================
# 示例 1-3: 多行文本
# ==========================================
eileen "这是第一行文本。\n这是第二行文本。"
# \n 是换行符，将文本分成两行显示

# 解释:
# - \n 是一个转义序列，表示"换行"
# - Ren'Py 会识别这个序列并在该位置换行
# - 适用于旁白和对话两种形式
```

### 1.2 文本变量

变量可以在文本中动态显示，让你的游戏更具交互性：

```renpy
# ==========================================
# 示例 1-4: 基本变量使用
# ==========================================
$ name = "玩家"          # 定义字符串变量
$ score = 100            # 定义数字变量
$ health = 75.5          # 定义浮点数变量

# 变量在文本中使用方括号 [] 包裹
# 方括号内的内容会被求值并显示
eileen "你好，[name]！"
# 输出: "你好，玩家！"

eileen "你的分数是 [score]。"
# 输出: "你的分数是 100。"

# ==========================================
# 示例 1-5: 表达式求值
# ==========================================
# 可以在 [] 中使用任意 Python 表达式
eileen "1 + 2 = [1 + 2]"
# 输出: "1 + 2 = 3"
# Ren'Py 会先计算 1 + 2，然后将结果 3 插入文本

eileen "你的名字长度是 [len(name)]。"
# 输出: "你的名字长度是 2。"
# len() 是 Python 内置函数，计算字符串长度

eileen "两年后你的分数是 [score + 2]。"
# 输出: "两年后你的分数是 102。"

# ==========================================
# 示例 1-6: 复杂表达式
# ==========================================
$ item = "剑"
$ damage = 50
$ critical = 2.5

# 字符串格式化
eileen "你获得了 [item]！"
# 输出: "你获得了 剑！"

# 数学运算
eileen "武器伤害: [damage * critical]"
# 输出: "武器伤害: 125.0"
# 50 * 2.5 = 125.0

# 方法调用
$ text = "hello world"
eileen "大写: [text.upper()]"
# 输出: "大写: HELLO WORLD"
```

## 2. 文本样式 (Text Styles)

### 2.1 内联样式

内联样式允许你直接在文本中设置格式，不需要预先定义样式类：

```renpy
# ==========================================
# 示例 2-1: 基本样式标签
# ==========================================

# 粗体标签 {b}...{/b}
eileen "这是{b}粗体{/b}文本。"
# 解释:
# - {b} 开启粗体模式
# - ... 是应用粗体的文本
# - {/b} 关闭粗体模式
# 注意: 每个开启标签都必须有对应的关闭标签

# 斜体标签 {i}...{/i}
eileen "这是{i}斜体{/i}文本。"

# 下划线标签 {u}...{/u}
eileen "这是{u}下划线{/u}文本。"

# 删除线标签 {s}...{/s}
eileen "这是{s}删除线{/s}文本。"

# ==========================================
# 示例 2-2: 颜色和大小
# ==========================================

# 颜色标签 {color=#rrggbb}...{/color}
# 格式: #rrggbb，其中 rr、gg、bb 是 16 进制数 (00-ff)
eileen "这是{color=#ff0000}红色{/color}文本。"
# #ff0000 = 红色 (255, 0, 0)

eileen "这是{color=#00ff00}绿色{/color}文本。"
# #00ff00 = 绿色 (0, 255, 0)

eileen "这是{color=#0000ff}蓝色{/color}文本。"
# #0000ff = 蓝色 (0, 0, 255)

eileen "这是{color=#ff00ff}紫色{/color}文本。"
# #ff00ff = 紫色 (255, 0, 255)

# 大小标签 {size=像素值}...{/size}
eileen "这是{size=30}大号{/size}文本。"
eileen "这是{size=16}小号{/size}文本。"

# ==========================================
# 示例 2-3: 字体和透明度
# ==========================================

# 字体标签 {font=路径}...{/font}
eileen "这是{font=fonts/NotoSans.ttf}特殊字体{/font}文本。"
# 注意: 字体文件需要放在 game 目录下
# 推荐使用 .ttf 或 .otf 格式

# 透明度标签 {alpha=0.0-1.0}...{/alpha}
eileen "这是{alpha=0.5}半透明{/alpha}文本。"
# alpha 值范围: 0.0 (完全透明) 到 1.0 (完全不透明)

# ==========================================
# 示例 2-4: 组合样式
# ==========================================
# 样式标签可以组合使用
eileen "这是{b}{color=#ff0000}粗体红色{/color}{/b}文本。"
# 执行顺序从内到外:
# 1. 先应用 color，文字变红
# 2. 再应用 b，红色文字变粗

eileen "这是{i}{size=24}斜体大号{/size}{/i}文本。"
# 1. 先应用 size，文字变大
# 2. 再应用 i，变大文字变斜

eileen "{b}{i}{color=#ff0000}粗体斜体红{/color}{/i}{/b}"
# 三种样式组合: 粗体 + 斜体 + 红色

# ==========================================
# 示例 2-5: 特殊标签
# ==========================================

# 上标标签 {sup}...{/sup}
eileen "x{sup}2{/sup} = 4"
# 输出: x² = 4

# 下标标签 {sub}...{/sub}
eileen "H{sub}2{/sub}O"
# 输出: H₂O

# 换行标签 {br}
eileen "第一行{br}第二行"
# 输出: 第一行（换行）第二行

# 不换行标签 {nw}
eileen "这是一段{nw}很长的文本"
# nw = "no wrap"，防止文本换行
```

### 2.2 文本样式标签详解

| 标签 | 描述 | 参数格式 | 示例 |
| :--- | :--- | :--- | :--- |
| **b** | 粗体 | 无 | `{b}文字{/b}` |
| **i** | 斜体 | 无 | `{i}文字{/i}` |
| **u** | 下划线 | 无 | `{u}文字{/u}` |
| **s** | 删除线 | 无 | `{s}文字{/s}` |
| **sup** | 上标 | 无 | `{sup}2{/sup}` |
| **sub** | 下标 | 无 | `{sub}2{/sub}` |
| **color** | 颜色 | `#rrggbb` 或颜色名 | `{color=#ff0000}文字{/color}` |
| **size** | 字体大小 | 像素值 | `{size=24}文字{/size}` |
| **font** | 字体文件 | 文件路径 | `{font=myfont.ttf}文字{/font}` |
| **alpha** | 透明度 | 0.0-1.0 | `{alpha=0.5}文字{/alpha}` |
| **pos** | 位置 | `x, y` | `{pos=100, 50}文字{/pos}` |
| **nw** | 不换行 | 无 | `{nw}文字{/nw}` |
| **br** | 换行 | 无 | `第一行{br}第二行` |

## 3. 高级文本功能

### 3.1 文本速度控制

```renpy
# ==========================================
# 示例 3-1: 文本显示速度
# ==========================================

# 默认文本速度由 config.default_text_cps 控制
# cps = characters per second (每秒字符数)
define config.default_text_cps = 30  # 默认每秒30字符

# 快速显示 - 使用 with fast
eileen "这是快速显示的文本。" with fast
# with fast 会立即显示完整文本，没有打字机效果

# 慢速显示 - 使用 with slow
eileen "这是慢速显示的文本。" with slow
# with slow 会使用比默认更慢的速度显示

# ==========================================
# 示例 3-2: 自定义文本速度
# ==========================================

# 通过 Character 定义自定义速度
define slow_eileen = Character(
    "艾琳",
    what_slow_cps=10  # 说话时每10字符每秒
)

define fast_eileen = Character(
    "艾琳",
    what_slow_cps=100  # 说话时每100字符每秒
)

slow_eileen "这段文字显示得很慢..."
fast_eileen "这段文字显示得很快！"

# ==========================================
# 示例 3-3: 暂停和继续
# ==========================================

# {p} 标签 - 暂停等待点击后继续
eileen "第一段文字{p}"
eileen "点击后显示这一段"

# {p} 等价于 {nw}\n\n
# nw = no wait，点击后换行继续

# {d} 标签 - 暂停指定时间
eileen "等{d=1.0}待一秒钟" with dissolve
# d=1.0 表示暂停1秒
```

### 3.2 文本效果

```renpy
# ==========================================
# 示例 3-4: 文本转场效果
# ==========================================

# dissolve - 溶解效果
eileen "文字溶解显示。" with dissolve
# 效果: 文字从透明渐变到不透明

# fade - 淡入淡出
eileen "文字淡入显示。" with fade
# 效果: 淡出然后淡入

# hpunch/vpunch - 震动效果
eileen "震惊！" with hpunch  # 水平震动
eileen "惊讶！" with vpunch  # 垂直震动

# ==========================================
# 示例 3-5: 文本回调函数
# ==========================================

init python:
    # 定义回调函数
    def on_text_show(event, interact=True, **kwargs):
        """
        文本开始显示时的回调
        event: 事件名称
        interact: 是否等待玩家交互
        """
        if event == "show":
            print("开始显示文本")

    def on_text_end(event, interact=True, **kwargs):
        """
        文本显示结束时的回调
        """
        if event == "end":
            print("文本显示完毕")

    def on_text_replace(event, interact=True, **kwargs):
        """
        替换旧文本时的回调
        """
        if event == "replace":
            print("正在替换文本")

# 使用回调
eileen "带回调的文本。" callback=on_text_end
```

## 5. 自定义文本标签

### 5.1 基础自定义标签

```renpy
# ==========================================
# 示例 5-1: 注册简单颜色标签
# ==========================================

init python:
    def colorize_tag(tag, argument, contents):
        """
        自定义文本标签处理器
        tag: 标签名 (如 "red")
        argument: 标签参数 (如 "#ff0000")
        contents: 标签内容 (包含样式的文本)
        """
        return f"{{color={argument}}}{contents}{{/color}}"

    # 注册自定义标签
    config.custom_text_tags["color"] = colorize_tag

# 使用自定义标签
eileen "这是{color=#ff0000}红色{/color}文字。"
eileen "这是{color=#00ff00}绿色{/color}文字。"
eileen "这是{color=#0000ff}蓝色{/color}文字。"

# ==========================================
# 示例 5-2: 预定义颜色标签
# ==========================================

init python:
    # 创建颜色标签的辅助函数
    def make_color_tag(color):
        def tag_handler(tag, argument, contents):
            return f"{{color={color}}}{contents}{{/color}}"
        return tag_handler

    # 注册多个预定义颜色标签
    config.custom_text_tags["red"] = make_color_tag("#ff0000")
    config.custom_text_tags["blue"] = make_color_tag("#0000ff")
    config.custom_text_tags["green"] = make_color_tag("#00ff00")
    config.custom_text_tags["gold"] = make_color_tag("#ffd700")

# 使用预定义标签
eileen "奖励: {gold}1000 金币{/gold}！"
eileen "{red}警告：危险！{/red}"
```

### 5.2 带参数的自定义标签

```renpy
# ==========================================
# 示例 5-3: 大小标签
# ==========================================

init python:
    def size_tag_handler(tag, argument, contents):
        """
        处理 {size=30}...{/size} 标签
        argument: 大小值 (字符串 "30")
        contents: 标签内容
        """
        return f"{{size={argument}}}{contents}{{/size}}"

    config.custom_text_tags["size"] = size_tag_handler

# 使用
eileen "大字 {size=36}很大{/size} 正常 {size=12}很小{/size}"

# ==========================================
# 示例 5-4: 阴影和轮廓效果
# ==========================================

init python:
    def shadow_tag(tag, argument, contents):
        """
        {shadow=颜色}...{/shadow} 添加阴影
        """
        color = argument if argument else "#000000"
        # Ren'Py 不直接支持阴影，使用轮廓模拟
        text = f"{{color={color}}}{contents}{{/color}}"
        return text

    def outline_tag(tag, argument, contents):
        """
        {outline=颜色}...{/outline} 添加轮廓
        """
        color = argument if argument else "#000000"
        # 使用 what_outlines 模拟
        return f"{{color={color}}}{contents}{{/color}}"

    config.custom_text_tags["shadow"] = shadow_tag
    config.custom_text_tags["outline"] = outline_tag

# ==========================================
# 示例 5-5: 特殊效果标签
# ==========================================

init python:
    def glow_tag(tag, argument, contents):
        """
        {glow=颜色}...{/glow} 发光效果
        """
        color = argument if argument else "#ffffff"
        # 使用 alpha 模拟发光
        return f"{{alpha=0.8}}{contents}{{/alpha}}"

    config.custom_text_tags["glow"] = glow_tag

# ==========================================
# 示例 5-6: 语音标签
# ==========================================

init python:
    def voice_tag(tag, argument, contents):
        """
        {voice=文件名} 播放语音
        这个标签会在显示时自动播放指定语音文件
        """
        renpy.play(f"voice/{argument}", "voice")
        return contents

    config.custom_text_tags["voice"] = voice_tag

# 使用
eileen "{voice=hello.mp3}你好！{/voice}"
```

## 7. 特殊文本功能

### 7.1 文本输入

```renpy
# ==========================================
# 示例 7-1: 基本文本输入
# ==========================================

# renpy.input() 函数显示一个输入对话框
# 返回玩家输入的字符串
$ player_name = renpy.input("请输入你的名字：")

# .strip() 移除首尾空白
# or "玩家" 提供默认值（如果输入为空）
$ player_name = player_name.strip() or "玩家"
```

### 8.3 打字机效果实现

```renpy
# ==========================================
# 示例 8-5: 自定义打字机效果
# ==========================================

init python:
    class TypewriterText(renpy.Displayable):
        """
        自定义打字机效果显示对象
        """
        def __init__(self, text, speed=0.05, **kwargs):
            super(TypewriterText, self).__init__(**kwargs)
            self.text = text
            self.speed = speed  # 每个字符之间的延迟（秒）
            self.shown_text = ""
            self.current_index = 0
            self.start_time = None

        def render(self, width, height, st, at):
            if self.start_time is None:
                self.start_time = st

            elapsed = st - self.start_time
            self.current_index = min(
                int(elapsed / self.speed),
                len(self.text)
            )
            self.shown_text = self.text[:self.current_index]

            return renpy.render(
                Text(self.shown_text),
                width, height, st, at
            )

        def event(self, ev, x, y, st):
            if ev.type == renpy.event.TOUCHUP:
                # 点击后立即显示完整文本
                self.current_index = len(self.text)
                self.shown_text = self.text
                raise renpy.IgnoreEvent()

# ==========================================
# 示例 8-6: 使用自定义打字机
# ==========================================

label typewriter_demo:
    $ tw = TypewriterText("这是一段打字机效果的文本...", speed=0.08)
    show expression tw at center
    pause
```

## 9. 性能优化

### 9.1 文本性能建议

```renpy
# ==========================================
# 性能优化指南
# ==========================================

# 1. 避免在热路径中使用复杂表达式
# 不推荐:
eileen "计算结果: [len([i for i in range(100000)])]"

# 推荐: 预先计算
$ result = len([i for i in range(100000)])
eileen "计算结果: [result]"

# 2. 减少内联样式标签
# 不推荐: 每个字都单独设置颜色
eileen "{color=#ff0000}红{/color}{color=#ff0000}红{/color}{color=#ff0000}红{/color}..."

# 推荐: 使用一个标签包含所有文字
eileen "{color=#ff0000}红色文字{/color}"

# 3. 预定义常用样式
define accent_text = "{color=#ffd700}"  # 金色

eileen "[accent_text]重要信息！[/color]"

# 4. 限制历史记录大小
```
