## 1. 转场效果概述 (Transitions Overview)

### 1.1 什么是转场

转场效果是 Ren'Py 中用于创建平滑过渡动画的工具。它们可以在以下情况使用：

- **场景切换** (`scene` 语句)
- **图像显示/隐藏** (`show`/`hide` 语句)
- **对话显示** (`say` 语句)
- **菜单显示** (`menu` 语句)
- **屏幕显示/隐藏** (`show screen`/`hide screen`

### 1.2 转场的执行流程

```renpy
# ==========================================
# 示例 1-1: 转场的执行流程
# ==========================================

scene bg room with fade
# 执行流程:
# 1. 获取当前显示的图像（bg room 之前的背景）
# 2. 获取即将显示的图像（bg room）
# 3. 创建转场效果（fade）
# 4. 在指定时间内从旧图像过渡到新图像
# 5. 移除旧图像，显示新图像

# ==========================================
# 示例 1-2: with 语句的作用
# ==========================================

# with 语句用于指定转场效果
show eileen happy with dissolve
#         ^^^^^^^^^ with + 转场效果

# dissolve 是内置的 Dissolve() 转场实例
# 等价于: show eileen happy with Dissolve(0.5)
```

## 2. 基本转场效果 (Basic Transitions)

### 2.1 内置转场详解

| 转场 | 类 | 描述 | 时间 |
| :--- | :--- | :--- | :--- |
| **dissolve** | Dissolve | 溶解效果，从透明渐变 | 0.5秒 |
| **fade** | Fade | 淡入淡出 | 0.5秒 |
| **fadeout** | Fade | 淡出（仅淡出） | 0.5秒 |
| **wipeleft** | CropMove | 从右向左擦除 | 0.5秒 |
| **wiperight** | CropMove | 从左向右擦除 | 0.5秒 |
| **wipeup** | CropMove | 从下向上擦除 | 0.5秒 |
| **wipedown** | CropMove | 从上向下擦除 | 0.5秒 |
| **slideleft** | CropMove | 从右向左滑动 | 0.5秒 |
| **slideright** | CropMove | 从左向右滑动 | 0.5秒 |
| **slideup** | CropMove | 从下向上滑动 | 0.5秒 |
| **slidedown** | CropMove | 从上向下滑动 | 0.5秒 |
| **vpunch** | Move | 垂直震动 | 0.15秒 |
| **hpunch** | Move | 水平震动 | 0.15秒 |
| **zoom** | ZoomFade | 缩放淡入 | 0.5秒 |

### 2.2 dissolve（溶解）

```renpy
# ==========================================
# 示例 2-1: dissolve 溶解效果
# ==========================================

# 基本用法
scene bg room with dissolve
# 效果: 当前场景逐渐溶解，显示新场景
# 原理: 旧图像像素逐渐变为新图像像素

show eileen happy with dissolve
# 效果: 角色图像从透明渐变到完全不透明

# ==========================================
# 示例 2-2: 自定义溶解时间
# ==========================================

# 快速溶解（0.3秒）
show eileen happy with Dissolve(0.3)

# 慢速溶解（1.5秒）
scene bg park with Dissolve(1.5)
```

### 2.3 fade（淡入淡出）

```renpy
# ==========================================
# 示例 2-3: fade 淡入淡出
# ==========================================

# 基本用法
scene bg room with fade
# 效果: 先淡出到黑色，再从黑色淡入到新场景

# ==========================================
# 示例 2-4: 自定义 Fade 参数
# ==========================================

# Fade(fadeout时间, 暂停时间, fadein时间)
# 创建一个先淡出(1秒), 暂停(0.5秒), 再淡入(1秒)的转场
define slow_fade = Fade(1.0, 0.5, 1.0)

scene bg room with slow_fade

# ==========================================
# 示例 2-5: 淡出到指定颜色
# ==========================================

# 淡出到红色
define red_fade = Fade(0.5, 0.0, 0.5, color="#ff0000")
scene bg room with red_fade

# 淡出到黑色（默认）
define black_fade = Fade(0.5, 0.0, 0.5, color="#000000")
scene bg room with black_fade
```

### 2.4 wipe（擦除）

```renpy
# ==========================================
# 示例 2-6: wipe 系列擦除效果
# ==========================================

# wipeleft: 从右向左擦除
scene bg room with wipeleft
# 效果: 新图像从右向左覆盖旧图像

# wiperight: 从左向右擦除
scene bg room with wiperight
# 效果: 新图像从左向右覆盖旧图像

# wipeup: 从下向上擦除
scene bg room with wipeup

# wipedown: 从上向下擦除
scene bg room with wipedown

# ==========================================
# 示例 2-7: 自定义擦除
# ==========================================

# CropMove(time, "wipe方向")
define custom_wipe = CropMove(1.0, "wipeleft")
scene bg room with custom_wipe
```

### 2.5 slide（滑动）

```renpy
# ==========================================
# 示例 2-8: slide 系列滑动效果
# ==========================================

# slideleft: 从右向左滑动（新图像滑入）
scene bg room with slideleft
# 与 wipeleft 的区别: slide 是新图像滑入，wipe 是覆盖

# slideright: 从左向右滑动
scene bg room with slideright

# slideup: 从下向上滑动
scene bg room with slideup

# slidedown: 从上向下滑动
scene bg room with slidedown
```

### 2.6 punch（震动）

```renpy
# ==========================================
# 示例 2-9: vpunch 和 hpunch 震动效果
# ==========================================

# vpunch: 垂直震动（上下抖动）
show eileen angry with vpunch
# 常用于: 震惊、惊讶、打击等情绪表达

# hpunch: 水平震动（左右抖动）
show eileen surprised with hpunch
# 常用于: 物理冲击、意外事件等

# ==========================================
# 示例 2-10: 震动效果原理
# ==========================================

# vpunch 的内部实现:
# - 向上移动 10 像素
# - 回到原位
# - 向下移动 10 像素
# - 回到原位
# 总时间约 0.15 秒
```

## 3. 自定义转场 (Custom Transitions)

### 3.1 转场类

| 转场类 | 描述 | 主要参数 |
| :--- | :--- | :--- |
| **Dissolve** | 溶解效果 | time, alpha |
| **Fade** | 淡入淡出 | fadeout, pause, fadein, color |
| **CropMove** | 裁剪移动 | time, mode, start, end |
| **Move** | 移动效果 | time, start, end |
| **ZoomFade** | 缩放淡入 | time, zoom |
| **Pan** | 平移效果 | time, start, end |
| **MoveTransition** | 组合转场 | time, enter, leave |
| **ComposeTransition** | 组合转场 | transition1, transition2 |

### 3.2 Dissolve（溶解）

```renpy
# ==========================================
# 示例 3-1: Dissolve 转场
# ==========================================

# Dissolve(time, alpha)
# time: 持续时间（秒）
# alpha: 溶解的透明度

# 快速溶解
define quick_dissolve = Dissolve(0.3)

# 慢速溶解
define slow_dissolve = Dissolve(1.5)

# 部分溶解（alpha 控制）
define partial_dissolve = Dissolve(0.5, alpha=0.8)

# 使用
show eileen happy with quick_dissolve
```

### 3.3 Fade（淡入淡出）

```renpy
# ==========================================
# 示例 3-2: Fade 转场参数详解
# ==========================================

# Fade(fadeout时间, 暂停时间, fadein时间, 颜色)
# 参数详解:
# - fadeout时间: 淡出到指定颜色的时间
# - 暂停时间: 在指定颜色处停留的时间
# - fadein时间: 从指定颜色淡入到最终图像的时间
# - 颜色: 过渡时显示的颜色（默认黑色）

# 快速淡入淡出
define fast_fade = Fade(0.3, 0.0, 0.3)

# 标准淡入淡出
define standard_fade = Fade(0.5, 0.0, 0.5)

# 戏剧性淡入淡出（淡出后停留）
define dramatic_fade = Fade(1.0, 0.5, 1.0)

# 白色淡入淡出（闪光效果）
define white_flash = Fade(0.2, 0.1, 0.3, color="#ffffff")

# 场景应用
scene bg room with standard_fade
```

### 3.4 CropMove（裁剪移动）

```renpy
# ==========================================
# 示例 3-3: CropMove 转场模式
# ==========================================

# CropMove(time, mode, start, end)
# mode: 移动模式
# start: 起始位置
# end: 结束位置

# 常用模式:
# "wipeleft" - 从右向左擦除
# "wiperight" - 从左向右擦除
# "wipeup" - 从下向上擦除
# "wipedown" - 从上向下擦除
# "slideleft" - 从右向左滑动
# "slideright" - 从左向右滑动
# "slideup" - 从下向上滑动
# "slidedown" - 从上向下滑动

# 自定义擦除速度
define slow_wipe = CropMove(1.5, "wipeleft")
define fast_slide = CropMove(0.3, "slideleft")

# ==========================================
# 示例 3-4: 自定义起止位置
# ==========================================

# CropMove(time, "模式", start, end)
# start/end 格式: (x, y, w, h) 表示矩形区域

# 从右下角开始
define corner_wipe = CropMove(0.5, "wipeleft", start=(0.5, 0.5, 1.0, 1.0)

# 水平分割擦除
define split_wipe = CropMove(0.5, "wipeleft", start=(0.0, 0.0, 1.0, 0.5))
```

### 3.5 Move（移动）

```renpy
# ==========================================
# 示例 3-5: Move 转场
# ==========================================

# Move(time, start, end)
# start: 起始位置 (x, y)
# end: 结束位置 (x, y)

# 简单移动
define slide_in = Move(0.5, start=(1.0, 0.0), end=(0.0, 0.0))
show eileen happy with slide_in

# 抖动效果
define shake = Move(0.1, start=(0.0, 0.0), end=(0.05, 0.0))

# ==========================================
# 示例 3-6: 震动效果
# ==========================================

# vpunch 和 hpunch 是预定义的 Move 转场
# vpunch: 垂直震动
# hpunch: 水平震动

# 自定义震动
define custom_vpunch = Move(0.15, start=(0.0, 0.0), end=(0.0, -0.03))
define custom_hpunch = Move(0.15, start=(0.0, 0.0), end=(-0.05, 0.0))
```

## 4. 转场组合 (Transition Composition)

### 4.1 ComposeTransition

```renpy
# ==========================================
# 示例 4-1: 组合两个转场
# ==========================================

# ComposeTransition(第一个转场, 第二个转场)
# 先执行第一个转场，再执行第二个转场

# 先震动后溶解
define punch_then_dissolve = ComposeTransition(hpunch, dissolve)
show eileen angry with punch_then_dissolve

# 先淡出后淡入
define fade_out_in = ComposeTransition(fadeout, fadein)
scene bg room with fade_out_in
```

### 4.2 MoveTransition

```renpy
# ==========================================
# 示例 4-2: 带移动的转场
# ==========================================

# MoveTransition(time, enter, leave, duration)
# enter: 进入时的变换
# leave: 离开时的变换

# 旧图像向左滑出，新图像从右滑入
define complex_slide = MoveTransition(0.5, enter=Move(0.5, start=(1.0, 0.0), end=(0.0, 0.0)), leave=Move(0.5, start=(0.0, 0.0), end=(-1.0, 0.0))))

scene bg room with complex_slide
```

## 7. 转场与 ATL 结合

### 7.1 转场与变换

```renpy
# ==========================================
# 示例 7-1: ATL 变换与转场结合
# ==========================================

# 变换定义
transform slide_from_right:
    xalign 1.0
    linear 0.5 xalign 0.0

transform fade_and_scale:
    alpha 0.0 zoom 1.5
    linear 0.5 alpha 1.0 zoom 1.0

# 使用变换作为转场
show eileen happy:
    at slide_from_right
with dissolve

# ==========================================
# 示例 7-2: 带变换的 show 语句
# ==========================================

show eileen happy at center with dissolve
# 解析:
# 1. at center: 应用位置变换
# 2. with dissolve: 应用溶解转场

show eileen happy at slide_from_right with fade
# 先应用滑动变换，同时应用淡入淡出效果
```

## 8. 性能考虑

### 8.1 转场性能对比

| 转场类型 | 性能消耗 | 适用场景 |
| :--- | :--- | :--- |
| **dissolve** | 中等 | 常规切换 |
| **fade** | 低 | 强调过渡 |
| **wipe/slide** | 低 | 线性移动 |
| **hpunch/vpunch** | 极低 | 震动效果 |
| **zoom** | 中等 | 强调效果 |

### 8.2 优化建议

```renpy

# ==========================================

# 不推荐:

# 推荐: 合并转场
# 使用组合转场

# 2. 使用轻量级转场
# 对于不重要的切换，使用 hpunch/vpunch

# 3. 预定义常用转场

```

## 9. 实际应用示例

### 9.1 完整场景切换示例

```renpy
# ==========================================
# 示例 9-1: 场景切换脚本
# ==========================================

# 定义常用转场
define fade_trans = Fade(0.5, 0.0, 0.5)
define slow_fade = Fade(1.0, 0.5, 1.0)
define wipe_trans = wipeleft

# 场景: 室内 -> 公园
label room_to_park():
    # 离开室内
    hide eileen with dissolve
    scene bg room with fade_trans

    # 过渡
    pause 0.5

    # 到达公园
    scene bg park with wipe_trans
    show eileen happy at center with dissolve

    return

# ==========================================
# 示例 9-2: 情绪变化脚本
# ==========================================

label emotional_change():
    show eileen happy at center with dissolve
    e "今天真开心！"

    # 意外消息
    show eileen surprised with hpunch
    e "什么？！"

    # 悲伤
    show eileen sad with Fade(0.5, 0.0, 0.5)
    e "怎么会这样..."

    return
```

## 11. 总结

### 11.1 关键要点

- **转场是平滑过渡的关键: 使用 `with` 语句应用转场
- **内置转场足够大部分需求: dissolve、fade、wipe、slide、punch
- **可以自定义转场参数: 通过创建转场实例
- **组合转场创造复杂效果**: ComposeTransition、MoveTransition
- **根据场景选择转场**: 情绪变化用震动，场景切换用淡入淡出
- **性能考虑**: 简单转场性能更好

### 11.2 学习建议

- **掌握内置转场**: 熟悉 dissolve、fade、wipe、slide 的效果
- **理解转场原理**: 溶解、淡出、移动的基本原理
- **练习组合转场**: 使用 ComposeTransition 创造独特效果
- **根据情绪选择**: 不同情绪使用不同转场
- **注意性能**: 避免短时间内多次转场
