## 1. 存档系统概述

### 1.1 存档内容

Ren'Py 存档包括：

- 当前语句和主控流程路径
- 显示的图像和可视组件
- 界面及界面内所有变量值
- 正在播放的音乐
- NVL 模式文本块列表
- 所有变动的 Python 变量

### 1.2 不存档内容

- 游戏开始后未改变的 Python 变量
- 配置项变量和样式
- 主控流程路径
- 渲染对象、迭代器、生成器
- 网络 socket 端口

## 2. 存档操作

### 2.1 自动存档

```renpy
# 自动存档点
label start():
    # 每次用户互动前自动检查存档

# 配置自动存档
define config.auto_save_after_input = True
define config.auto_save_on_mobile_background = True
```

### 2.2 手动存档

```renpy
# 存档
renpy.save(slot_name)

# 带描述的存档
renpy.save("slot1", "第一章结束")

# 快速存档
renpy.quick_save()
renpy.quick_save("quick")
```

### 2.3 读取存档

```renpy
# 读取指定存档
renpy.load(slot_name)

# 快速读档
renpy.quick_load()
renpy.quick_load("quick")
```

### 2.4 检查存档存在

```renpy
if renpy.save_exists(slot_name):
    "存档存在，可以加载"
else:
    "没有存档"
```

### 2.5 删除存档

```renpy
renpy.delete_save(slot_name)
```

## 4. 回滚功能

### 4.1 启用/禁用回滚

```renpy
# 禁用回滚
define config.rollback_enabled = False

# 启用回滚（默认）
define config.rollback_enabled = True
```

### 4.2 回滚限制

```renpy
# 最大回滚步数
define config.rollback_length = 50

# 回滚回调
init python:
    def my_rollback_callback():
        renpy.notify("已回滚到之前的状态")

    define config.rollback_callbacks.append(my_rollback_callback)
```

### 4.3 强制回滚

```renpy
# 回滚一步
renpy.rollback()

# 带检查的回滚
if renpy.can_rollback():
    renpy.rollback()
```

## 5. 持久化数据

### 5.1 persistent 对象

```renpy
# 设置持久化数据
$ persistent.best_score = 1000
$ persistent.unlocked_cg = [ "cg1.png", "cg2.png" ]
$ persistent.played_routes = ["eileen", "sophie"]

# 读取持久化数据
if persistent.best_score > 500:
    "恭喜你创造了新高分！"

# 初始化持久化数据
init python:
    if not hasattr(persistent, "visited"):
        persistent.visited = False
```

### 5.2 持久化回调

```renpy
# 游戏启动时检查持久化数据
init python:
    def check_persistent():
        if persistent.first_run:
            persistent.first_run = False
            renpy.notify("欢迎首次游玩！")

    config.start_callbacks.append(check_persistent)
```

## 6. 存档优化

### 6.1 优化存储大小

```renpy
# 减少截图尺寸
define config.thumbnail_height = 40

# 排除大对象
init python:
    def save_filter(name, obj):
        if hasattr(obj, "large_data"):
            return False
        return True

    config.save_id_map_filter = save_filter
```

### 6.2 延迟存档

```renpy
# 在非关键时刻存档
label save_point():
    if renpy.can_save():
        renpy.save("auto_save")
    return
```

### 6.3 存档兼容性

```renpy
# 检查存档版本
init python:
    def check_save_compatibility(slot):
        if "version" not in slot:
            return False
        return slot["version"] == config.version

    config.load_save_callback = check_save_compatibility
```

## 9. 最佳实践

1. **定期存档点**: 在关键剧情点设置存档点
2. **回滚限制**: 在某些小游戏或交互中禁用回滚
3. **持久化数据**: 用于存储游戏设置、解锁内容、统计数据
4. **存档版本**: 在存档中存储版本号，确保兼容性
5. **避免别名**: 不要在存档和不存档变量间建立别名关系
6. **截图优化**: 适当减小截图尺寸以节省空间
