## 1. 核心概念 | Core Concepts

### 1.1 脚本与语言

| 术语 | 英文 | 定义 |
| :--- | :--- | :--- |
| 标签 | Label | 脚本中的命名位置，用于跳转和控制流程 |
| 语句 | Statement | Ren'Py 脚本的基本执行单元 |
| 角色 | Character | 定义对话行为的对象，控制谁说什么 |
| 场景 | Scene | 一组可视组件的集合，通常指代游戏中的一个状态 |
| 对话 | Dialogue | 角色说话时显示的文本 |

### 1.2 可视组件

| 术语 | 英文 | 定义 |
| :--- | :--- | :--- |
| 可视组件 | Displayable | 可在屏幕上显示的对象基类 |
| 变换 | Transform | 控制可视组件的位置、旋转、缩放等属性 |
| ATL | ATL (Animation and Transform Language) | 用于定义动画和变换的专用语言 |
| 转场 | Transition | 两个场景或状态之间的过渡效果 |
| 图层 | Layer | 图像显示的层次结构 |

### 1.3 界面系统

| 术语 | 英文 | 定义 |
| :--- | :--- | :--- |
| 屏幕 | Screen | 由可视组件组成的用户界面 |
| 屏幕语言 | Screen Language | 定义屏幕的 DSL，类似于声明式语法 |
| SL2 | SL2 (Screen Language 2) | Ren'Py 7+ 的屏幕语言版本 |
| 控件 | Widget | 屏幕中的交互元素，如按钮、文本框 |
| 动作 | Action | 控件被激活时执行的操作 |

## 11. 常用函数速查 | Quick Function Reference

### 11.1 renpy 模块

```renpy
renpy.notify(message)           # 显示通知
renpy.choice_for_skipping()    # 允许跳过
renpy.can_rollback()           # 检查能否回滚
renpy.rollback()               # 执行回滚
renpy.save(slot)               # 存档
renpy.load(slot)               # 读档
renpy.transition(trans)        # 设置转场
```

### 11.2 renpy.music 模块

```renpy
renpy.music.play(filename, channel="music")     # 播放音乐
renpy.music.stop(channel="music")               # 停止音乐
renpy.music.pause(channel="music")              # 暂停音乐
renpy.music.unpause(channel="music")            # 恢复音乐
renpy.music.set_volume(channel, volume)          # 设置音量
renpy.music.is_playing(channel)                  # 检查是否播放中
```

### 11.3 renpy.screen 模块

```renpy
renpy.show_screen(screen_name, **kwargs)       # 显示屏幕
renpy.hide_screen(screen_name)                  # 隐藏屏幕
renpy.ui.remap(new_action, old_action)          # 重新映射动作
```
