## 1. 音频系统概述

### 1.1 支持的音频格式

Ren'Py 支持多种音频文件格式：

- **Opus** - Web 优化的音频格式
- **Ogg Vorbis** - 开源音频格式
- **MP3** - 通用音频格式
- **MP2** - 旧版音频格式
- **FLAC** - 无损音频格式
- **WAV** - 未压缩的有符号 16bit PCM

### 1.2 音频通道

Ren'Py 支持多个音频通道：

| 通道名 | 描述 | 用途 |
| :--- | :--- | :--- |
| `music` | 音乐通道 | 背景音乐，支持循环 |
| `sound` | 音效通道 | 短音效，只播放一次 |
| `voice` | 语音通道 | 角色对话语音 |
| `audio` | 混合通道 | 同时播放多个音频 |

## 2. play 语句

### 2.1 基本语法

```renpy
play music "opening.ogg"
play sound "click.mp3"
play voice "eileen_happy.mp3"
```

### 2.2 完整语法

```renpy
play <channel> <file> [fadeout <seconds>] [fadein <seconds>] [loop|noloop] [volume <0.0-1.0>] [if_changed]
```

### 2.3 示例

```renpy
# 基本用法
play music "bgm_main.ogg"

# 带淡入淡出
play music "battle.ogg" fadeout 1.0 fadein 1.0

# 循环播放
play music "ambient.ogg" loop

# 播放列表（顺序播放）
play music [ "a.ogg", "b.ogg", "c.ogg" ]

# 播放时检测是否正在播放同一文件
play music market if_changed
```

## 3. queue 语句

```renpy
queue music "next_track.ogg"
queue sound [ "sfx1.mp3", "sfx2.mp3" ]
```

## 4. stop 语句

```renpy
stop music fadeout 2.0
stop sound
stop voice
```

## 5. 音频命名空间

### 5.1 自动注册

存放在 `game/audio/` 目录下的音频文件会自动注册到 audio 命名空间：

```renpy
# 文件: game/audio/Town_theme.ogg
play music town_theme
```

### 5.2 手动定义

```renpy
define audio.sunflower = "my_music/sun-flower-slow-jam.ogg"
define audio.rain = "ambient/rain.ogg"

label start():
    play music sunflower
```

## 6. 语音系统

### 6.1 基本使用

```renpy
# 启用语音
define config.enable_voice = True

# 语音文件命名: 角色名_动画名.mp3
# 例如: eileen_happy.mp3, eileen_sad.mp3

voice "eileen_happy.mp3"
e "哈哈，太有趣了！"
```

### 6.2 语音前缀

```renpy
# 为特定角色设置语音前缀
define config.voice_filename_format = "{filename}"
```

## 7. 音量控制

### 7.1 全局音量

```renpy
$ renpy.music.set_volume(0.5, channel='music')
$ renpy.music.set_volume(0.8, channel='sound')
```

### 7.2 通道音量

```renpy
play music "bgm.ogg" volume 0.7
play sound "sfx.mp3" volume 0.5
```

## 8. 音频函数

### 8.1 常用函数

```renpy
# 检查通道是否正在播放
if renpy.music.is_playing('music'):
    "音乐正在播放"

# 获取当前播放的音乐
current = renpy.music.get_playing('music')

# 暂停/恢复
renpy.music.pause('music')
renpy.music.unpause('music')

# 停止所有音频
renpy.music.stop('music')
```

### 8.2 播放状态

```renpy
# 注册新通道
renpy.music.register_channel("my_channel")

# 设置通道音量
renpy.music.set_volume(0.5, 0, "my_channel")

# 设置通道混音
renpy.music.set_mixer("my_channel", "sfx")
```

## 9. 视频播放

### 9.1 Movie 控件

```renpy
# 定义视频
image movie = Movie(play="video/opening.ogv", size=(1920, 1080))

# 显示视频
show movie

# 带声音的视频
image movie_with_sound = Movie(play="video/opening.ogv", size=(1920, 1080), audio="video/audio.ogg")
```

### 9.2 播放控制

```renpy
# 在屏幕语言中
screen video_screen():
    add "video_frame":
        xalign 0.5 yalign 0.5

    textbutton "暂停":
        xalign 0.5 yalign 0.9
        action PauseAudio("movie")
```

## 10. 最佳实践

1. **音频文件管理**: 将音频文件组织在 `game/audio/` 目录下
2. **使用 audio 命名空间**: 通过 define 语句定义音频变量
3. **淡入淡出**: 切换音乐时使用淡入淡出效果
4. **语音文件命名**: 遵循 `{character}_{action}.mp3` 格式
5. **性能优化**: 预加载常用音频，避免游戏过程中加载卡顿
6. **Web 音频**: 针对 Web 平台测试音频兼容性
