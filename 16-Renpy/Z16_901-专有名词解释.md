# Ren'Py 专有名词解释

## 1. 基础概念类

### 1.1 Ren'Py

**英文名称**：Ren'Py

**中文译名**：Ren'Py 视觉小说引擎

**出现位置**：C16_101-概述与原理.md 第1章

**定义**：
Ren'Py 是一个专门用于开发视觉小说的免费开源引擎，基于 Python 开发，由 Tom Rothamel 于 2004 年创建。

### 1.2 视觉小说

**英文名称**：Visual Novel

**中文译名**：视觉小说

**出现位置**：C16_101-概述与原理.md 第1.1章

**定义**：
视觉小说是一种以文字叙述为主、结合图像、声音和选择的互动叙事游戏类型。

### 1.3 Ren'Py Script

**英文名称**：Ren'Py Script Language

**中文译名**：Ren'Py脚本语言

**出现位置**：C16_101-概述与原理.md 第1章

**定义**：
Ren'Py Script 是 Ren'Py 的专用脚本语言，语法简洁，基于 Python，用于编写游戏剧情和对话。

### 1.4 ATL

**英文名称**：Animation and Transformation Language

**中文译名**：动画变换语言

**出现位置**：C16_101-概述与原理.md 第1.1章

**定义**：
ATL 是 Ren'Py 强大的动画语言，用于创建复杂的动画效果，如移动、缩放、淡入淡出等。

### 1.5 引擎架构

**英文名称**：Engine Architecture

**中文译名**：引擎架构

**出现位置**：C16_101-概述与原理.md 第2.1章

**定义**：
Ren'Py 的核心架构包括脚本系统、渲染系统、音频系统、输入系统、存档系统和 Python 集成。

### 1.6 Python 集成

**英文名称**：Python Integration

**中文译名**：Python集成

**出现位置**：C16_101-概述与原理.md 第2.5章

**定义**：
Ren'Py 与 Python 的集成允许开发者在脚本中直接使用 Python 代码处理复杂逻辑。
