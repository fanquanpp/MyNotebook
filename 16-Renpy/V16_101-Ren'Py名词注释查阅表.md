## 1. 基础概念类

### 1.1 Ren'Py

**名称**：Ren'Py 视觉小说引擎（Ren'Py）

**首次出现位置**：C16_101-概述与原理.md 第1章

**定义**：
Ren'Py 是一个专门用于开发视觉小说的免费开源引擎，基于 Python 开发，由 Tom Rothamel 于 2004 年创建。

### 1.2 视觉小说

**名称**：视觉小说（Visual Novel）

**首次出现位置**：C16_101-概述与原理.md 第1.1章

**定义**：
视觉小说是一种以文字叙述为主、结合图像、声音和选择的互动叙事游戏类型。

### 1.3 Ren'Py Script

**名称**：Ren'Py脚本语言（Ren'Py Script Language）

**首次出现位置**：C16_101-概述与原理.md 第1章

**定义**：
Ren'Py Script 是 Ren'Py 的专用脚本语言，语法简洁，基于 Python，用于编写游戏剧情和对话。

### 1.4 ATL

**名称**：动画变换语言（Animation and Transformation Language）

**首次出现位置**：C16_101-概述与原理.md 第1.1章

**定义**：
ATL 是 Ren'Py 强大的动画语言，用于创建复杂的动画效果，如移动、缩放、淡入淡出等。

### 1.5 引擎架构

**名称**：引擎架构（Engine Architecture）

**首次出现位置**：C16_101-概述与原理.md 第2.1章

**定义**：
Ren'Py 的核心架构包括脚本系统、渲染系统、音频系统、输入系统、存档系统和 Python 集成。

### 1.6 Python 集成

**名称**：Python集成（Python Integration）

**首次出现位置**：C16_101-概述与原理.md 第2.5章

**定义**：
Ren'Py 与 Python 的集成允许开发者在脚本中直接使用 Python 代码处理复杂逻辑。

## 2. 核心元素类

### 2.1 标签

**名称**：标签（Label）

**首次出现位置**：C16_102-基础脚本语法.md 第1.1章

**定义**：
标签是 Ren'Py 脚本中标记位置的名，用于实现跳转和控制流程，类似编程语言中的标签或函数名。

### 2.2 对话

**名称**：对话（Dialogue）

**首次出现位置**：C16_102-基础脚本语法.md 第1.2章

**定义**：
对话是游戏中显示角色台词的语句，使用双引号包裹，是 Ren'Py 中最基本的叙事元素。

### 2.3 旁白

**名称**：旁白（Narration）

**首次出现位置**：C16_102-基础脚本语法.md 第1.2章

**定义**：
旁白是不属于任何角色的叙述性文字，用于描述场景或角色内心独白。

### 2.4 角色

**名称**：角色（Character）

**首次出现位置**：C16_102-基础脚本语法.md 第2章

**定义**：
角色是在游戏中定义对话者的对象，可以设置名称、颜色、头像等属性。

### 2.5 场景

**名称**：场景（Scene）

**首次出现位置**：C16_102-基础脚本语法.md 第1.3章

**定义**：
scene 语句用于切换背景图片，定义当前游戏画面显示的背景。

### 2.6 显示

**名称**：显示（Show）

**首次出现位置**：C16_102-基础脚本语法.md 第1.3章

**定义**：
show 语句用于在当前画面上显示图像，如角色立绘、道具等。

### 2.7 隐藏

**名称**：隐藏（Hide）

**首次出现位置**：C16_102-基础脚本语法.md 第1.3章

**定义**：
hide 语句用于从当前画面上移除之前显示的图像。

### 2.8 菜单

**名称**：菜单（Menu）

**首次出现位置**：C16_102-基础脚本语法.md 第1.4章

**定义**：
menu 语句用于显示选择菜单，让玩家做出选择并影响游戏进程。

### 2.9 跳转

**名称**：跳转（Jump）

**首次出现位置**：C16_102-基础脚本语法.md 第1.5章

**定义**：
jump 语句用于无条件跳转到指定标签，改变脚本执行的流向。

### 2.10 调用

**名称**：调用（Call）

**首次出现位置**：C16_102-基础脚本语法.md 第1.5章

**定义**：
call 语句用于调用标签，执行完成后会返回调用点继续执行。

## 3. 屏幕系统类

### 3.1 屏幕

**名称**：屏幕（Screen）

**首次出现位置**：C16_103-屏幕系统与UI设计.md 第1章

**定义**：
屏幕是 Ren'Py 中定义用户界面的机制，使用 Screen Language 编写，定义了 UI 元素的布局和行为。

### 3.2 Screen Language

**名称**：屏幕语言（Screen Language）

**首次出现位置**：C16_103-屏幕系统与UI设计.md 第1章

**定义**：
Screen Language 是 Ren'Py 提供的声明式 UI 语法，用于创建游戏界面。

### 3.3 可显示对象

**名称**：可显示对象（Displayable）

**首次出现位置**：C16_103-屏幕系统与UI设计.md 第2章

**定义**：
Displayable 是屏幕上可显示的元素，如图像、文本、按钮等。

### 3.4 文本按钮

**名称**：文本按钮（Text Button）

**首次出现位置**：C16_103-屏幕系统与UI设计.md 第3章

**定义**：
textbutton 是屏幕语言中的文本按钮组件，用于响应用户点击操作。

### 3.5 动作

**名称**：动作（Action）

**首次出现位置**：C16_103-屏幕系统与UI设计.md 第3章

**定义**：
Action 是与界面元素交互时执行的操作，如 Start()、ShowMenu()、Quit() 等。

### 3.6 转换器

**名称**：转换器（Transformer）

**首次出现位置**：C16_103-屏幕系统与UI设计.md

**定义**：
Transformer 是用于修改显示对象属性的组件，如添加动画效果、改变大小等。

### 3.7 层级

**名称**：层级（Layer）

**首次出现位置**：C16_101-概述与原理.md 第2.2章

**定义**：
Ren'Py 使用层级来组织显示元素，包括 Master 层（背景）、Transient 层（精灵）、Screens 层（UI）、Overlay 层（覆盖）。

## 4. 图像系统类

### 4.1 分层图像

**名称**：分层图像（Layered Image）

**首次出现位置**：C16_205-分层图像.md 第1章

**定义**：
分层图像是 Ren'Py 的图像系统，允许将角色图像分解为多层（如身体、表情、服装），可独立切换。

### 4.2 图像属性

**名称**：图像属性（Image Attribute）

**首次出现位置**：C16_205-分层图像.md 第2章

**定义**：
图像属性是分层图像中可独立切换的部分，如 happy、sad、surprised 等表情状态。

### 4.3 图像定义

**名称**：图像定义（Image Definition）

**首次出现位置**：C16_205-分层图像.md 第3章

**定义**：
image 语句用于定义图像，为图像指定一个标识名称以便在脚本中引用。

### 4.4 自动加载

**名称**：自动加载（Auto-loading）

**首次出现位置**：C16_205-分层图像.md

**定义**：
自动加载是指 Ren'Py 根据文件名约定自动识别和加载图像的功能。

## 5. 动画与转场类

### 5.1 转场效果

**名称**：转场效果（Transitions）

**首次出现位置**：C16_204-转场效果.md 第1章

**定义**：
转场效果是场景或图像切换时的过渡动画，如淡入淡出、溶解、滑动等。

### 5.2 with 语句

**名称**：with语句（With Statement）

**首次出现位置**：C16_204-转场效果.md 第1章

**定义**：
with 语句用于在 Ren'Py 中触发转场效果，如 with fade、with dissolve 等。

### 5.3 ATL 动画

**名称**：ATL动画（ATL Animation）

**首次出现位置**：C16_204-转场效果.md 第2章

**定义**：
ATL 动画使用 ATL 语言定义，显示图像的动画效果，包括位置变化、大小缩放、旋转等。

### 5.4 变换

**名称**：变换（Transform）

**首次出现位置**：C16_204-转场效果.md 第2章

**定义**：
Transform 用于控制显示对象的位置、大小、旋转、透明度等属性。

### 5.5 pause

**名称**：暂停（Pause）

**首次出现位置**：C16_204-转场效果.md 第2章

**定义**：
pause 是 ATL 中的时间控制命令，用于在动画中创建暂停效果。

### 5.6 repeat

**名称**：重复（Repeat）

**首次出现位置**：C16_204-转场效果.md 第2章

**定义**：
repeat 是 ATL 中的循环命令，使动画无限循环播放。

## 6. 数据管理类

### 6.1 存档

**名称**：存档（Save）

**首次出现位置**：C16_101-概述与原理.md 第2.1章

**定义**：
存档是保存游戏当前状态的功能，Ren'Py 内置完整的存档系统，自动处理存档数据的序列化和存储。

### 6.2 读档

**名称**：读档（Load）

**首次出现位置**：C16_101-概述与原理.md 第2.1章

**定义**：
读档是加载之前保存的游戏状态，继续游戏的功能。

### 6.3 回滚

**名称**：回滚（Rollback）

**首次出现位置**：C16_101-概述与原理.md 第2.3章

**定义**：
回滚是 Ren'Py 的独特功能，允许玩家回溯游戏进度，重新做出选择。

### 6.4 persistent 数据

**名称**：持久化数据（Persistent Data）

**首次出现位置**：C16_101-概述与原理.md

**定义**：
Persistent 数据是在游戏通关后仍然保留的数据，如解锁的 CG、音色等。

### 6.5 游戏状态

**名称**：游戏状态（Game State）

**首次出现位置**：C16_101-概述与原理.md 第2.3章

**定义**：
游戏状态包括当前标签、变量值、显示的图像、背景音乐等所有影响游戏进程的数据。

## 7. NVL 模式类

### 7.1 NVL 模式

**名称**：全屏对话模式（NVL Mode）

**首次出现位置**：C16_102-基础脚本语法.md

**定义**：
NVL 模式是 Ren'Py 提供的全屏对话显示模式，与默认的对话窗口模式不同。

### 7.2 对话窗口

**名称**：对话窗口（Say Screen）

**首次出现位置**：C16_102-基础脚本语法.md

**定义**：
对话窗口是默认的对话显示模式，对话显示在屏幕下方的文本框中。

### 7.3 气泡模式

**名称**：气泡模式（Bubble Mode）

**首次出现位置**：C16_102-基础脚本语法.md

**定义**：
Bubble 模式以类似即时通讯软件的气泡形式显示对话，增强视觉表现力。

**名称**：背景音乐（Background Music (BGM)）

音效是游戏中的短音频效果，如点击声、爆炸声等，使用 play sound 语句控制。

### 8.3 语音

**名称**：语音（Voice）

语音是角色对话的配音，使用 voice 语句播放与对话同步的音频文件。

### 8.4 channel

**名称**：音频通道（Audio Channel）

**首次出现位置**：C16_102-基础脚本语法.md 第3章

Ren'Py 使用多个音频通道同时播放不同的音频，如 music、sound、voice 各自独立通道。
