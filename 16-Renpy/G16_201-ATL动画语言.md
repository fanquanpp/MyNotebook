## 1. ATL 概述 (Overview)

ATL (Animation and Transformation Language) 是 Ren'Py 中用于定义图像变换和动画的专用语言，它允许开发者创建各种复杂的动画效果，包括位置移动、缩放、旋转、透明度变化等。

ATL 是 **Transforms** 文档的一部分，与以下主题相关：
- **Transforms** (变换) - 包含 ATL 动画语言
- **Transform Properties** (变换属性) - 位置、缩放、旋转等属性
- **Transitions** (转场) - 溶解、淡入淡出等效果

### 1.1 ATL 代码位置

ATL 代码可以在以下三个位置使用：

1. **`transform`** 语句：创建可重用的变换
2. **`image`** 语句的 ATL 块：定义动画图像
3. **`scene`** 和 **`show`** 语句的 ATL 块：直接应用动画效果

## 2. ATL 语法基础 (ATL Syntax Basics)

### 2.1 基本 transform 语句

```renpy
transform my_move:
    xalign 0.0 yalign 1.0
    linear 2.0 xalign 1.0
```

### 2.2 变换属性 (Transform Properties)

- **位置**: `xpos`, `ypos`, `xalign`, `yalign`, `xoffset`, `yoffset`
- **缩放**: `zoom`, `xzoom`, `yzoom`
- **旋转**: `rotate`, `rotatey`, `rotatez`
- **透明度**: `alpha`
- **颜色**: `color`
- **混合模式**: `blend`
- **滤镜**: `blur`, `dropshadow`, `outline`

## 3. ATL 语句 (ATL Statements)

### 3.1 时间控制语句

| 语句 | 描述 |
| :--- | :--- |
| **`linear`** | 线性插值，均匀变化 |
| **`ease`** | 平滑过渡，先快后慢 |
| **`easein`** | 缓入，从慢到快 |
| **`easeout`** | 缓出，从快到慢 |
| **`easeinout`** | 缓入缓出，先慢后快再慢 |
| **`pause`** | 暂停指定时间 |
| **`wait`** | 等待指定时间后继续 |
| **`time`** | 设置动画总时间 |

### 3.2 控制流语句

| 语句 | 描述 |
| :--- | :--- |
| **`repeat`** | 无限循环 |
| **`repeat n`** | 循环n次 |
| **`parallel`** | 并行执行多个动画 |
| **`block`** | 定义动画块 |
| **`if/elif/else`** | 条件判断 |
| **`while`** | 循环执行 |
| **`choice`** | 随机选择动画路径 |

### 3.3 Properties 语句

设置属性的默认值：

```renpy
transform my_transform:
    properties:
        xalign 0.5
        yalign 0.5
```

### 3.4 Number 语句

数值动画：

```renpy
transform bounce:
    ypos 0
    easeout 0.5 ypos 100
    easein 0.5 ypos 0
    repeat
```

### 3.5 Interpolation 语句

带缓动函数的动画：

```renpy
transform slide:
    xalign 0.0
    ease 1.0 xalign 1.0
```

### 3.6 Pass 语句

空操作，用于占位：

```renpy
transform wait_and_fade:
    pause 1.0
    pass
    linear 1.0 alpha 0.0
```

### 3.7 Repeat 语句

循环动画：

```renpy
transform spin:
    rotate 0
    linear 2.0 rotate 360
    repeat
```

### 3.8 Block 语句

组合多个动画：

```renpy
transform complex:
    block:
        linear 1.0 xalign 1.0
        linear 1.0 xalign 0.0
        repeat
```

### 3.9 Parallel 语句

并行执行多个动画：

```renpy
transform complex_animation:
    parallel:
        linear 2.0 xalign 1.0
        linear 2.0 yalign 0.0
    parallel:
        linear 2.0 zoom 1.5
        linear 2.0 rotate 360
```

### 3.10 Choice 语句

随机选择动画：

```renpy
transform random_animation:
    choice:
        linear 0.5 xalign 0.0
    choice:
        linear 0.5 xalign 1.0
    choice:
        linear 0.5 xalign 0.5
```

### 3.11 Animation 语句

图像序列动画：

```renpy
image eileen walk:
    animation
    "eileen_walk1.png"
    pause 0.1
    "eileen_walk2.png"
    pause 0.1
    repeat
```

### 3.12 On 语句

事件触发动画：

```renpy
transform reactive:
    on show:
        alpha 0.0
        linear 0.5 alpha 1.0
    on hide:
        linear 0.5 alpha 0.0
```

### 3.13 Displayable 语句

嵌入其他显示对象：

```renpy
transform with_overlay:
    Displayable("snow.png")
    linear 5.0 alpha 0.0
```

### 3.14 Contains Block 语句

条件包含：

```renpy
transform conditional_contains:
    contains:
        "character.png"
    contains:
        "effect.png" if effects_enabled
```

### 3.15 Function 语句

调用 Python 函数：

```renpy
transform with_function:
    function my_transform_function
```

## 4. 常用动画示例 (Common Animation Examples)

### 4.1 呼吸效果 (Breathing)

```renpy
transform breathe:
    zoom 1.0
    linear 2.0 zoom 1.05
    linear 2.0 zoom 1.0
    repeat
```

### 4.2 抖动效果 (Shake)

```renpy
transform shake:
    xoffset 0
    linear 0.1 xoffset 10
    linear 0.1 xoffset -10
    linear 0.1 xoffset 0
    repeat
```

### 4.3 淡入淡出效果 (Fade In/Out)

```renpy
transform fade_in:
    alpha 0.0
    linear 1.0 alpha 1.0

transform fade_out:
    alpha 1.0
    linear 1.0 alpha 0.0
```

### 4.4 旋转效果 (Rotation)

```renpy
transform spin:
    rotate 0
    linear 2.0 rotate 360
    repeat
```

### 4.5 路径动画 (Path Animation)

```renpy
transform path_move:
    subpixel True
    xpos 100 ypos 100
    bezier (200, 50, 300, 150, 400, 100)
    time 3.0
```

### 4.6 复合动画 (Combined Animation)

```renpy
transform complex_animation:
    parallel:
        linear 2.0 xalign 1.0
        linear 2.0 yalign 0.0
    parallel:
        linear 2.0 zoom 1.5
        linear 2.0 rotate 360
    pause 1.0
    linear 1.0 alpha 0.0
```

### 4.7 弹跳效果 (Bounce)

```renpy
transform bounce:
    ypos 0
    easeout 0.5 ypos 100
    easein 0.5 ypos 0
    repeat
```

## 5. ATL 高级特性 (Advanced Features)

### 5.1 带参数的 Transform

```renpy
transform variable_animation(speed=1.0):
    xalign 0.0
    linear speed xalign 1.0
    linear speed xalign 0.0
    repeat
```

### 5.2 状态机 (State Machine)

```renpy
transform state_machine:
    idle:
        zoom 1.0
        pause 1.0
        choice:
            0.5: linear 0.2 zoom 1.05
            0.5: pass
        repeat idle
    hover:
        zoom 1.1
        linear 0.3 zoom 1.15
        linear 0.3 zoom 1.1
        repeat hover
```

### 5.3 图像序列动画 (Image Sequence)

```renpy
define eileen_walk = Animation(
    "eileen_walk1.png", 0.1,
    "eileen_walk2.png", 0.1,
    "eileen_walk3.png", 0.1,
    "eileen_walk2.png", 0.1,
    loop=True
)
```

### 5.4 外部事件 (External Events)

ATL 可以响应外部事件：

```renpy
transform my_transform:
    on show:
        alpha 0.0
        linear 0.5 alpha 1.0
    on hide:
        linear 0.5 alpha 0.0
    on replace:
        linear 0.3 zoom 1.2
        linear 0.3 zoom 1.0
```

### 5.5 ATL 柯里化和偏参数传递 (ATL Curry and Partial Parameter Passing)

```renpy
transform base_transform(duration=1.0):
    linear duration alpha 0.5

transform derived_transform:
    contains base_transform(duration=2.0)
```

## 6. 内置变换 (Built-in Transforms)

| 变换 | 描述 |
| :--- | :--- |
| **center** | 水平居中，底部对齐 |
| **default** | 默认位置（可配置） |
| **left** | 左下角对齐 |
| **right** | 右下角对齐 |
| **top** | 顶部居中 |
| **topleft** | 左上角对齐 |
| **topright** | 右上角对齐 |
| **truecenter** | 完全居中 |
| **offscreenleft** | 屏幕左侧外 |
| **offscreenright** | 屏幕右侧外 |
| **reset** | 重置所有属性 |

## 7. 官方推荐的最佳实践 (Official Best Practices)

### 7.1 性能优化

- **使用 `subpixel True`** 获得更平滑的动画
- **避免在动画中使用复杂计算**，特别是在循环动画中
- **对于重复动画，使用 `repeat` 而不是循环结构**
- **合理使用 `parallel`** 减少动画嵌套，提高性能
- **使用 `cache` 属性** 缓存复杂变换，减少重复计算
- **优化图像大小**，避免使用过大的图像进行动画

### 7.2 代码组织

- **将复杂动画拆分为多个简单变换**，提高可维护性
- **为常用动画创建可重用的变换定义**
- **使用变量控制动画参数**，便于调整和维护
- **结合 `LiveComposite`** 创建更复杂的动画效果
- **使用有意义的命名**，使动画代码更易理解

### 7.3 调试技巧

- **使用 `show expression`** 实时预览动画效果
- **调整动画速度进行调试**：`config.animation_sound_delay = 0.1`
- **使用 `renpy.display.screen.Screen`** 测试复杂动画
- **添加临时 `pause`** 语句观察动画中间状态
- **使用 `renpy.trace()`** 跟踪动画执行过程

### 7.4 高级技巧

- **使用 `transform` 继承** 创建动画变体
- **结合 `ConditionSwitch`** 创建条件动画
- **使用 `Function`** 调用 Python 函数控制动画
- **实现动画状态管理**，根据游戏状态切换动画
- **创建动画库**，复用常用动画效果

## 10. 总结

### 10.1 关键要点

- **ATL 是 Transforms 的一部分**: ATL 动画语言是用于创建变换的高级语言
- **时间控制**: 使用 linear、ease、pause 等语句控制动画时间
- **控制流**: 利用循环、条件、并行等控制流创建复杂动画
- **事件响应**: 使用 on 语句响应外部事件（show、hide、replace）
- **性能优化**: 遵循官方推荐的性能最佳实践
- **代码组织**: 保持动画代码的清晰和可维护性

- **从简单开始**: 先学习基本的变换和位置属性

- **参考示例**: 学习官方示例和社区项目中的动画实现

通过不断学习和实践，开发者可以掌握 ATL 动画语言，为视觉小说增添生动的动画效果，提升游戏的视觉体验。
