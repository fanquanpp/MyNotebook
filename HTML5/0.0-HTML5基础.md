## 1. 基础标签

### 1.1 注释
**示例：** `<!--注释内容-->`
**注释快捷键：** Ctrl+/

### 1.2 HTML基本骨架

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>标题</title>
    </head>
    <body>
        <!--网页内容-->
    </body>
</html>
```

**HTML基本骨架快捷键：** !+回车

### 1.3 标题标签
**标签名称：** h1-h6

**显示特点：**
- 文字加粗，大小不同
- 字号逐渐减小
- 独占一行，自动换行

**示例：**
```html
<h1>标题</h1>
```

### 1.4 段落标签
**标签名称：** p

**显示特点：**
- 独占一行，自动换行
- 段落之间有空行

**示例：**
```html
<p>段落</p>
```

### 1.5 换行标签
**标签名称：** br

**显示特点：**
- 不占行，不换行
- 自动换行

**示例：**
```html
<br />
```

### 1.6 水平线标签
**标签名称：** hr

**显示特点：**
- 占一行，自动换行
- 水平线

**示例：**
```html
<hr />
```

### 1.7 文本标签
**标签名称：** strong（b）、em（i）、del（s）、ins（u）、mark、small、sub、sup

**显示特点：**
- strong：加粗
- em：斜体
- del：删除线
- ins：下划线
- mark：标记
- small：小号
- sub：下标
- sup：上标

**示例：**
```html
<strong>加粗</strong>
<em>斜体</em>
<del>删除线</del>
<ins>下划线</ins>
<mark>标记</mark>
<small>小号</small>
<sub>下标</sub>
<sup>上标</sup>
```

### 1.8 超链接标签
**标签名称：** a

**显示特点：**
- 超链接
- 鼠标悬停时显示提示信息
- 点击时跳转到指定页面

**示例：**
```html
（默认）当前窗口打开
    <a href="http://www.baidu.com">百度</a>
（默认）当前窗口打开，并显示提示信息
    <a href="http://www.baidu.com" title="百度">百度</a>
（新窗口打开）
    <a href="http://www.baidu.com" target="_blank">百度</a>
（当前窗口打开）
    <a href="http://www.baidu.com" target="_self">百度</a>
（父窗口打开）仅在嵌套页面中有效
    <a href="http://www.baidu.com" target="_parent">百度</a>
（最顶层窗口打开）
    <a href="http://www.baidu.com" target="_top">百度</a>
```

### 1.9 图片标签
**标签名称：** img

**显示特点：**
- 图片
- 鼠标悬停时显示提示信息
- 点击时跳转到指定页面

**示例：**
```html
（默认）
    <img src="图片地址" alt="图片描述" />
（鼠标悬停时显示提示信息）
    <img src="图片地址" alt="图片描述" title="提示信息" />
```

### 1.10 音频标签
**标签名称：** audio
**属性名称：** src、autoplay、controls、loop、muted、preload

**属性特点：**
- src：音频地址
- autoplay：自动播放
- controls：显示控制按钮
- loop：循环播放
- muted：静音
- preload：预加载   

**显示特点：**
- 音频
- 自动播放：autoplay="true"
- 显示控制按钮：controls="true"
- 循环播放：loop="true"
- 静音：muted="true"
- 预加载：preload="auto"

**示例：**
```html
<audio src="音频地址" autoplay="true" controls="true" loop="true" muted="true" preload="auto"></audio>
```

**注意：** 在HTML5中，如果属性名称与属性值相同，可以省略属性值的双引号。如下：
```html
<audio src="音频地址" autoplay controls loop muted preload></audio>
```

**音频标签快捷键：** au+回车

### 1.11 视频标签
**标签名称：** video
**属性名称：** src、autoplay、controls、loop、muted、preload

**属性特点：**
- src：视频地址
- autoplay：自动播放
- controls：显示控制按钮
- loop：循环播放
- muted：静音
- preload：预加载   

**显示特点：**
- 视频
- 自动播放：autoplay="true"
- 显示控制按钮：controls="true"
- 循环播放：loop="true"

**示例：**
```html
<video src="视频地址" autoplay="true" controls="true" loop="true" muted="true" preload="auto"></video>
```

**注意：** 在HTML5中，如果属性名称与属性值相同，可以省略属性值的双引号。如下：
```html
<video src="视频地址" autoplay controls loop muted preload></video>
```

## 2. 路径

### 2.1 路径符号含义：
- `/` 表示根目录，表示进入某个文件夹
- `.` 表示当前目录
- `..` 表示上一级目录
- `*` 表示任意字符
- `?` 表示任意字符（除了 /）
- `[abc]` 表示任意字符，其中 a、b、c 表示任意字符
- `[^abc]` 表示任意字符，其中 a、b、c 表示任意字符，但排除 a、b、c

### 2.2 分类
1. **绝对路径**：绝对路径是指从根目录（盘符）开始，直到文件或目录的完整路径。
2. **相对路径**：相对路径是指从当前目录开始，直到文件或目录的完整路径。

### 2.3 绝对路径和相对路径的转换

**绝对路径转换为相对路径：**
1. 获取绝对路径的盘符
2. 获取绝对路径的目录
3. 获取绝对路径的文件名
4. 拼接相对路径

**相对路径转换为绝对路径：**
1. 获取相对路径的目录
2. 获取相对路径的文件名

### 2.4 绝对路径的写法
- D:/a/b/c.txt

### 2.5 相对路径的写法
- a/b/c.txt

## 3. 列表标签

### 3.1 列表分类
- 有序列表
- 无序列表
- 定义列表（HTML5中更名为"描述列表"）

### 3.2 列表标签名称
- `<ol>`  有序列表标签
- `<ul>`  无序列表标签
- `<dl>`  定义列表标签
- `<dd>`  描述列表标签

### 3.3 无序列表使用方法：嵌套使用
- ul嵌套li
- ul为父标签，li为子标签

**示例：**
```html
<ul>
    <li>1.1</li>
    <li>1.2</li>
    <li>1.3</li>
</ul>
```

### 3.4 有序列表使用方法：嵌套使用
- ol嵌套li
- ol为父标签，li为子标签

**示例：**
```html
<ol>
    <li>1.1</li>
    <li>1.2</li>
    <li>1.3</li>
</ol>
```

### 3.5 定义（描述）列表使用方法：嵌套使用
- dl嵌套dt和dd
- dl为父标签，dt和dd为子标签
- dl为定义列表标签，dt为定义标签（相当于放标题），dd为描述标签（相当于放内容）

**示例：**
```html
<dl>
    <dt>1.1</dt>
    <dd>1.1.1</dd>
    <dd>1.1.2</dd>
    <dd>1.1.3</dd>
    <dt>1.2</dt>
    <dd>1.2.1</dd>
    <dd>1.2.2</dd>
    <dd>1.2.3</dd>
</dl>
```

## 4. 表格标签

### 4.1 标签名：
table（表格） tr（行） td（表头单元格） th（内容单元格）

- **table标签**：table标签用于定义一个表格，table标签包含多个tr标签，tr标签包含多个td标签，td标签包含多个th标签。
- **tr标签**：tr标签用于定义表格中的行，tr标签包含多个td标签。
- **td标签**：td标签用于定义表格中的列，td标签包含多个内容。
- **th标签**：th标签用于定义表格中的标题，th标签包含多个内容。

### 4.2 使用方法：创建一个表格，并添加内容。（table嵌套tr标签和td标签，tr嵌套td标签，td标签包含多个内容）

**示例：**
```html
<table border="1">
    <tr>
        <th>姓名</th>
        <th>性别</th>
        <th>年龄</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>男</td>
        <td>18</td>
    </tr>
    <tr>
        <td>李四</td>
        <td>女</td>
        <td>19</td>
    </tr>
</table>
```

### 4.3 关于表格标签的属性
- **border属性**：border属性用于设置表格的边框宽度，单位为像素。
- **cellpadding属性**：cellpadding属性用于设置单元格之间的间距，单位为像素。
- **cellspacing属性**：cellspacing属性用于设置单元格之间的间距，单位为像素。
- **width属性**：width属性用于设置表格的宽度，单位为像素。
- **height属性**：height属性用于设置表格的高度，单位为像素。
- **bgcolor属性**：bgcolor属性用于设置表格的背景颜色

## 5. 表单标签

表单是用户与网页交互的重要方式，用于收集用户输入的信息。

### 5.1 表单基本结构

```html
<form action="提交地址" method="提交方式">
    <!-- 表单元素 -->
</form>
```

### 5.2 常用表单元素

#### 5.2.1 文本输入框

```html
<input type="text" name="username" placeholder="请输入用户名">
```

#### 5.2.2 密码输入框

```html
<input type="password" name="password" placeholder="请输入密码">
```

#### 5.2.3 单选按钮

```html
<input type="radio" name="gender" value="male"> 男
<input type="radio" name="gender" value="female"> 女
```

#### 5.2.4 复选框

```html
<input type="checkbox" name="hobby" value="reading"> 阅读
<input type="checkbox" name="hobby" value="music"> 音乐
```

#### 5.2.5 下拉菜单

```html
<select name="city">
    <option value="beijing">北京</option>
    <option value="shanghai">上海</option>
    <option value="guangzhou">广州</option>
</select>
```

#### 5.2.6 文本域

```html
<textarea name="comment" rows="4" cols="50">请输入评论</textarea>
```

#### 5.2.7 提交按钮

```html
<input type="submit" value="提交">
<button type="submit">提交</button>
```

## 6. 语义化标签

HTML5引入了一系列语义化标签，使网页结构更加清晰，有利于搜索引擎优化和无障碍访问。

### 6.1 常用语义化标签

| 标签 | 描述 |
| :--- | :--- |
| `<header>` | 页面或区块的头部 |
| `<nav>` | 导航链接区域 |
| `<main>` | 页面的主要内容 |
| `<section>` | 文档中的区块 |
| `<article>` | 独立的内容区块 |
| `<aside>` | 侧边栏或附加内容 |
| `<footer>` | 页面或区块的底部 |
| `<figure>` | 图表、图片等媒体内容 |
| `<figcaption>` | 媒体内容的标题 |

### 6.2 语义化布局示例

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>语义化布局示例</title>
</head>
<body>
    <header>
        <h1>网站标题</h1>
        <nav>
            <ul>
                <li><a href="#">首页</a></li>
                <li><a href="#">关于我们</a></li>
                <li><a href="#">联系我们</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <section>
            <h2>新闻资讯</h2>
            <article>
                <h3>新闻标题1</h3>
                <p>新闻内容1...</p>
            </article>
            <article>
                <h3>新闻标题2</h3>
                <p>新闻内容2...</p>
            </article>
        </section>
        <aside>
            <h3>侧边栏</h3>
            <p>侧边栏内容...</p>
        </aside>
    </main>
    <footer>
        <p>版权信息</p>
    </footer>
</body>
</html>
```

## 7. 常见问题与解决方案

### 7.1 标签嵌套问题

**问题描述**：标签嵌套不正确，导致页面显示异常。

**原因分析**：HTML标签必须正确嵌套，不能交叉嵌套。

**解决方案**：
- 确保标签正确嵌套，如 `<p><strong>文本</strong></p>`
- 使用代码编辑器的自动缩进功能检查嵌套结构
- 使用在线HTML验证工具检查代码

### 7.2 路径引用问题

**问题描述**：图片、CSS等资源无法加载。

**原因分析**：路径引用错误，导致浏览器无法找到资源文件。

**解决方案**：
- 检查相对路径是否正确
- 使用绝对路径时确保路径完整
- 确认资源文件确实存在于指定位置

### 7.3 表单提交问题

**问题描述**：表单提交后没有反应或提交到错误地址。

**原因分析**：表单的action属性或method属性设置错误。

**解决方案**：
- 确保action属性指向正确的服务器地址
- 选择合适的method属性（GET或POST）
- 检查表单元素的name属性是否正确设置

### 7.4 语义化标签兼容性问题

**问题描述**：语义化标签在旧浏览器中显示异常。

**原因分析**：旧版本浏览器不支持HTML5语义化标签。

**解决方案**：
- 在CSS中为语义化标签添加 `display: block;`
- 使用HTML5 Shiv库兼容旧浏览器
- 优先使用语义化标签，同时保留兼容性考虑

## 8. 总结与最佳实践

### 8.1 核心概念

- **HTML**：超文本标记语言，用于构建网页结构
- **标签**：HTML的基本构建单元，如 `<div>`、`<p>` 等
- **属性**：标签的附加信息，如 `id`、`class`、`src` 等
- **语义化**：使用有意义的标签来描述内容

### 8.2 最佳实践

1. **代码规范**
   - 使用小写标签名
   - 正确嵌套标签
   - 为标签添加必要的属性
   - 使用缩进和空格提高代码可读性

2. **语义化设计**
   - 优先使用语义化标签
   - 合理划分页面结构
   - 为图片添加alt属性
   - 为链接添加title属性

3. **性能优化**
   - 减少HTML文件大小
   - 合理使用标签，避免过度嵌套
   - 优化图片资源
   - 使用CDN加载外部资源

4. **可访问性**
   - 为表单元素添加label标签
   - 使用语义化标签提高屏幕阅读器兼容性
   - 确保链接文本有意义
   - 提供适当的颜色对比度

5. **开发工具**
   - 使用现代代码编辑器（如VSCode）
   - 利用浏览器开发者工具调试
   - 使用HTML验证工具检查代码
   - 版本控制（如Git）管理代码
