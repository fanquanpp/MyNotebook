## 1. 基础标签

### 1.1 注释
**示例：** `<!--注释内容-->`
**注释快捷键：** Ctrl+/

### 1.2 HTML基本骨架

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>标题</title>
    </head>
    <body>
        <!--网页内容-->
    </body>
</html>
```

**HTML基本骨架快捷键：** !+回车

### 1.3 标题标签
**标签名称：** h1-h6

**显示特点：**
- 文字加粗，大小不同
- 字号逐渐减小
- 独占一行，自动换行

**示例：**
```html
<h1>标题</h1>
```

### 1.4 段落标签
**标签名称：** p

**显示特点：**
- 独占一行，自动换行
- 段落之间有空行

**示例：**
```html
<p>段落</p>
```

### 1.5 换行标签
**标签名称：** br

**显示特点：**
- 不占行，不换行
- 自动换行

**示例：**
```html
<br />
```

### 1.6 水平线标签
**标签名称：** hr

**显示特点：**
- 占一行，自动换行
- 水平线

**示例：**
```html
<hr />
```

### 1.7 文本标签
**标签名称：** strong（b）、em（i）、del（s）、ins（u）、mark、small、sub、sup

**显示特点：**
- strong：加粗
- em：斜体
- del：删除线
- ins：下划线
- mark：标记
- small：小号
- sub：下标
- sup：上标

**示例：**
```html
<strong>加粗</strong>
<em>斜体</em>
<del>删除线</del>
<ins>下划线</ins>
<mark>标记</mark>
<small>小号</small>
<sub>下标</sub>
<sup>上标</sup>
```

### 1.8 超链接标签
**标签名称：** a

**显示特点：**
- 超链接
- 鼠标悬停时显示提示信息
- 点击时跳转到指定页面

**示例：**
```html
（默认）当前窗口打开
    <a href="http://www.baidu.com">百度</a>
（默认）当前窗口打开，并显示提示信息
    <a href="http://www.baidu.com" title="百度">百度</a>
（新窗口打开）
    <a href="http://www.baidu.com" target="_blank">百度</a>
（当前窗口打开）
    <a href="http://www.baidu.com" target="_self">百度</a>
（父窗口打开）仅在嵌套页面中有效
    <a href="http://www.baidu.com" target="_parent">百度</a>
（最顶层窗口打开）
    <a href="http://www.baidu.com" target="_top">百度</a>
```

### 1.9 图片标签
**标签名称：** img

**显示特点：**
- 图片
- 鼠标悬停时显示提示信息
- 点击时跳转到指定页面

**示例：**
```html
（默认）
    <img src="图片地址" alt="图片描述" />
（鼠标悬停时显示提示信息）
    <img src="图片地址" alt="图片描述" title="提示信息" />
```

### 1.10 音频标签
**标签名称：** audio
**属性名称：** src、autoplay、controls、loop、muted、preload

**属性特点：**
- src：音频地址
- autoplay：自动播放
- controls：显示控制按钮
- loop：循环播放
- muted：静音
- preload：预加载   

**显示特点：**
- 音频
- 自动播放：autoplay="true"
- 显示控制按钮：controls="true"
- 循环播放：loop="true"
- 静音：muted="true"
- 预加载：preload="auto"

**示例：**
```html
<audio src="音频地址" autoplay="true" controls="true" loop="true" muted="true" preload="auto"></audio>
```

**注意：** 在HTML5中，如果属性名称与属性值相同，可以省略属性值的双引号。如下：
```html
<audio src="音频地址" autoplay controls loop muted preload></audio>
```

**音频标签快捷键：** au+回车

### 1.11 视频标签
**标签名称：** video
**属性名称：** src、autoplay、controls、loop、muted、preload

**属性特点：**
- src：视频地址

- loop：循环播放

**显示特点：**


## 2. 路径

### 2.1 路径符号含义：
- `/` 表示根目录，表示进入某个文件夹
- `.` 表示当前目录
- `..` 表示上一级目录
- `*` 表示任意字符
- `?` 表示任意字符（除了 /）
- `[abc]` 表示任意字符，其中 a、b、c 表示任意字符
- `[^abc]` 表示任意字符，其中 a、b、c 表示任意字符，但排除 a、b、c

### 2.2 分类
1. **绝对路径**：绝对路径是指从根目录（盘符）开始，直到文件或目录的完整路径。
2. **相对路径**：相对路径是指从当前目录开始，直到文件或目录的完整路径。

### 2.3 绝对路径和相对路径的转换

**绝对路径转换为相对路径：**
1. 获取绝对路径的盘符
2. 获取绝对路径的目录
3. 获取绝对路径的文件名
4. 拼接相对路径

**相对路径转换为绝对路径：**
1. 获取相对路径的目录
2. 获取相对路径的文件名

### 2.4 绝对路径的写法
- D:/a/b/c.txt

### 2.5 相对路径的写法
- a/b/c.txt

## 3. 列表标签

### 3.1 列表分类
- 有序列表
- 无序列表
- 定义列表（HTML5中更名为"描述列表"）

### 3.2 列表标签名称
- `<ol>`  有序列表标签
- `<ul>`  无序列表标签
- `<dl>`  定义列表标签
- `<dd>`  描述列表标签

### 3.3 无序列表使用方法：嵌套使用
- ul嵌套li
- ul为父标签，li为子标签

**示例：**
```html
<ul>
    <li>1.1</li>
    <li>1.2</li>
    <li>1.3</li>
</ul>
```

### 3.4 有序列表使用方法：嵌套使用
- ol嵌套li
- ol为父标签，li为子标签

**示例：**
```html
<ol>
    <li>1.1</li>
    <li>1.2</li>
    <li>1.3</li>
</ol>
```

### 3.5 定义（描述）列表使用方法：嵌套使用
- dl嵌套dt和dd
- dl为父标签，dt和dd为子标签
- dl为定义列表标签，dt为定义标签（相当于放标题），dd为描述标签（相当于放内容）

**示例：**
```html
<dl>
    <dt>1.1</dt>
    <dd>1.1.1</dd>
    <dd>1.1.2</dd>
    <dd>1.1.3</dd>
    <dt>1.2</dt>
    <dd>1.2.1</dd>
    <dd>1.2.2</dd>
    <dd>1.2.3</dd>
</dl>
```

## 4. 表格标签

### 4.1 标签名：
table（表格） tr（行） td（表头单元格） th（内容单元格）

- **table标签**：table标签用于定义一个表格，table标签包含多个tr标签，tr标签包含多个td标签，td标签包含多个th标签。
- **tr标签**：tr标签用于定义表格中的行，tr标签包含多个td标签。
- **td标签**：td标签用于定义表格中的列，td标签包含多个内容。
- **th标签**：th标签用于定义表格中的标题，th标签包含多个内容。

### 4.2 使用方法：创建一个表格，并添加内容。（table嵌套tr标签和td标签，tr嵌套td标签，td标签包含多个内容）

**示例：**
```html
<table border="1">
    <tr>
        <th>姓名</th>
        <th>性别</th>
        <th>年龄</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>男</td>
        <td>18</td>
    </tr>
    <tr>
        <td>李四</td>
        <td>女</td>
        <td>19</td>
    </tr>
</table>
```

### 4.3 关于表格标签的属性
- **border属性**：border属性用于设置表格的边框宽度，单位为像素。
- **cellpadding属性**：cellpadding属性用于设置单元格之间的间距，单位为像素。
- **cellspacing属性**：cellspacing属性用于设置单元格之间的间距，单位为像素。
- **width属性**：width属性用于设置表格的宽度，单位为像素。
- **height属性**：height属性用于设置表格的高度，单位为像素。
- **bgcolor属性**：bgcolor属性用于设置表格的背景颜色
