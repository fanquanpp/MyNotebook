## 1. 简介 | Introduction

Vue 是由尤雨溪（Evan You）于 2014 年创建的渐进式 JavaScript 框架。Vue 3 是其 2020 年发布的重大版本，引入了组合式 API、更好的性能、更小的包体积、更好的 TypeScript 支持等重要改进。
