## 1. 计算机的存储规则

### 1.1 文本文字
- **二进制**：由0与1组成，代码中以0b开头
- **十进制**：由0-9组成，无前缀
- **八进制**：由0-7组成，代码中以0开头
- **十六进制**：由0-9与a-f组成，代码中以0x开头

*以上特性为JDK开始存在的特性，JDK8开始支持*

**示例：**

```java
public class Demo {
    public static void main(String[] args) {
        int a = 0b1010; // 二进制
        int b = 010; // 八进制
        int c = 0x10; // 十六进制
        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
    }
}
```

### 1.2 任意进制转换为十进制

**公式：** 系数*基数的权次幂（相加）

```
a*b^n + a*b^(n-1) + ... + a*b^1 + a*b^0
```

- **系数**：每一位上的数字
- **基数**：当前进制数
- **权**：从右往左依次递增（0、1、2、3、4、5…………）

**示例：**
- 101（二进制）-->5(十进制)
  ```
  1               0               1
（1*2^2）        (0*2^1)          （1*2^0）
  |               |               |
  4         +     0        +      1   =   5( 十进制)
  ```
- 101（八进制）-->65(十进制)
- abc（十六进制）-->2748(十进制)

### 1.3 二进制转十进制（8421快速转化法）

**原理：** 每一位二进制值的1都对应一个固定的值，将每一位1代表的固定值相加，即为十进制的值

```
二进制：1    1   1   1    1   1   1   1
对应值：128  64  32  16   8   4   2   1
十进制：（271）
```

**示例：**
- 二进制：1101-->(8、4、0、1)
- 十进制：13  （8+4+0+1=13）

### 1.4 十进制转换成其他禁止（除基取余法）

**原理：** 不断除以基数（几进制基数就是几）得到余数，直到商为0，再将余数倒序拼接即可

1. 除以基数，取余数
2. 将余数保存起来
3. 除以基数，取余数
4. 将余数保存起来
5. 重复3、4步骤，直到商为0
6. 将余数保存的数组倒序输出

**示例：**
- 10（十进制）-->1010（二进制）
  ```
  10/2 = 5   10%2 = 0
  5/2 = 2   5%2 = 1
  2/2 = 1   2%2 = 0
  1/2 = 0   1%2 = 1
  
  0   1   1   0
  ```
- 10（十进制）-->1001（八进制）
- 11（十进制）-->1011 (二进制)
- 7（十进制）--> 111（二进制）
