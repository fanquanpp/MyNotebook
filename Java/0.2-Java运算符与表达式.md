## 2. 算术运算符

| 运算符 | 描述     | 示例                 |
| :----- | :------- | :------------------- |
| `+`    | 加       | `a + b`              |
| `-`    | 减       | `a - b`              |
| `*`    | 乘       | `a * b`              |
| `/`    | 除       | `a / b`              |
| `%`    | 取余     | `a % b`              |
| `++`   | 自增     | `i++` 或 `++i`       |
| `--`   | 自减     | `i--` 或 `--i`       |

**示例：**

```java
int i = 10;
i++;
System.out.println(i); // 11

int j = 10;
j--;
System.out.println(j); // 9
```

## 4. 三元运算符

**格式：** `条件表达式 ? 表达式1 : 表达式2;`

**核心逻辑：** 先执行条件表达式，得到布尔值（boolean）。若结果为真（即条件表达式成立或正确：true）则输出表达式一；若结果为假（即条件表达式不成立或错误：false）则输出表达式二。

**等价于:** 

```java
if(条件表达式) {
    表达式1;
} else {
    表达式2;
}
```

**示例：**

```java
//true为成立    false为不成立
int a=1, b=2;
int max=(a > b) ? a : b;    //false为不成立，输出b
System.out.println(max);    //2
```

## 5. 自增自减运算符

### 5.1 自增运算符

**`++i` 先自增再使用：**

```java
int a, b;
a = 10;
b = ++a;
System.out.println(b); // 11
System.out.println(a); // 11
```

**`i++` 先使用再自增：**

```java
int a, b;
a = 10;
b = a++;
System.out.println(b); // 10
System.out.println(a); // 11
```

## 7. 逻辑运算符

| 运算符 | 描述       | 示例                 |
| :----- | :--------- | :------------------- |
| `&&`   | 逻辑与     | `a && b`             |
| `||`   | 逻辑或     | `a || b`             |
| `!`    | 逻辑非     | `!a`                 |
| `&`    | 按位与     | `a & b`              |

| `>>`   | 右移       | `a >> b`             |
| `>>>`  | 无符号右移 | `a >>> b`            |

**逻辑运算符优先级:** `&& > || > ! > & > | > ^ > ~ > << > >> > >>>`

### 7.1 逻辑运算符的运算逻辑

**逻辑与 `&&` 运算符：**
- 如果两个运算数都为真，则结果为真
- 如果两个运算数有一个为假，则结果为假
- 如果两个运算数都为假，则结果为假

**逻辑或 `||` 运算符：**
- 如果两个运算数都为真，则结果为真
- 如果两个运算数有一个为真，则结果为真
- 如果两个运算数都为假，则结果为假

**逻辑非 `!` 运算符：**
- 如果运算数为真，则结果为假
- 如果运算数为假，则结果为真

**按位与 `&` 运算符：**
- 如果两个运算数都为1，则结果为1
- 如果两个运算数有一个为0，则结果为0
- 如果两个运算数都为0，则结果为0

**按位或 `|` 运算符：**
- 如果两个运算数都为1，则结果为1
- 如果两个运算数有一个为1，则结果为1
- 如果两个运算数都为0，则结果为0

**按位异或 `^` 运算符：**
- 如果两个运算数都为1，则结果为0
- 如果两个运算数有一个为1，则结果为1
- 如果两个运算数都为0，则结果为0

**按位取反 `~` 运算符：**
- 如果运算数为1，则结果为0
- 如果运算数为0，则结果为1

**左移 `<<` 运算符：**
- 左移运算符将运算数向左移动指定位数，并把移出的位用0填充

**右移 `>>` 运算符：**
- 右移运算符将运算数向右移动指定位数，并把移出的位用0填充

**无符号右移 `>>>` 运算符：**
- 无符号右移运算符将运算数向右移动指定位数，并把移出的位用0填充

**示例：**

```java
int a = 10;
a <<= 2;
System.out.println(a); // 40

int b = 10;
b >>= 2;
System.out.println(b); // 2

int c = 10;
c >>>= 2;
System.out.println(c); // 2
```

**示例:**

