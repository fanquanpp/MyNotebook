## 3 . 数制表示

<a id="3.1"></a>
### 3.1 常用进制

| 进制 | 组成元素 | 前缀 | 示例 |
| :--- | :--- | :--- | :--- |
| 二进制 | 0, 1 | 0b | 0b1010 |
| 八进制 | 0-7 | 0 | 010 |
| 十进制 | 0-9 | 无 | 10 |
| 十六进制 | 0-9, a-f | 0x | 0x10 |

<a id="3.2"></a>
### 3.2 进制表示示例

```java
public class Demo {
    public static void main(String[] args) {
        int a = 0b1010; // 二进制
        int b = 010; // 八进制
        int c = 0x10; // 十六进制
        System.out.println(a); // 输出：10
        System.out.println(b); // 输出：8
        System.out.println(c); // 输出：16
    }
}
```

## 4 . 进制转换

<a id="4.1"></a>
### 4.1 任意进制转换为十进制

**公式：** 系数 × 基数的权次幂（相加）

```
a×b^n + a×b^(n-1) + ... + a×b^1 + a×b^0
```

- **系数**：每一位上的数字
- **基数**：当前进制数
- **权**：从右往左依次递增（0、1、2、3、4、5……）

**示例：**
- 101（二进制）→ 5(十进制)
  ```
  1               0               1
 （1×2^2）        (0×2^1)          （1×2^0）
  |               |               |
  4         +     0        +      1   =   5( 十进制)
  ```
- 101（八进制）→ 65(十进制)
- abc（十六进制）→ 2748(十进制)

<a id="4.2"></a>
### 4.2 二进制转十进制（8421 快速转化法）

**原理：** 每一位二进制值的 1 都对应一个固定的值，将每一位 1 代表的固定值相加，即为十进制的值

```
二进制：1    1   1   1    1   1   1   1
对应值：128  64  32  16   8   4   2   1
十进制：（255）
```

**示例：**
- 二进制：1101 → (8、4、0、1)
- 十进制：13 （8+4+0+1=13）

<a id="4.3"></a>
### 4.3 十进制转换成其他进制（除基取余法）

**原理：** 不断除以基数（几进制基数就是几）得到余数，直到商为 0，再将余数倒序拼接即可

1. 除以基数，取余数
2. 将余数保存起来
3. 除以基数，取余数
4. 将余数保存起来
5. 重复 3、4 步骤，直到商为 0
6. 将余数保存的数组倒序输出

**示例：**
- 10（十进制）→ 1010（二进制）
  ```
  10/2 = 5   10%2 = 0
  5/2 = 2   5%2 = 1
  2/2 = 1   2%2 = 0
  1/2 = 0   1%2 = 1
  
  余数倒序：1010
  ```

## 5 . 数据存储

<a id="5.1"></a>
### 5.1 文本数据

- **数字**：转换为二进制存储
- **字母**：查询 ASCII 码表，转换为二进制存储
- **汉字**：查询 Unicode 码表，转换为二进制存储

<a id="5.2"></a>
### 5.2 图片数据

- **黑白图**：每个像素用 1 位表示（0/1）
- **灰度图**：每个像素用 8 位表示（0-255）
- **彩图**：每个像素用 RGB 三原色表示，每个颜色通道用 8 位（0-255）

**可用进制表示：**
- 十进制：（255 红， 255 绿， 255 蓝）
- 十六进制：(FF 红， FF 绿， FF 蓝)

<a id="5.3"></a>
### 5.3 声音数据

对声波的波形图进行采样，将模拟信号转换为数字信号存储。

## 6 . 常见问题与解决方案

<a id="6.1"></a>
### 6.1 进制表示错误

**问题描述：** 编写代码时进制表示错误，导致编译失败。

**解决方案：**
- 二进制：确保以 `0b` 开头，只包含 0 和 1
- 八进制：确保以 `0` 开头，只包含 0-7
- 十六进制：确保以 `0x` 开头，只包含 0-9 和 a-f/A-F

<a id="6.2"></a>
### 6.2 进制转换错误

**问题描述：** 手动进行进制转换时出错。

**解决方案：**
- 使用 8421 快速转化法进行二进制到十进制的转换
- 使用除基取余法进行十进制到其他进制的转换
- 对于复杂转换，使用计算器或编程工具辅助

## 7 . 总结与最佳实践

<a id="7.1"></a>
### 7.1 核心概念

- **进制**：二进制、八进制、十进制、十六进制
- **进制转换**：任意进制到十进制、十进制到其他进制
- **数据存储**：文本、图片、声音等数据的存储方式

<a id="7.2"></a>
### 7.2 最佳实践

1. **代码中的进制使用**
   - 二进制：用于位操作和底层编程
   - 八进制：较少使用，主要用于某些特定场景
   - 十六进制：用于表示颜色、内存地址等
   - 十进制：日常编程中最常用

2. **进制转换技巧**
   - 掌握 8421 快速转化法
   - 熟悉除基取余法
   - 利用编程工具进行复杂转换

3. **数据存储理解**
   - 了解不同类型数据的存储方式
   - 理解二进制在计算机中的核心地位
   - 掌握数据类型的大小和范围

<a id="7.3"></a>
### 7.3 个人实践总结
- 熟练掌握各进制的表示方法和转换技巧
- 理解数据在计算机中的存储原理
- 在编程中根据需要选择合适的进制表示
- 注意进制表示的语法正确性，避免编译错误
