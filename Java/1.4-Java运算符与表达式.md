## 1. 运算符分类

### 1.1 分类概览

| 分类 | 运算符 | 示例 |
| :--- | :--- | :--- |
| 算术运算符 | +, -, *, /, %, ++, -- | `a + b`, `i++` |
| 关系运算符 | ==, !=, >, <, >=, <= | `a > b`, `x == y` |
| 逻辑运算符 | &&, \|\|, ! | `a && b`, `!flag` |

## 2. 算术运算符

### 2.1 基本算术运算符

| 运算符 | 描述 | 示例 |
| :--- | :--- | :--- |
| `+` | 加 | `a + b` |
| `-` | 减 | `a - b` |
| `*` | 乘 | `a * b` |
| `/` | 除 | `a / b` |
| `%` | 取余 | `a % b` |

### 2.2 自增自减运算符

| 运算符 | 描述 | 示例 |
| :--- | :--- | :--- |
| `++` | 自增 | `i++`, `++i` |
| `--` | 自减 | `i--`, `--i` |

**示例：**

```java
int i = 10;
i++;
System.out.println(i); // 11

int j = 10;
j--;
System.out.println(j); // 9
```

### 2.3 自增自减运算符的区别

**`++i` 先自增再使用：**

```java
int a, b;
a = 10;
b = ++a;
System.out.println(b); // 11
System.out.println(a); // 11
```

**`i++` 先使用再自增：**

```java
int a, b;
a = 10;
b = a++;
System.out.println(b); // 10
System.out.println(a); // 11
```

## 5. 逻辑运算符

### 5.1 逻辑运算符列表

| 运算符 | 描述 | 示例 |
| :--- | :--- | :--- |
| `&&` | 逻辑与 | `a && b` |
| `\|\|` | 逻辑或 | `a \|\| b` |
| `!` | 逻辑非 | `!a` |

### 5.2 逻辑运算符的运算逻辑

**逻辑与 `&&` 运算符：**
- 如果两个运算数都为真，则结果为真
- 如果两个运算数有一个为假，则结果为假
- 如果两个运算数都为假，则结果为假

**逻辑或 `\|\|` 运算符：**
- 如果两个运算数都为真，则结果为真
- 如果两个运算数有一个为真，则结果为真
- 如果两个运算数都为假，则结果为假

**逻辑非 `!` 运算符：**
- 如果运算数为真，则结果为假
- 如果运算数为假，则结果为真

**示例：**

```java
boolean a = true, b = false;
System.out.println(a && b); // false
System.out.println(a \|\| b); // true
System.out.println(!a); // false
System.out.println(!b); // true
```

### 5.3 短路特性

- `&&` 当第一个操作数为 false 时，第二个操作数不执行
- `\|\|` 当第一个操作数为 true 时，第二个操作数不执行

**示例：**

```java
int a = 10;
boolean result = (a > 20) && (++a > 10);
System.out.println(result); // false
System.out.println(a); // 10，因为短路，++a 没有执行

int b = 10;
boolean result2 = (b < 20) \|\| (++b > 10);
System.out.println(result2); // true
System.out.println(b); // 10，因为短路，++b 没有执行
```

## 6. 三元运算符

### 6.1 语法格式

```java
条件表达式 ? 表达式1 : 表达式2;
```

### 6.2 核心逻辑

先执行条件表达式，得到布尔值（boolean）。若结果为真（true）则输出表达式一；若结果为假（false）则输出表达式二。

### 6.3 等价于

```java
if(条件表达式) {
    表达式1;
} else {
    表达式2;
}
```

### 6.4 示例

```java
int a = 1, b = 2;
int max = (a > b) ? a : b; // false，输出 b
System.out.println(max); // 2

String result = (a > b) ? "a 大于 b" : "a 小于等于 b";
System.out.println(result); // a 小于等于 b
```

## 7. 位运算符

### 7.1 位运算符列表

| 运算符 | 描述 | 示例 |
| :--- | :--- | :--- |
| `&` | 按位与 | `a & b` |
| `\|` | 按位或 | `a \| b` |
| `^` | 按位异或 | `a ^ b` |
| `~` | 按位取反 | `~a` |
| `<<` | 左移 | `a << b` |
| `>>` | 右移 | `a >> b` |
| `>>>` | 无符号右移 | `a >>> b` |

### 7.2 位运算符的运算逻辑

**按位与 `&` 运算符：**
- 如果两个运算数都为 1，则结果为 1
- 如果两个运算数有一个为 0，则结果为 0
- 如果两个运算数都为 0，则结果为 0

**按位或 `\|` 运算符：**
- 如果两个运算数都为 1，则结果为 1
- 如果两个运算数有一个为 1，则结果为 1
- 如果两个运算数都为 0，则结果为 0

**按位异或 `^` 运算符：**
- 如果两个运算数都为 1，则结果为 0
- 如果两个运算数有一个为 1，则结果为 1
- 如果两个运算数都为 0，则结果为 0

**按位取反 `~` 运算符：**
- 如果运算数为 1，则结果为 0
- 如果运算数为 0，则结果为 1

**左移 `<<` 运算符：**
- 左移运算符将运算数向左移动指定位数，并把移出的位用 0 填充

**右移 `>>` 运算符：**
- 右移运算符将运算数向右移动指定位数，并把移出的位用 0 填充

**无符号右移 `>>>` 运算符：**
- 无符号右移运算符将运算数向右移动指定位数，并把移出的位用 0 填充

### 7.3 示例

```java
int a = 10; // 二进制：1010
a <<= 2;
System.out.println(a); // 40，二进制：101000

int b = 10; // 二进制：1010
b >>= 2;
System.out.println(b); // 2，二进制：10

int c = 10; // 二进制：1010
c >>>= 2;
System.out.println(c); // 2，二进制：10
```

## 8. 运算符优先级

### 8.1 优先级从高到低

1. **括号**：`()`
2. **一元运算符**：`++`、`--`、`!`、`~`
3. **算术运算符**：`*`、`/`、`%` 高于 `+`、`-`
4. **移位运算符**：`<<`、`>>`、`>>>`
5. **关系运算符**：`>`、`<`、`>=`、`<=`、`instanceof` 高于 `==`、`!=`
6. **位运算符**：`&` 高于 `^` 高于 `\|`
7. **逻辑运算符**：`&&` 高于 `\|\|`
8. **三元运算符**：`? :`
9. **赋值运算符**：`=`、`+=`、`-=`、`*=`、`/=`、`%=` 等

**注意：** 使用括号可以改变运算顺序，提高代码可读性。

## 9. 表达式

### 9.1 表达式类型

- **算术表达式**：执行算术运算，如 `a + b`
- **关系表达式**：执行比较运算，返回布尔值，如 `a > b`
- **逻辑表达式**：执行逻辑运算，返回布尔值，如 `a && b`
- **赋值表达式**：执行赋值操作，如 `a = b`
- **三元表达式**：根据条件返回不同值，如 `a > b ? a : b`

### 9.2 表达式示例

```java
// 算术表达式
int sum = 10 + 20;
int average = (10 + 20) / 2;

// 关系表达式
boolean isGreater = 10 > 5;
boolean isEqual = 10 == 10;

// 逻辑表达式
boolean result = (10 > 5) && (5 < 8);

// 赋值表达式
int x = 10;
x += 5; // 等价于 x = x + 5

// 三元表达式
int max = (a > b) ? a : b;
```

## 10. 常见问题与解决方案

### 10.1 整数除法问题

**问题描述：** 两个整数相除，结果为整数，小数部分被截断。

**原因分析：** Java 中整数除法默认返回整数类型。

**解决方案：** 将其中一个操作数转换为浮点数，如 `(double)a / b`

### 10.2 自增自减运算符问题

**问题描述：** 自增自减运算符在表达式中的行为容易混淆。

**原因分析：** `i++` 和 `++i` 的执行顺序不同。

**解决方案：**
- `i++`：先使用后自增
- `++i`：先自增后使用
- 避免在复杂表达式中使用自增自减运算符

### 10.3 逻辑运算符短路问题

**问题描述：** 使用 `&&` 和 `\|\|` 时，可能会发生短路现象。

**原因分析：** `&&` 当第一个操作数为 false 时，第二个操作数不执行；`\|\|` 当第一个操作数为 true 时，第二个操作数不执行。

**解决方案：** 注意短路特性，避免在逻辑表达式中包含有副作用的操作。

### 10.4 运算符优先级问题

**问题描述：** 表达式运算顺序不符合预期。

**原因分析：** 运算符优先级不明确。

**解决方案：** 使用括号明确指定运算顺序，提高代码可读性。

## 11. 总结与最佳实践

### 11.1 核心概念

- **运算符**：用于执行各种运算操作的符号
- **表达式**：由运算符和操作数组成的代码片段
- **优先级**：决定表达式中运算的执行顺序
- **结合性**：决定同一优先级运算符的执行顺序

### 11.2 最佳实践

1. **代码可读性**
   - 使用括号明确运算顺序
   - 合理使用空格和换行，提高代码可读性
   - 避免在复杂表达式中使用自增自减运算符

2. **运算安全**
   - 避免整数除法导致的精度丢失
   - 注意浮点数运算的精度问题
   - 避免除以零的错误

3. **性能优化**
   - 合理使用位运算提高性能
   - 避免不必要的运算
   - 注意逻辑运算符的短路特性

4. **代码风格**
   - 运算符两侧添加空格
   - 一行只包含一个表达式
   - 复杂表达式拆分为多个简单表达式

5. **常见场景**
   - 使用三元运算符简化条件判断
   - 使用复合赋值运算符提高代码简洁性
   - 合理使用位运算处理二进制数据

### 11.3 个人实践总结
- 使用括号明确运算顺序，提高代码可读性
- 注意运算安全，避免除以零和精度丢失
- 合理使用位运算提高性能
- 保持代码风格一致，运算符两侧添加空格
- 利用三元运算符简化条件判断
- 注意逻辑运算符的短路特性
