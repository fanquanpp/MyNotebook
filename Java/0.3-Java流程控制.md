## 1. if语句

### 1.1 if的单分支结构

**格式：**

```java
if(条件表达式){
    //条件成立时执行的代码内容；
}
```

**运行逻辑：**
当条件成立时（true），执行{}里面的代码，不成立则直接跳过不执行。

**例子：**

```java
int i= 10086;   //定义变量i
if(i > 114514){     //条件为i大于114514(条件不成立)
     System.out.println("宝宝巴士教我上厕所秘诀");  //此句不会输出
}
```

### 1.2 if的双分支结构

**例子：**

```java
int i = 10; //定义一个变量
if(i >= 10){    //判断条件
     System.out.println("天上浮云似白衣，斯须改变成苍狗。");    //成立则执行此命令
}else{
     System.out.println("山重水尽疑无路，柳暗花明又一村。");    //否则（条件不成立，则执行另外一条命令）
}
```

### 1.3 if的多分支结构

**结构：**

```java
if(条件一){
    System.out.println("最是人间留不住，朱颜辞镜花辞树。"); //条件一成立时执行的代码
}else if(条件二){
    System.out.println("苔花如米小，也学牡丹开。"); //条件二成立时执行的代码
}else if(条件三){
    System.out.println("夕阳西下，断肠人在天涯。"); //条件三成立时执行的代码
}else{  //此处可以为else，也可以为else if，if可以省略
    System.out.println("但行好事，莫问前程。"); //三个条件都不成立时执行的代码
}
```

**运行逻辑：**
当条件一成立时，执行{}里面的代码，否则进入下一条判断条件，当条件二成立时，执行{}里面的代码，否则进入下一条判断条件，当条件三成立时，执行{}里面的代码，否则执行else{}里面的代码。

### 1.4 if的嵌套结构

**嵌套结构：**

```java
if(条件一){ //当条件一不成立，{}内的代码不会执行，直接跳过。
    if(条件二){
        System.out.println("最是人间留不住，朱颜辞镜花辞树。"); //条件一和条件二都成立时执行的代码
    }else{
        System.out.println("苔花如米小，也学牡丹开。"); //条件一成立而条件二不成立时执行的代码
    }
}else{
    System.out.println("夕阳西下，断肠人在天涯。");}    //条件一不成立时执行的代码
```

## 2. for循环

### 2.1 格式：

```java
for(初始化语句;条件判断语句;条件控制语句){
    循环体;
}
```

### 2.2 解析：

```java
for（定义一个初始值变量;判断结束的条件;变量变化的逻辑）{
    循环体（这里的内容会重复执行，直到满足条件才会结束循环）;
}
```

### 2.3 举例：

```java
package loopdemo;

public class ForDemo1
{
    public static void main(String[] args)
    {
        /**for循环：
         * for(初始化语句;条件判断语句;条件控制语句)
         * {
         *     循环体;
         * }
          */

        for (int i = 1; i <= 10; i++)
        {
            System.out.println("HelloWorld!!!");    //输出10次
        }
    }

}
```

### 2.4 总结：

for循环，循环次数是固定的，循环次数由for循环的初始化语句、条件判断语句、条件控制语句决定。

### 2.5 解构：

```java
for(初始化语句;条件判断语句;条件控制语句)
{
    循环体;
}
        
        for循环的初始化语句：
            int i = 1;
            
        for循环的判断语句：
            i <= 10;
            
        for循环的条件控制语句：
            i++;
            
        for循环的循环体：
            System.out.println("HelloWorld!!!");
```

## 3. while循环

### 3.1 while循环结构：

```java
while(条件表达式){
    循环体
}
```

### 3.2 例子：

```java
public class Demo01While {
    public static void main(String[] args) {
        int a = 1;  //初始化条件
        while(a <= 5){  //条件表达式
            System.out.println("hello world");  //循环体
            a++;    //更新条件
        }
    }
}    //输出结果：hello world 五次
```

### 3.3 循环的结束条件：
1. 条件表达式为false时，循环结束
2. 条件表达式为true时，循环继续

### 3.4 注意：
1. while循环至少会执行一次循环体
2. while循环的结束条件一定要是可判断的，否则会死循环

### 3.5 while循环的升级：

**1. while循环的升级：**

```java
while(条件表达式){
    循环体
}
```

**2. do...while循环：**

```java
do{
    循环体
}while(条件表达式);
```

**注意：** 
do...while循环至少会执行一次循环体
do...while循环的结束条件一定要是可判断的，否则会死循环

**3. while循环升级的例子：**

```java
public class Demo02DoWhile {
    public static void main(String[] args) {
        int a = 1;  //初始化条件
        while(a <= 5){  //条件表达式
            System.out.println("hello world");  //循环体
            a++;    //更新条件
        }

        int b = 1;  //初始化条件
    }
}
```

**4. do...while循环的例子：**

```java
public class Demo03DoWhile {
    public static void main(String[] args) {
        int a = 1;  //初始化条件
        do{
            System.out.println("hello world");  //循环体
            a++;    //更新条件
        }while(a <= 5);  //条件表达式
    }
}
```

## 4. switch语句

### 4.1 switch语句结构(基础):

```java
switch(表达式){
    case 1: 
        语句1;
        break;
    case 2: 
        语句2;
        break;
    case 3: 
        语句3;
        break;
    default: 
        语句4;
}
```

**例子：**

```java
switch(i){
    case 1: 
        System.out.println("星期一");   //i=1,输出星期一
        break;
    case 2: 
        System.out.println("星期二");   //i=2,输出星期二
        break;
    case 3: 
        System.out.println("星期三");   //i=3,输出星期三
        break;
    case 4: 
        System.out.println("星期四");   //i=4,输出星期四
        break;
    case 5: 
        System.out.println("星期五");   //i=5,输出星期五
        break;  //这个break是必须的
}
```

**break的作用：**
用于跳出循环结构（如 for、while 和 do-while 循环）或 switch 语句。当遇到 break 语句时，程序将立即跳出当前循环或 switch 语句，继续执行紧跟在循环或 switch 语句后面的代码。

### 4.2 switch语句(case穿透法)

**case穿透:** 当不同case的结果一样时，可以把break省略,只留下最后一个break.

**例子：**

```java
switch(i){ 
    case 1: 
    case 2:
    case 3:
    case 4:
    case 5:
        System.out.println("工作日");   //i=1,i=2,i=3,i=4,i=5,输出工作日
        break;      
    case 6: 
    case 7: 
        System.out.println("休息日");   //i=6,i=7,输出休息日
        break;
        default: 
        System.out.println("输入错误");
        break;
}
```

### 4.3 switch语句(新特性:箭头语法)

**箭头语法:** switch语句的case穿透方法，也可以使用箭头语法。

**例子(无穿透)：**

```java
switch(i){ 
    case 1 -> System.out.println("工作日");   //i=1,输出工作日
    case 2 -> System.out.println("工作日");   //i=2,输出工作日
    case 3 -> System.out.println("工作日");   //i=3,输出工作日
    case 4 -> System.out.println("工作日");   //i=4,输出工作日
    case 5 -> System.out.println("工作日");   //i=5,输出工作日
    case 6 -> System.out.println("休息日");   //i=6,输出休息日
    case 7 -> System.out.println("休息日");   //i=7,输出休息日
    default -> System.out.println("输入错误"); 
}
```

**例子(有穿透)：**

```java
switch(i){ 
    case 1,2,3,4,5 -> System.out.println("工作日");   //i=1,i=2,i=3,i=4,i=5,输出工作日
    case 6,7 -> System.out.println("休息日");   //i=6,i=7,输出休息日
    default -> System.out.println("输入错误");  
}
```

**箭头语法的语法格式为：**

```java
case 值1, 值2, 值3, ... -> 语句;
```

**注意:** 箭头语法时,不需要break。

**箭头语法的优点:**
1. 代码简洁
2. 可读性高

**箭头语法的特点:**
1. 只能用于switch语句
2. 自带阻断效果,不需要break
