# MySQL SQL 基础语法

<a id="1"></a>
## 1 . 概述

SQL (Structured Query Language) 是用于管理关系型数据库的标准语言，MySQL 完全支持 SQL 标准。SQL 语句可以分为四大类：DDL (数据定义语言)、DML (数据操作语言)、DCL (数据控制语言) 和 TCL (事务控制语言)。本教程将详细介绍 MySQL 中常用的 SQL 语法。

<a id="2"></a>
## 2 . SQL 分类

<a id="2.1"></a>
### 2.1 DDL (Data Definition Language)

数据定义语言，用于创建、修改和删除数据库对象，如数据库、表、索引等。

**主要语句**：
- `CREATE`：创建数据库对象
- `ALTER`：修改数据库对象
- `DROP`：删除数据库对象
- `TRUNCATE`：清空表数据

<a id="2.2"></a>
### 2.2 DML (Data Manipulation Language)

数据操作语言，用于插入、更新、删除和查询数据。

**主要语句**：
- `INSERT`：插入数据
- `UPDATE`：更新数据
- `DELETE`：删除数据
- `SELECT`：查询数据

<a id="2.3"></a>
### 2.3 DCL (Data Control Language)

数据控制语言，用于控制用户权限。

**主要语句**：
- `GRANT`：授予权限
- `REVOKE`：撤销权限
- `CREATE USER`：创建用户
- `DROP USER`：删除用户

<a id="2.4"></a>
### 2.4 TCL (Transaction Control Language)

事务控制语言，用于管理事务。

**主要语句**：
- `START TRANSACTION`：开始事务
- `COMMIT`：提交事务
- `ROLLBACK`：回滚事务
- `SAVEPOINT`：设置保存点

<a id="3"></a>
## 3 . DDL 语句

<a id="3.1"></a>
### 3.1 创建数据库

```sql
-- 创建数据库
CREATE DATABASE database_name;

-- 创建数据库并指定字符集
CREATE DATABASE database_name CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建数据库（如果不存在）
CREATE DATABASE IF NOT EXISTS database_name;
```

<a id="3.2"></a>
### 3.2 使用数据库

```sql
USE database_name;
```

<a id="3.3"></a>
### 3.3 删除数据库

```sql
-- 删除数据库
DROP DATABASE database_name;

-- 删除数据库（如果存在）
DROP DATABASE IF EXISTS database_name;
```

<a id="3.4"></a>
### 3.4 创建表

```sql
CREATE TABLE table_name (
    column1 datatype constraints,
    column2 datatype constraints,
    ...
    PRIMARY KEY (column_name),
    FOREIGN KEY (column_name) REFERENCES other_table(column_name)
);
```

**示例**：

```sql
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL,
    email VARCHAR(100) UNIQUE,
    age INT DEFAULT 18,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

<a id="3.5"></a>
### 3.5 修改表

**添加列**：

```sql
ALTER TABLE table_name ADD column_name datatype constraints;
```

**修改列**：

```sql
ALTER TABLE table_name MODIFY column_name datatype constraints;
```

**删除列**：

```sql
ALTER TABLE table_name DROP column_name;
```

**重命名表**：

```sql
ALTER TABLE table_name RENAME TO new_table_name;
```

<a id="3.6"></a>
### 3.6 删除表

```sql
-- 删除表
DROP TABLE table_name;

-- 删除表（如果存在）
DROP TABLE IF EXISTS table_name;
```

<a id="3.7"></a>
### 3.7 清空表

```sql
TRUNCATE TABLE table_name;
```

<a id="4"></a>
## 4 . DML 语句

<a id="4.1"></a>
### 4.1 插入数据

**插入单行数据**：

```sql
INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);
```

**插入多行数据**：

```sql
INSERT INTO table_name (column1, column2, ...) VALUES
(value1, value2, ...),
(value3, value4, ...),
(value5, value6, ...);
```

**示例**：

```sql
INSERT INTO users (name, email, age) VALUES
('张三', 'zhangsan@example.com', 25),
('李四', 'lisi@example.com', 30),
('王五', 'wangwu@example.com', 35);
```

<a id="4.2"></a>
### 4.2 更新数据

```sql
UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;
```

**示例**：

```sql
-- 更新单个字段
UPDATE users SET age = 26 WHERE id = 1;

-- 更新多个字段
UPDATE users SET age = 31, email = 'new_email@example.com' WHERE id = 2;

-- 更新所有记录
UPDATE users SET age = age + 1;
```

<a id="4.3"></a>
### 4.3 删除数据

```sql
DELETE FROM table_name WHERE condition;
```

**示例**：

```sql
-- 删除指定记录
DELETE FROM users WHERE id = 3;

-- 删除符合条件的记录
DELETE FROM users WHERE age > 30;

-- 删除所有记录
DELETE FROM users;
```

<a id="4.4"></a>
### 4.4 查询数据

**基本查询**：

```sql
SELECT column1, column2, ... FROM table_name WHERE condition;
```

**查询所有列**：

```sql
SELECT * FROM table_name;
```

**带条件查询**：

```sql
SELECT * FROM users WHERE age > 25 AND name LIKE '%张%';
```

**排序**：

```sql
-- 升序
SELECT * FROM users ORDER BY age ASC;

-- 降序
SELECT * FROM users ORDER BY age DESC;
```

**限制结果数量**：

```sql
-- 限制返回 5 条记录
SELECT * FROM users LIMIT 5;

-- 从第 3 条开始，返回 5 条记录
SELECT * FROM users LIMIT 2, 5;
```

**分组**：

```sql
SELECT age, COUNT(*) FROM users GROUP BY age;
```

**分组后过滤**：

```sql
SELECT age, COUNT(*) FROM users GROUP BY age HAVING COUNT(*) > 1;
```

**连接查询**：

```sql
-- 内连接
SELECT users.name, orders.product FROM users INNER JOIN orders ON users.id = orders.user_id;

-- 左连接
SELECT users.name, orders.product FROM users LEFT JOIN orders ON users.id = orders.user_id;

-- 右连接
SELECT users.name, orders.product FROM users RIGHT JOIN orders ON users.id = orders.user_id;

-- 全连接（MySQL 不直接支持，可通过 UNION 实现）
SELECT users.name, orders.product FROM users LEFT JOIN orders ON users.id = orders.user_id
UNION
SELECT users.name, orders.product FROM users RIGHT JOIN orders ON users.id = orders.user_id;
```

**子查询**：

```sql
-- 子查询作为条件
SELECT * FROM users WHERE age > (SELECT AVG(age) FROM users);

-- 子查询作为表
SELECT * FROM (SELECT * FROM users WHERE age > 25) AS mature_users;
```

<a id="5"></a>
## 5 . DCL 语句

<a id="5.1"></a>
### 5.1 创建用户

```sql
CREATE USER 'username'@'host' IDENTIFIED BY 'password';
```

**示例**：

```sql
-- 创建本地用户
CREATE USER 'user1'@'localhost' IDENTIFIED BY 'password123';

-- 创建允许远程连接的用户
CREATE USER 'user2'@'%' IDENTIFIED BY 'password123';
```

<a id="5.2"></a>
### 5.2 授予权限

```sql
GRANT privileges ON database_name.table_name TO 'username'@'host';
```

**示例**：

```sql
-- 授予所有权限
GRANT ALL PRIVILEGES ON *.* TO 'user1'@'localhost';

-- 授予特定权限
GRANT SELECT, INSERT, UPDATE ON testdb.* TO 'user2'@'%';

-- 授予权限并允许授权
GRANT ALL PRIVILEGES ON testdb.* TO 'user3'@'localhost' WITH GRANT OPTION;
```

<a id="5.3"></a>
### 5.3 撤销权限

```sql
REVOKE privileges ON database_name.table_name FROM 'username'@'host';
```

**示例**：

```sql
-- 撤销所有权限
REVOKE ALL PRIVILEGES ON *.* FROM 'user1'@'localhost';

-- 撤销特定权限
REVOKE DELETE ON testdb.* FROM 'user2'@'%';
```

<a id="5.4"></a>
### 5.4 删除用户

```sql
DROP USER 'username'@'host';
```

**示例**：

```sql
DROP USER 'user1'@'localhost';
```

<a id="5.5"></a>
### 5.5 查看权限

```sql
SHOW GRANTS FOR 'username'@'host';
```

**示例**：

```sql
SHOW GRANTS FOR 'user1'@'localhost';
```

<a id="6"></a>
## 6 . TCL 语句

<a id="6.1"></a>
### 6.1 开始事务

```sql
START TRANSACTION;
-- 或
BEGIN;
```

<a id="6.2"></a>
### 6.2 提交事务

```sql
COMMIT;
```

<a id="6.3"></a>
### 6.3 回滚事务

```sql
ROLLBACK;
```

<a id="6.4"></a>
### 6.4 设置保存点

```sql
SAVEPOINT savepoint_name;
```

<a id="6.5"></a>
### 6.5 回滚到保存点

```sql
ROLLBACK TO savepoint_name;
```

<a id="6.6"></a>
### 6.6 释放保存点

```sql
RELEASE SAVEPOINT savepoint_name;
```

**示例**：

```sql
START TRANSACTION;

-- 插入数据
INSERT INTO users (name, email) VALUES ('赵六', 'zhaoliu@example.com');

-- 设置保存点
SAVEPOINT sp1;

-- 更新数据
UPDATE users SET age = 40 WHERE name = '赵六';

-- 回滚到保存点
ROLLBACK TO sp1;

-- 提交事务
COMMIT;
```

## 7 . 常用函数

<a id="7.1"></a>
### 7.1 聚合函数

- `COUNT()`：计数
- `SUM()`：求和
- `AVG()`：平均值
- `MAX()`：最大值
- `MIN()`：最小值

**示例**：

```sql
SELECT COUNT(*) FROM users;
SELECT SUM(age) FROM users;
SELECT AVG(age) FROM users;
SELECT MAX(age) FROM users;
SELECT MIN(age) FROM users;
```

<a id="7.2"></a>
### 7.2 字符串函数

- `CONCAT()`：连接字符串
- `SUBSTRING()`：截取字符串
- `LENGTH()`：字符串长度
- `UPPER()`：转换为大写
- `LOWER()`：转换为小写
- `TRIM()`：去除首尾空格

**示例**：

```sql
SELECT CONCAT(name, ' - ', email) FROM users;
SELECT SUBSTRING(name, 1, 2) FROM users;
SELECT LENGTH(name) FROM users;
SELECT UPPER(name) FROM users;
SELECT LOWER(email) FROM users;
SELECT TRIM(name) FROM users;
```

<a id="7.3"></a>
### 7.3 日期函数

- `NOW()`：当前日期时间
- `CURDATE()`：当前日期
- `CURTIME()`：当前时间
- `DATE()`：提取日期部分
- `TIME()`：提取时间部分
- `DATEDIFF()`：计算日期差
- `DATE_ADD()`：添加日期

**示例**：

```sql
SELECT NOW();
SELECT CURDATE();
SELECT CURTIME();
SELECT DATE(created_at) FROM users;
SELECT TIME(created_at) FROM users;
SELECT DATEDIFF(CURDATE(), created_at) FROM users;
SELECT DATE_ADD(created_at, INTERVAL 7 DAY) FROM users;
```

<a id="7.4"></a>
### 7.4 数值函数

- `ABS()`：绝对值
- `ROUND()`：四舍五入
- `CEIL()`：向上取整
- `FLOOR()`：向下取整
- `RAND()`：随机数

**示例**：

```sql
SELECT ABS(-10);
SELECT ROUND(3.14159, 2);
SELECT CEIL(3.14);
SELECT FLOOR(3.99);
SELECT RAND();
```

## 8 . 运算符

<a id="8.1"></a>
### 8.1 算术运算符

- `+`：加
- `-`：减
- `*`：乘
- `/`：除
- `%`：取模

<a id="8.2"></a>
### 8.2 比较运算符

- `=`：等于
- `<>` 或 `!=`：不等于
- `<`：小于
- `>`：大于
- `<=`：小于等于
- `>=`：大于等于
- `BETWEEN`：在范围内
- `IN`：在集合中
- `LIKE`：模糊匹配
- `IS NULL`：为空
- `IS NOT NULL`：不为空

<a id="8.3"></a>
### 8.3 逻辑运算符

- `AND`：与
- `OR`：或
- `NOT`：非

<a id="8.4"></a>
### 8.4 位运算符

- `&`：按位与
- `|`：按位或
- `^`：按位异或
- `~`：按位取反
- `<<`：左移
- `>>`：右移

## 10 . 总结

SQL 是使用 MySQL 的核心语言，通过本教程的学习，你应该已经掌握了 MySQL 中常用的 SQL 语法，包括 DDL、DML、DCL 和 TCL 语句，以及常用函数和运算符。

在实际应用中，你需要根据具体的业务需求，编写相应的 SQL 语句。以下是一些使用 SQL 的最佳实践：

- **使用参数化查询**：避免 SQL 注入
- **优化查询**：使用索引，避免全表扫描
- **合理使用事务**：确保数据的一致性和完整性
- **定期备份**：防止数据丢失
- **使用适当的权限**：遵循最小权限原则

通过不断学习和实践，你将能够熟练使用 SQL 来管理和操作 MySQL 数据库，为你的应用提供高效、可靠的数据支持。
